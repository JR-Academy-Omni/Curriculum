// 报名前最常问

FAQ · 9 条已覆盖你想问的

我现在大一 / 大二，能来吗？
大一大二能来，效果不同。大三 / Master 倒数第二学期来最对口 — 你接下来就要投提前批了。大一大二来是『提前看地图』 — 知道未来要做什么准备（GPA / 实习 / 学生干部 / 党员入党申请 / 学校 vs 专业的权重）。最不适合的是已经毕业 2 年没工作的同学 — 应届生身份基本失效，建议另找 mid-career 课程或公考转向。
我目标是公考 / 选调生，这课讲吗？
讲，但只是 sector 概览（L02 + L06 各点到）。原因：公考 / 选调有独立逻辑（行测申论备考 + 政审 + 报考资格 + 海归限制），单独需要 4-6 小时才能讲透。本课定位是 5 大 sector 横扫，让你判断公考是否适合自己。深度公考 / 选调战术，建议参加单独的『公考 / 选调专题』Workshop（Phase 2 上线）。
海归身份在国央企 / 公考会不会被歧视？
不是『歧视』而是『限制』。L02 会详细讲。简要说：① 国央企（华为 / 国家电网 / 中石化等）— 部分要党员 + 户口，海归没党员不一定能进，但技术岗 / 海外业务岗反而欢迎 ② 公务员考试 — 各省市政策不同，部分选调生不接受海归（要求本地应届身份 + 党员），部分省 / 中央国家机关无限制 ③ 央选 / 选调生 — 限制最严，看清招考公告。建议：海归身份适合 互联网 / 外企 / 银行 / 部分国央企，不适合大部分公考选调。
学历认证现在不办来不及吗？
看你目标 cycle。学历认证（中国教育部留学服务中心）1-2 个月办理。秋招提前批（6-8 月）要求网申前出认证报告 → 5 月就要启动。秋招正式批（9-10 月）→ 7-8 月办理。春招（2-5 月）→ 12 月-1 月办理。L01 会讲完整时间线 + 操作步骤。补救：部分企业接受『认证办理中』+ 学位证作为临时凭证，但少数 + 风险高，不建议靠这个。
我没实习，只有海外兼职 / 校园活动，能投秋招吗？
能。L03 会讲『没实习简历怎么写』。关键是把海外兼职 / 校园活动 / 学生干部 / 学术项目用 STAR + 量化数字讲出含金量。海外 retail / cafe 兼职 = 客户服务 + 团队协作 + 冲突处理（这些是 GP 招聘官看的）。校园活动 = Leadership + 组织能力 + 影响力。学术项目 = 分析 + 团队 + 交付。Workshop L03 会现场改 2 段没实习的学员简历，你能直接对照。
我已经回国投了 30+ 家全挂，这课还能救我吗？
能诊断卡在哪关 · 不能保证一定救回来。投 30+ 全挂通常卡在 3 处：① 简历直接翻译英文版（HR 看不懂海外校名 + 没量化）— L03 改写 ② 网测没准备（行测 / 北森 / iTalent 各家不一样）— L04 给路径 ③ 群面 / 业务面被国内本土候选人比下去 — L05 + L05a Quest 给你差异化战术。课上能定位卡哪关 + 课后 1-2 周针对性改 → 下一轮春招（2-5 月）能重新冲。但如果你毕业 < 3 个月 + GPA 不到 3.5/4 + 0 实习，建议先冲春招中小公司或先考公转身份。
用 ChatGPT / Claude 写中文简历会被发现吗？
用 AI 优化中文表达 / 词汇 / 语法 — 没问题，HR 自己也用。但有 3 条红线 ⛔ ① 不要让 AI 编造你没做过的实习 / 经历（背调严，部分公司会查实习单位）② 不要直接复制 AI 写的整段中文（HR 一眼识破『AI 味』）③ 太完美无个性 → ATS 系统能识别『太机械』反而扣分。L03 会讲『AI 辅助 vs AI 代写』的边界，用 AI 改 bullet 不被识别为机器人。
我已经毕业 1.5 年还算应届吗？
看公司 + 看 cycle。教育部规定海外留学生毕业 2 年内未就业可保留应届生资格（以学位证日期为准，不是回国日期）。具体：2024 年 6 月毕业 → 应届身份至 2026 年 6 月。腾讯 2026 届校招通常面向 2025 年 9 月 - 2026 年 8 月毕业；字节 / 阿里 / 美团政策类似。关键判断：① 没缴过社保（必须）② 没签过正式劳动合同 ③ 学位证 < 2 年。L01 会讲怎么核查 + 如何在简历上写应届身份。
不满意可以退款吗？
可以。课后 7 天内无理由退款 — 直接邮件助教，无需说明理由，48h 内全额退回。Workshop 卖的是方法不是结果 · 课后 80% 完课率 + 80% NPS 是我们看的指标 · 真不适合你的话不为难。但请注意：退款后录播访问会同步关闭。
