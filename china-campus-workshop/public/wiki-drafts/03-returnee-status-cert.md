## 一、留学生"应届生身份" 4 维度认定（最核心）

国内招聘"应届生"判定**不是单一标准**，是 4 个维度同时看：

| 维度 | 标准 | 留学生注意 |
|------|------|----------|
| **① 毕业时间** | 学位证日期 < 2 年（部分公司放宽 3 年） | 以**学位证日期为准**，不是回国日期 / 不是签证日期 |
| **② 社保记录** | 从未缴纳过社保（含个人 / 公司缴） | 海外工作不算 · 但回国实习若公司缴了即失资格 |
| **③ 劳动合同** | 从未签过正式劳动合同 | 三方协议不算 · 兼职合同看具体性质 |
| **④ 报到证 / 派遣证** | 没报到 / 派遣（2026 取消中） | 2026 年起逐步取消 · 影响减弱但部分国央企仍查 |

### 实战判断
```
2024.06 毕业（学位证日期）
+ 海外工作 2024.07-2025.06（1 年）
+ 2025.07 回国求职
+ 没缴过国内社保
+ 没签过国内正式劳动合同
= 仍是应届生（至 2026.06 学位证 2 年内）
```

### 互联网大厂 2026 届校招"留学生应届"窗口（参考）
- **腾讯**：2025.09 - 2026.08 毕业的海内外应届生
- **字节跳动**：2025.08 - 2026.07
- **阿里**：2025.10 - 2026.08
- **美团**：2025.10 - 2026.08
- **拼多多**：2025.07 - 2026.08

> 这是公开档期 · 具体年份每年微调 · 以**官网为准**

## 二、学历认证操作（教育部留学服务中心 · 必办）

### 为什么必办
- 95% 国央企 + 大部分私企校招要求"学历认证编号"才能网申
- 投简历前没办 = 大概率被网申系统过滤
- 部分公司允许"认证办理中" + 学位证 + 学校录取通知书作为临时凭证（仅小厂 / 灵活公司）

### 申请流程（2026 cycle）

| 步骤 | 操作 | 周期 |
|------|------|------|
| 1 | 注册 教育部留学服务中心账号（zwfw.cscse.edu.cn） | 1 天 |
| 2 | 在线填申请表 + 上传材料 | 1-2 天 |
| 3 | 在线缴费（¥360 / 套） | 实时 |
| 4 | **核查 + 认证**（最长环节） | 10-20 工作日 |
| 5 | 电子认证书出具（可在线下载） | 1 天 |
| **总周期** | | **1-2 个月** |

### 必备材料
- ✅ 护照首页 + 签证页扫描
- ✅ 学位证 / 毕业证（外文原件 + 中文译本）
- ✅ 成绩单（外文原件 + 中文译本）
- ✅ 学校录取通知书（如适用 · 部分国家学位证不显示入学时间）
- ✅ 出入境记录（公安局打印 / 移民局打印）
- ⚠️ 部分国家要求合作核查（英国 HEDD / 美国 Parchment 等）

### 海外学校 vs 中文译本
- **官方中文译本**：通过 lvyou.cscse.edu.cn 关联翻译机构（标准价 ¥80-200 / 份）
- **自译**：不接受 · 必须公证 / 留服中心合作翻译机构

### 申请时间表（按目标 cycle 倒推）

| 目标 cycle | 学历认证启动时间 | 认证书出具时间 |
|-----------|---------------|--------------|
| 秋招提前批（6-8 月开放） | **最晚 5 月初** | 6 月底前出 |
| 秋招正式批（9-10 月开放） | **最晚 7 月初** | 8 月底前出 |
| 春招（2-5 月开放） | **最晚 12 月-1 月** | 2 月前出 |

> ⚠️ **暑假回国"再办"** = 错过秋招提前批 · 是留学生最大失误

### 申请前 4 项自检
- [ ] 护照 / 签证页 / 学位证 / 成绩单 / 出入境记录 5 件齐全
- [ ] 学位证日期 = 实际毕业日期（部分学校发证晚于毕业 · 影响应届身份）
- [ ] 中文译本来自留服合作机构（自译会被拒）
- [ ] 邮箱可正常收 jypj.cscse.edu.cn 系统邮件（垃圾箱定期查）

## 三、报到证 / 派遣证（2026 政策变化）

### 背景
"报到证"全称 **《全国普通高等学校毕业生就业报到证》** · 1986 年起实施 · 是中国境内毕业生**派遣到工作单位 / 人才市场**的凭证。

### 2026 起政策变化
- **2026 届毕业生起，国家逐步取消报到证**（部分省 2026 即取消 · 全部取消预计 2027）
- 影响：
  - ✅ 应届身份判定更看"社保 + 劳动合同"
  - ✅ 户口 / 档案不再需要报到证作为前置文件
  - ⚠️ 部分国央企 / 公考过渡期仍查（建议保留学校开的"派遣意愿书"）

### 海外留学生**没有报到证** · 怎么办
- 海外学位本来就没派遣证（国内学校才有）
- **教育部留学服务中心可开《留学回国人员证明》** 替代部分场景（已逐步停发 · 2024 后改用学历认证书）
- 国央企招聘提到"报到证" = 看 ② 学历认证编号

## 四、户口落定（5 大常见路径）

### 路径 A · 一线城市公司集体户（最常见）
- 北京 / 上海 / 深圳 / 广州 / 杭州大公司有集体户口
- 留学生回国满足条件可直接落（具体条件每年微调）
- 例：北京海归落户要求"出国留学满 365 天 + 学位 + 回国 2 年内"

### 路径 B · 一线城市人才市场集体户
- 公司没集体户的 · 挂到人才市场（街道办 / 区人才中心）
- 续费每年 ¥200-1000（看城市）
- 落户后该城市常住 / 买房 / 子女入学

### 路径 C · 落老家
- 回原户籍地落户 · 程序最简单
- 适合不在一线城市工作 / 不需要城市户口的同学

### 路径 D · 不落户（学生集体户）
- 部分国央企（特别二三线）入职后**不主动办户口**
- 落户事项学员自己看城市政策

### 路径 E · 落"特殊集体户"
- 部分省（湖北 / 江苏 / 浙江）有"留学生人才集体户"
- 福利政策 + 子女入学便利

### 选择建议
```
计划在一线城市长期 → 路径 A 公司集体户（最稳）
还没决定城市 → 路径 C 落老家（先观望）
小城市工作 → 路径 D 不落或落老家
计划海外再发展 → 路径 D（不浪费指标）
```

## 五、档案调动（学籍档案 vs 人事档案）

### 学籍档案
- 学校管 · 包含本科 / 硕士成绩单 / 学位审议表 / 教师评语等
- 毕业后 1 个月学校自动**派到目标地**（看你毕业去向声明）
- ⚠️ 不要自己拿（"私拆"作废 · 不能再用）

### 人事档案
- 工作单位管 · 包含劳动合同 / 转正 / 调动记录等
- 海外学习期间**没有**人事档案 · 入职后开始建立

### 海外留学生档案路径
```
本科 → 国内（毕业时学校派遣到家乡人才市场）
留学 → 档案"挂存"人才市场（每年付存档费 ¥120-360）
回国就业 → 转到新工作单位 / 新人才市场
```

### 不能踩的坑
- ❌ 自己拿档案放在家里
- ❌ 拒收档案不办理
- ❌ 档案在 A 城市但人在 B 城市工作（部分国央企会查"档案所在地"）
- ❌ 档案丢失（影响公务员 / 国央企未来调动）

## 六、留学生独有 6 项操作清单

### 课前 1 周（T-7d）
- [ ] 教育部留学服务中心 zwfw.cscse.edu.cn 注册账号
- [ ] 准备 5 件认证材料扫描（护照 / 签证 / 学位证 / 成绩单 / 出入境记录）
- [ ] 中文译本通过留服合作机构办（不要自译）

### 校招前（T-3 月）
- [ ] 学历认证书已出具（电子版下载 + 编号记好）
- [ ] 检查应届身份 4 维度（毕业 / 社保 / 合同 / 报到证）
- [ ] 决定户口落定路径（A/B/C/D/E）

### 投递 / 面试期
- [ ] 简历填学历认证编号
- [ ] 网申表中"政治面貌 / 户籍 / 应届身份" 4 个字段准确填
- [ ] 面试被问应届身份 → 用 4 维度自证（社保未缴 + 没签合同 + 学位证 2 年内）

### Offer 后
- [ ] 三方协议（如公司发）/ 录用意向书 → 妥善保留
- [ ] 确认公司给办户口 / 集体户（A 路径）vs 不办（D 路径）
- [ ] 档案接收：公司开具《档案接收函》→ 转到公司

### 入职前 1 月
- [ ] 学历认证书原件提交公司（背调用）
- [ ] 户口办理资料齐备（视路径 A/B/C/D/E）
- [ ] 档案转入完成

### 入职后 30 天
- [ ] 户口正式落定（部分城市 30 天内办完）
- [ ] 社保 / 公积金开户
- [ ] 档案"在岗确认"

## 七、常见疑问 FAQ

### Q1 · 我 2023.07 毕业 · 至 2026.07 还算应届吗？
**A**：理论上是（学位证 2 年内 = 应届）。但实操中：
- 2026.07 已超过部分公司"2 年内"标准（腾讯 / 字节通常严卡 1 年内）
- 即使大公司"放宽 2 年"，也优先看 2025-2026 毕业的同学
- **建议**：投递时强调"无社保 + 无正式合同 + 学位证 < 2 年" · 部分企业会单独评估

### Q2 · 我在海外公司全职工作 3 年了 · 还能投应届校招吗？
**A**：基本不能。"全职工作"= 失去应届身份（即使没缴国内社保）。
- 部分公司有"3 年内毕业 + 工作经历可考虑" · 但这是社招通道
- **建议**：走社招线（决策 Coach / 内推 / 猎头模式 · 见 china-career hub）

### Q3 · 我的海外学位不是教育部认证名单内的学校 · 怎么办？
**A**：
1. 大部分英美澳加新加坡 + Top 学校在留服中心可查
2. 不在的话也能办认证 · 但流程更长 + 部分认证可能"建议性"不强
3. 出国前查 mohe.edu.cn 留服中心"涉外监管网" 看你的学校是否合作
4. 已经回国发现不在名单 → 还是可以办（评估时间长 1-2 月）

### Q4 · 公考要求"基层服务期" 留学生算不算？
**A**：
- 多数省级公考"基层"指国内乡镇 / 街道 / 农村 等
- 海外志愿者 / 海外社区 一般**不被认可**
- 部分省级招考有"留学生人才专项" · 单独走流程（关注省级公务员考试招考公告）

### Q5 · 我在海外结婚了 · 配偶非中国籍 · 影响国央企政审吗？
**A**：
- 部分敏感 sector（涉密国央企 / 部分军工 / 涉外部委）**会问** · 个别岗位限制
- 大部分商业国央企 / 互联网 **不影响**
- 涉外岗位（外交 / 国际业务）反而**加分**
- 建议：投递前 contact HR 确认目标岗位是否涉密
