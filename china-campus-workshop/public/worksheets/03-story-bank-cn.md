## 一、12 类国内校招高频 Behavioural 题

| # | 类目 | 国内 HR / 业务面常问 | 评什么 |
|---|------|------------------|--------|
| 1 | **Leadership** | "讲一次你带过的团队 / 学生干部经历" | 主动 + 推进 + 影响力 |
| 2 | **Teamwork** | "讲一次和别人合作的经历" | 协作 + 包容 |
| 3 | **Conflict** | "和同事 / 同学意见不合怎么办" | 沟通 + 情绪 |
| 4 | **Failure** | "讲一次你失败的经历" | 自省 + 复盘 |
| 5 | **Initiative** | "主动做过什么超出本职的事" | 主动 + 创新 |
| 6 | **Problem-solving** | "解决过的最难问题" | 结构化思维 |
| 7 | **Time pressure** | "DDL 极紧怎么完成的" | 优先级 + 执行 |
| 8 | **Learning** | "短时间学新东西的经历" | 学习能力 |
| 9 | **Customer focus** | "服务客户 / 用户的经历" | 服务意识 |
| 10 | **Communication** | "向不懂技术的人讲技术问题" | 表达 |
| 11 | **Why 回国** ⭐ | "为什么选择回国 / 为什么不留在海外" | 真诚 + 长远视角 |
| 12 | **Why this 公司 / 岗位** | "为什么是我们公司 / 这个岗位" | 调研 + 匹配度 |

> ⭐ **第 11/12 类是留学生独有 · 必答** · 国内本土同学不一定被问

## 二、中文 STAR 公式（4 维度评分）

| 维度 | 占比 | 关键 |
|------|------|------|
| **情境 (Situation)** | 20% | 一句话让 HR 入戏（什么时间 / 什么角色 / 什么场景） |
| **任务 (Task)** | 10% | 要解决什么具体问题 · 不要泛泛说 |
| **行动 (Action)** | 50% | "**我** 做了什么 + 为什么这么做 + 我和别人不同的地方" |
| **结果 (Result)** | 20% | 数字 / 百分比 / 影响范围 + 学到了什么 |

### 留学生中文 STAR 5 个差异化要点

1. **主语用"我"** · 不要"我们团队"（中文 HR 比英文更敏感）
2. **海外经历用国内对标说明** · "项目相当于国内 [XX 公司] 做的 [XX]"
3. **党员 / 学生干部 / 海外经历** · 按目标 sector 差异化处理
4. **结果必须中文化** · "提升 user retention" → "用户次日留存 +12%"
5. **Why 回国答得真诚** · 家人 / 行业机会 / 中国市场体量 · 不要贬低海外

## 五、Why 回国 答法模板（必准备）

```
真诚答法（3 段）：
1. 个人视角 · 1-2 句
   "家在 [城市]，希望离家人近 / 父母年龄大需要照顾 / ..."

2. 行业 / 市场视角 · 2-3 句（最关键）
   "国内 [行业] 这两年发展非常快 — 比如 [具体公司] 在做 [具体业务]，
    海外同类公司 [XX] 还在早期阶段 / 国内已经领先一代 / ..."

3. 个人匹配 · 1-2 句
   "我在 [海外学校 / 实习] 接触过 [类似业务]，希望能把这套经验
    带回国内场景应用 / 见证国内 [行业] 下一阶段 / ..."
```

### ❌ 不要这样答
- "国内发展机会大"（太空）
- "不喜欢国外 / 没适应"（消极）
- "听父母的"（不主动）
- "看不上国外公司"（傲慢）

### ✅ 加分细节
- 提具体公司 / 具体产品 / 具体技术（证明你做过功课）
- 把海外经历变成"我能带回来的东西"（不是简单"我回国了"）

## 六、Why this 公司 / 岗位 模板

```
3 个具体理由（每个理由 1 段）：

理由 1 · 公开可查的事实
"贵公司最近 [Q1 财报 / 新产品 / 战略调整]，我对 [具体方向] 很感兴趣
 因为 [我自己经历里的连接]"

理由 2 · 团队 / 价值观
"我读了 [某位 senior 在 LinkedIn / 公众号 / 知乎] 提到的 [某个观点]，
 这和我在 [我自己经历] 里学到的 [XX] 完全一致"

理由 3 · 个人长期方向
"我未来 3-5 年想在 [sector / 方向] 深耕，贵公司 [独特位置 / 资源]
 是少有的能让我在这条路上走深的环境"
```
