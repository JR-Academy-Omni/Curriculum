> 配套 L01 三轮制时间窗 · 留学生回国求职**第一件必办的事** · 错过认证 = 错过秋招

## 二、5 件认证材料齐全检查

- [ ] **护照首页 + 签证页**扫描清晰 · 主页 + 签证 + 出入境章页都拍
- [ ] **学位证 / 毕业证**外文原件 + 中文译本（留服合作机构译）
- [ ] **成绩单**外文原件 + 中文译本（最近一次成绩单 + 总成绩单）
- [ ] **学校录取通知书**（部分国家学位证不显示入学时间 · 需要它）
- [ ] **出入境记录**（公安局 / 移民局打印 · 显示在外停留时间）

### ⚠️ 自译 vs 留服合作翻译机构
- ❌ **自译**：不接受 · 必须公证 / 留服合作机构译
- ✅ **留服合作**：通过 lvyou.cscse.edu.cn 关联翻译机构（¥80-200 / 份）

## 三、认证申请 6 步操作

### Step 1 · 注册账号（1 天）
- URL：**zwfw.cscse.edu.cn**
- 教育部留学服务中心 · 实名认证
- 邮箱必须真实可收信（垃圾箱定期查 · 系统邮件易被拦）

### Step 2 · 在线填申请表（1-2 天）
- 个人信息 / 教育背景 / 留学经历 / 出入境时间
- ⚠️ 学位日期 = 实际毕业日期（部分学校发证晚于毕业典礼 · 必须以**学位证**日期为准）
- 校对 3 遍再提交（一旦提交不能改）

### Step 3 · 上传材料（1 天）
- 5 件材料按要求格式上传
- PDF 限制 5MB / 图片限制 2MB
- 中文译本必须是留服合作机构出的（带翻译机构盖章）

### Step 4 · 在线缴费（实时）
- 标准费用：¥360 / 套
- 支持微信 / 支付宝 / 银行卡

### Step 5 · 核查 + 认证（10-20 工作日 · 最长环节）
- 系统自动核查 + 部分需要人工
- **部分国家需要国际合作核查**：
  - 英国：HEDD
  - 美国：Parchment
  - 这些国家可能慢 5-10 天
- 状态变化邮件通知（垃圾箱定期查）

### Step 6 · 认证书出具（1 天）
- 在线下载电子版（PDF）
- 认证书编号 → 写进简历
- 部分公司要求邮寄纸质版（额外申请）

## 四、应届身份 4 维度自检（招聘 HR 看的）

国内招聘"应届生"判定**不是单一标准** · 4 个维度同时看：

### ① 毕业时间
- [ ] 学位证日期 < 2 年（部分公司放宽 3 年）
- [ ] 不是回国日期 / 不是签证日期 · 是**学位证日期**
- [ ] 知道目标公司具体卡几年（腾讯 / 字节通常 1 年内 · 部分国央企放宽 2 年）

### ② 社保记录
- [ ] **从未缴纳过国内社保**（含个人 + 公司缴）
- [ ] 海外工作不算（海外社保不影响中国应届身份）
- [ ] 国内实习如果公司缴了社保 → 失资格（很多大厂实习不缴）
- ⚠️ 自检：登录"国家社保公共服务平台"查个人缴纳记录

### ③ 劳动合同
- [ ] **从未签过国内正式劳动合同**
- [ ] 三方协议 ≠ 劳动合同（三方协议不影响应届身份）
- [ ] 实习协议看具体性质（一般不影响 · 但部分长期实习算）

### ④ 报到证 / 派遣证（2026 取消中）
- [ ] 没在国内学校学籍系统"已派遣"状态
- [ ] 海外学位本来就没派遣证 → 不影响
- [ ] 2026 起部分省取消 · 2027 全国取消（过渡期还可能查）

### 实战判断
```
✅ 应届身份保留 = 上述 4 项全部满足
⚠️ 个别 ≠ 失资格（部分公司单独评估）
❌ 失资格 = 走社招通道（china-career hub 社招线）
```

## 五、留学生回国求职 6 件事执行清单

### T-1 月（出国前 / 学业末期）
- [ ] 准备 5 件认证材料 · 中文译本提前送翻译机构
- [ ] 注册 zwfw.cscse.edu.cn 账号
- [ ] 整理出入境记录（移民局 + 国内公安局）

### T-2 周（回国 / 毕业前）

- [ ] 准备简历 v1 + 目标 sector
- [ ] 关注目标公司 careers 页面（秋招提前批 6 月开）

### 投递期
- [ ] 简历填学历认证编号
- [ ] 网申表政治面貌 / 户籍 / 应届身份 准确填

### Offer 后
- [ ] 三方协议保留（不要弄丢）

- [ ] 档案接收信息确认

### 入职前

## 六、常见疑问 5 问

### Q1 · 我学位证 2025.07 · 但 2026.06 才办认证 · 来得及秋招吗？
**A**：来得及秋招正式批（9-10 月）· 错过秋招提前批（6-8 月）· 建议：**今晚就启动**学历认证

### Q2 · 我中文译本能自己翻吗（专业是外语 / 翻译）？
**A**：不能 · 必须通过留服合作机构（带翻译机构盖章）· 自译会被退回

### Q3 · 我之前办过一次旧版认证（多年前）· 现在还能用吗？
**A**：2018 年后改版 · 旧版不再被新版网申系统识别 · 必须重新办

### Q4 · 我有多个海外学位（本科 + 硕士）· 都要认证吗？
**A**：建议都办 · 但秋招重点办**最高学历**（硕士）· 本科有时间再补

### Q5 · 我在英国 / 美国双学位项目 · 怎么办？
**A**：双学位通常需要分别在两国办认证（费用 × 2）· 部分中外合办项目可一次办（看学校）
