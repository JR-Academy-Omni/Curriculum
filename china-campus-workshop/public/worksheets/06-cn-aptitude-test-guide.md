## 二、行测 5 大题型 + 备考路径

### 题型分布（典型 90 分钟 120 题）

| 模块 | 题数 | 时长 | 难度 | 留学生主要失分点 |
|------|------|------|------|----------------|
| **言语理解** | 30-40 | 25 min | ⭐⭐⭐ | 成语 / 古文 / 中文表达细微差别 |
| **数量关系** | 10-15 | 20 min | ⭐⭐⭐⭐ | 数学公式记不住 / 中文应用题 |
| **判断推理** | 30-40 | 30 min | ⭐⭐⭐ | 逻辑题 / 图形推理 |
| **资料分析** | 15-20 | 25 min | ⭐⭐⭐ | 计算速度 / 中文数据表 |
| **常识** | 15-20 | 5 min | ⭐⭐⭐⭐⭐ | **中国国情 / 政治 / 历史 / 文化** 留学生最弱 |

### 备考资源（按 ROI 排序）

1. **粉笔 APP**（必装 · 免费 + 付费）— 行测 / 申论刷题主力 · 题库最全
2. **公考雷哥**（系统课）— 言语 + 判断 拆解最深
3. **B 站 UP 主**：李梦娇（常识）/ 花生十三（数量关系）/ 老 K（资料分析）
4. **小红书 / 知乎**："留学生考公" + "行测 80+" tag

### 行测 70+ 通关路径（4 周）

```
Week 1：基础刷题 · 各模块 30 题 / 天 + 看错题 · 找弱项
Week 2：弱项突破 · 重点刷数量 + 常识 + 资料分析
Week 3：套题模拟 · 1 套真题 / 天 + 严格 90 分钟
Week 4：复盘 + 提速 · 错题二刷 + 蒙题技巧
```

## 三、申论备考（国央企 / 公考必)

### 题型
- **材料概括**（500-800 字）· 找重点
- **综合分析**（800-1200 字）· 多角度阐述
- **文章写作**（1500-2000 字）· 议论文 / 应用文

### 留学生 3 大坑
1. **古文 / 引用**：申论好文章必须用古文 + 中国典故（备 5-10 个万能引用）
2. **政治术语**：习近平思想 / 五位一体 / 四个全面 等必备（背 30 个高频术语）
3. **行文逻辑**：中文议论文"总分总" · 不是英文 "argue + counter-argue"

### 备考资源
- 《半月谈》杂志（政论文模板）
- 粉笔申论批改服务
- B 站 "石磊申论" / "李梦娇申论"

### 30 天通关路径
```
Week 1：背高频引用（孟子 / 老子 / 王阳明 / 鲁迅 ×5）+ 政治术语 30 个
Week 2：精读 5 篇人民日报评论 + 拆解结构
Week 3：模考 3 篇 · 找人批改
Week 4：再模考 3 篇 + 提速到 180 min 内完成
```

## 四、北森 / iTalent（性格 + SJT · 不能刷）

### 题型
- **性格测试**（200-300 题 · 同一性格 5-10 道反向问）
- **SJT 情境判断**（30-50 题 · 工作场景四选一）
- **行为面 vidoe**（部分公司 · 30 秒思考 + 2 分钟答）

### 国央企 / 互联网偏好的"性格 profile"

| 维度 | 国央企偏好 | 互联网偏好 |
|------|----------|----------|
| 风险偏好 | 低 · 稳健 | 中 · 平衡 |
| 决策风格 | 团队协商 | 数据驱动 |
| 沟通风格 | 圆滑 + 服从 | 直接 + 透明 |
| 学习能力 | 中 + 经验主义 | 高 + 主动 |
| 客户优先 | 配合上级 | 用户至上 |

### 5 大不能踩的坑
1. **性格反向题不一致**（同一维度 5-10 道反向问 · 矛盾会被识别）
2. **极端选项扎堆**（全选"非常同意 / 非常不同意" → 异常）
3. **SJT 选"个人英雄"**（应该选"团队协商" / "汇报上级"）
4. **答得太快**（< 25 分钟答完 200 题 = 异常）
5. **多次重做**（系统检测重复 IP + 短时间多次提交）

### Pymetrics（中国版 · 部分公司用）
- 12 个 mini-game · 测决策风格
- ❌ 不能刷 · 系统检测刻意行为
- ✅ 唯一准备：提前体验官方 sample（aristotle.pymetrics.ai）

## 五、牛客 / 赛码（互联网大厂编程笔试）

### 题型
| 类型 | 题数 | 时长 | 通过率 |
|------|------|------|--------|
| **算法编程** | 2-4 题 | 60-90 min | 30-50% |
| **选择题** | 30-50 | 30-45 min | 70-90% |
| **数据库 SQL** | 1-3 题 | 30 min | 50-70% |
| **系统设计**（部分公司） | 1 题 | 60 min | 30-40% |

### 算法重点（按出现频率）
1. **数组 / 字符串**（最常考）· 双指针 / 滑动窗口
2. **二叉树**（DFS / BFS / 二叉搜索树）
3. **动态规划**（背包 / 编辑距离 / 最长子序列）
4. **图论**（BFS / DFS / Dijkstra）
5. **回溯**（全排列 / N 皇后）

### 备考路径（已学过算法 · 30 天通关）
```
Week 1：LeetCode Hot 100 刷 30 题（中等难度）
Week 2：再刷 30 题 + 看大厂面经
Week 3：模拟 10 套笔试（真题 / 牛客模拟）
Week 4：复盘 + 提速 + 反作弊练习（开摄像头）
```

### 备考资源
- **LeetCode**（中国版 · 题量最大）
- **NowCoder 牛客网** · 真题题库 + 编程平台
- **代码随想录**（公众号 + 知乎）· 算法系统讲解
- 《剑指 Offer》 · 经典题集（必读）

- 编程（牛客系统）+ 智力题 + 性格

### 美团 · 美团笔试

### 拼多多 · 拼多多笔试
- 编程为主（赛码）+ 数据分析
- 题量大 / 时间紧（90 min 4 题）

## 七、考前 24 小时准备清单

### 环境
- [ ] 网络稳定（≥ 20Mbps · 不要用 wifi 高峰期 · 优先有线）
- [ ] 电脑（笔试不能用手机 / iPad）· 测好摄像头 + 麦克风
- [ ] 关闭微信 / QQ / 浏览器通知 · 全屏模式
- [ ] 准备纸笔 / 草稿纸（行测算资料分析用）
- [ ] 杯水 / 不要重餐（90-120 分钟不能上厕所）

### 反作弊摄像头要求（牛客 / 北森 等）
- ⚠️ **摄像头必须开**（部分公司要求 2 个摄像头 · 一个看屏幕一个看周围）
- ⚠️ 不能开第二屏 / 第二浏览器（被检测 = 自动作废）
- ⚠️ 不能离开摄像头视野超过 30 秒
- ⚠️ 不能有第二个人在房间（被检测 = 自动作废）

### 时间选择
- 早上 9-11 点 / 下午 2-4 点（精力最好）
- 避免 21 点 - 23 点（疲劳期）

### 心态
- 第一题做不出来不慌 · 跳过 · 后面回头看
- 数量关系会一道是一道 · 不会的猜（不留空）
- 资料分析最容易出分 · 优先做
