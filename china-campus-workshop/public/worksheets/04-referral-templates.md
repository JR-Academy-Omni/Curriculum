## 一、内推 ≠ 走后门（必读）

### 国内内推真实运作
- **85% 留学生没有真实可达校友 channel**（不是没认识人 · 是认识的人不在目标公司 / 不愿 push）
- 内推优先级 > 普通网申（HR 优先看 + 反馈快 + 笔试 / 面试通过率高）
- 内推**不保 offer**（HR 决策权不在推荐人）

### 找谁内推 4 层优先级

| 优先级 | 对象 | 触达方式 | 回复率 |
|--------|------|---------|--------|
| ⭐⭐⭐ | 同校 / 同专业校友（在目标公司 Grad / Associate） | 微信 / LinkedIn | **40-60%** |
| ⭐⭐ | 海外实习同事（已回国进目标 sector） | 微信 / 邮件 | 30-50% |
| ⭐ | 行业前辈（公众号 / 知乎 / 不直接认识） | LinkedIn / 知乎私信 | 15-25% |
| ❌ | 直接找 VP / 总监 / 老板 | — | < 5% |

## 二、微信邀约 3 模板

### 模板 A · 同校校友（强 connection）

```
你好 [姓名] 学长 / 学姐，

我是 [大学] [专业] 2026 届 [毕业月份] 的 [你名字]
看你 LinkedIn / 校友群知道你在 [公司] 做 [岗位]
2026 届 [秋招 / 春招] 我准备投 [公司] [目标岗位 / sector]

冒昧加好友 · 想麻烦你 [10-15 分钟] 聊一下：
① 你在 [公司] 这 [N] 年最大感受 / 最大坑
② [岗位 / sector] 一般招什么样背景的人
③ 内推码 / 网申建议

简历附上 · 我自己 3 句话介绍：[1 行教育 + 1 行实习 + 1 行 cohort 优势]

打扰了 · 不论是否方便都谢谢！
```

### 模板 B · 海外实习同事（已回国 · 弱信任）

```
[姓名] · 还记得吗 · 我是 [实习公司] 那段时间一起在 [项目] 的 [你名字]

最近回国找 2026 届 offer · 看你 LinkedIn 已经回国进了 [公司]
有空 [10-15 分钟] 聊一下吗？
① [公司] 2026 届校招开了吗 / 你那条线在招吗
② 你看我 [简历附上] · 你觉得我有没有戏 / 投哪条线

我请你喝奶茶 · 线上线下都行 · 你方便就好
```

### 模板 C · 行业前辈（弱 connection · 不认识）

```
[姓名] · 你好

我是 [大学] [专业] 2026 届的 [你名字] · 关注你 [公众号 / 知乎 / LinkedIn] 已经 [N] 个月
你 [某篇文章 / 某个 post] 提到的 [具体点] 启发很大

冒昧联系 · 我准备 2026 [秋招 / 春招] 投 [sector] 方向
你能不能给个 [10 分钟] · 我请教你 1 个具体问题：
[具体问题：不是"你帮我内推吗"·  是"你觉得 [sector] [具体方向] 应该重点准备什么"]

简历附上仅供参考 · 不期待立即回复 · 谢谢看完！
```

### 微信 5 个 NG

| ❌ NG 行为 | 后果 |
|----------|------|
| 没自我介绍直接发简历 | 拉黑 |
| 上来就要"帮我内推" | 拉黑 |

## 三、邮件正式版（适合 HR / 部门 leader / 外企）

### 主题
```
[内推请教] [姓名] · [大学] · 2026 届 [岗位]
```

### 正文（4 段）

```
尊敬的 [先生 / 女士]，

【段 1 · 自我介绍 1 句】
我是 [大学] [专业] 2026 届 [毕业月份] 的 [你名字]
通过 [校友会 / 同事 / LinkedIn] 了解到你在 [公司] [团队 / 岗位]

【段 2 · 联系原因 2 句】
2026 届 [秋招 / 春招] 我准备投 [公司] [目标岗位 / sector]
个人方向：[本科 / 硕士主修方向] + [N] 段实习（[公司 1] / [公司 2] / [公司 3]）
专长：[2-3 个关键技能 / 工具]

【段 3 · 具体请教 / 请求 2-3 句】
冒昧致信请教 [1-2 个具体问题]：
① [问题 1]
② [问题 2]

【段 4 · 简历附上 + 礼貌结尾】
随附简历（中英双版本）仅供参考 · 期待你回复
不论是否方便 · 谢谢看完！

此致
敬礼

[你名字]
[手机] / [邮箱] / [LinkedIn]
[日期]
```

### 邮件 vs 微信 选择
- **邮件**：外企 / 国央企 / 银行 / 不熟的人
- **微信**：互联网 / 同校校友 / 已经认识的同事

## 七、内推话术 6 大红线

- ❌ 直接要"帮我内推"（应该问"建议怎么准备 + 如果方便能否推荐"）
- ❌ 抱怨之前公司 / 之前面试
- ❌ 问对方薪资具体数字
- ❌ 一次 chat 超时（30 分钟到了主动收）
- ❌ 索取手机号 / 微信（让对方决定）
- ❌ 约第二次（除非对方主动提）
