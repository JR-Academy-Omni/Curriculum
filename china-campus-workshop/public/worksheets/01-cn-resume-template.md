## 一、中文简历 1 页结构（7 段）

```
┌─────────────────────────────────────────────────────┐
│  [姓名]                              [应届 / 海归]    │
│  📧 邮箱 · 📱 手机 · 📍 所在地 · 🔗 LinkedIn         │
├─────────────────────────────────────────────────────┤
│  教育背景 (2026.09 - 2027.07 预计毕业)               │
│  ─ [学校] · [专业] · GPA / 排名 · 学历认证编号        │
│                                                     │
│  实习经历 (按时间倒序 · 3-4 段)                      │
│  ─ [公司] · [岗位] · STAR 中文 bullet × 3-4 条       │
│                                                     │
│  项目经历 (海外学校项目 / 个人开源)                  │
│                                                     │
│  校园经历 (学生干部 / 社团 / 奖项)                   │
│                                                     │
│  技能证书 (英语 + 行业证书 + 工具)                   │
│                                                     │
│  自我评价 (3-4 句 · 不空洞)                          │
└─────────────────────────────────────────────────────┘
```

## 二、海归特色字段（国内简历独有）

### 必填
- **学历认证编号**（教育部留学服务中心）· 写在教育背景"备注"或紧跟学校名
- **英语水平**（雅思 / 托福 / GRE 具体分数 · "良好" 不够）
- **政治面貌**（党员 / 团员 / 群众）· 国央企 / 公考必看
- **籍贯 / 户籍**（国央企部分要求 · 互联网可不写）

### 选填（按 sector）
- 海外实习公司用国内对标说明（"相当于国内 [XX 公司]"）
- 海外学校 QS 排名 / 国内对标（"墨大 QS Top 14 ≈ 国内 985 中等"）

## 三、5 sector 版本差异

| Sector | 简历重点 | 党员 / 学生干部 | 海外经历包装 |
|--------|---------|----------------|------------|
| **互联网大厂** | 技术 + 业务能力 + 项目深度 | 选填 · 不强调 | 视野优势 · 全球团队协作 |
| **国央企** | 学校 + 政治面貌 + 稳定性 | **必写** + 学生干部加分 | 弱化 · 突出本土学校 + 实习 |
| **银行金融** | CFA / FRM + 数据分析 + 风控 | 写但不强调 | 海外金融硕士有加成 |
| **外企管培** | 英文能力 + 跨文化 + Leadership | 不写 | **强强调** · global mindset |
| **公考 / 选调** | 学校 + 党员 + 政治素质 | **必写** | 弱化（部分省限制海归） |

## 四、STAR 中文 Bullet 改写公式

### 公式
```
[强动词] + [具体做了什么] + [量化数字 / 工具 / 范围]
        + [结果 / 影响 / 奖项]
```

### 改写对照

| ❌ 原 bullet | ✅ STAR 改写 |
|------------|-----------|
| 负责数据整理 | 主导整理 200+ 客户数据 + Excel 自动化（节省 6h / 周）· 报告被采纳进总部季度复盘 |
| 参与社团活动组织 | 牵头组织 800 人新生节 · 协调 12 个赞助商 + 4 个学校部门 · 满意度 4.7 / 5 · 获校优秀学生组织奖 |
| 在 EY 实习 | 进 EY 北京 Audit · 跟进 3 家 A 股上市公司年报审计 · 完成 2 个底稿章节获 Senior 优评 |
| 写论文 | 主笔某细分赛道 8000 字 paper · 中文核心期刊在审 · 数据采样 300+ 样本（含问卷 + 深访） |
| 完成项目 | 4 人小组完成新能源车 LCA 全周期碳排建模 · 数据获导师认可推荐至业界会议海报展 |

### 5 个 sector 高频动词

- **互联网**：主导 / 设计 / 上线 / 优化 / 迭代 / 推动 / 落地 / A/B 测试 / 数据驱动
- **国央企**：牵头 / 协调 / 推进 / 完成 / 配合 / 落实 / 调研 / 编制 / 上报
- **银行**：跟进 / 监控 / 评估 / 测算 / 建模 / 风控 / 合规 / 复盘
- **外企**：Led / drove / coordinated / partnered with / delivered（中英混搭）
- **公考**：调研 / 协助 / 拟稿 / 落实 / 推进 / 服务 / 保障

## 五、自我评价（最容易翻车的一段）

### ❌ 不要写
- "本人勤奋好学 / 沟通能力强 / 团队合作精神好"（空洞）
- "希望加入贵公司发挥才能"（套话）
- 长段落 3-5 行（HR 不看）

### ✅ 这样写（3-4 句 · 每句 1 个具体点）
```
[专业方向] 应届 · 海外 [学校 / 学历] · 雅思 7.5 / 党员
[1-2 段真实实习] + [1 段校园 Leadership] 经历可复用
长期关注 [目标 sector] [具体细分方向]，自学 [工具 / 证书]
求职意向：[城市] · [岗位] · 可接受 [出差 / 加班 / FIFO]
```
