## 一、中文 STAR 4 维度（与英文版差异）

```
┌─────────────────────────────────────────────────────┐
│  情境 (S) 20% → 任务 (T) 10% → 行动 (A) 50% → 结果 (R) 20%  │
└─────────────────────────────────────────────────────┘
```

### 中英 STAR 4 大差异（留学生必看）

| 维度 | 英文 STAR | 中文 STAR | 差异原因 |
|------|----------|----------|---------|
| **S Situation** | 一句话场景 | 一句话场景 + 团队 / 角色明示 | 中文 HR 先看你"在哪个团队什么角色" |
| **T Task** | 描述任务 | 任务 + 衡量标准 | 中文偏好"可量化目标" |
| **A Action** | "I led / I drove" 主语清晰 | **"我"绝对不能换成"我们团队"** | 中文 HR 比英文更敏感（爱说"我们"= 评分扣 8-12 分） |
| **R Result** | 数字 / 百分比 / 影响 | 数字 + **延伸影响**（"被采纳"/"被复用") | 中文偏好"留下持续影响" |

## 二、中文 STAR 改写公式（直接套）

### 三段式公式
```
S：[时间 / 地点 / 角色] · [背景一句话]
T：[要解决什么 + 衡量标准]
A：[强动词] + [我做了什么] + [工具 / 方法] + [我和别人不同的地方]
R：[数字结果] + [延伸影响] + [可选：学到了什么]
```

### 5 个 sector 高频中文动词

| Sector | 高频强动词 |
|--------|----------|
| **互联网** | 主导 / 设计 / 上线 / 推动 / 优化 / 迭代 / A/B 测试 / 数据驱动 |
| **国央企** | 牵头 / 协调 / 推进 / 完成 / 落实 / 编制 / 调研 / 上报 |
| **银行 / 金融** | 跟进 / 监控 / 评估 / 测算 / 建模 / 风控 / 合规 / 复盘 |
| **外企** | 中英混搭：led / drove / coordinated / partnered with（口语中文翻译） |
| **公考 / 选调** | 协助 / 拟稿 / 服务 / 保障 / 推进 / 调研 / 上报 |

### ❌ 中文弱动词（不要用）
- 负责 / 协助 / 参与 / 进行 / 处理 / 帮助 / 配合 / 学习

## 四、党员 / 学生干部 / 海外经历 5 sector 差异化转化

### 党员 / 学生干部 怎么处理

| Sector | 党员怎么写 | 学生干部怎么写 | 海外经历怎么写 |
|--------|-----------|--------------|---------------|
| **互联网大厂** | 不强调 · 选填 | 写但不渲染 | **强强调** · 全球视野 |
| **国央企** | **必写**（红字）· 转正日期写清 | **必写**详细 · 学校规模 + 团队大小 + 具体活动 | 弱化 · 强调本土实习 |
| **银行 / 金融** | 写但不渲染 | 学生会 / 商赛获奖加分 | 海外金融硕士 + CFA / FRM 强调 |
| **外企管培** | **不写**（外企忌讳）/ 写"团员" | 写但用英文表述 | **强强调** · 跨文化协作 |
| **公考 / 选调** | **必写** + 政治学习经历加分 | **必写** + 基层调研 / 志愿服务 | 部分省限制 · 弱化 |

### 海外经历"国内化"3 技巧

1. **对标说明**：海外公司 → 对标国内同业（"X = 国内 Y 公司同类业务"）
2. **量级翻译**：英文表述 → 中文量化（"global team" → "8 国 12 人协作"）
3. **影响延伸**：海外项目 → 强调"可迁移"（"项目方法可应用到国内 [XX] 场景"）

## 五、自查 5 问（每条 bullet 都过一遍）

每写完一条 STAR bullet，自问：

1. **主语是"我"还是"我们 / 团队"**？ — 必须是"我"
2. **有具体数字吗**？ — 没有就硬挤（¥ / % / 人数 / 时长 / 频率）
3. **强动词够强吗**？ — 替换弱动词
4. **海外经历"国内化"了吗**？ — 对标 / 量级 / 影响
5. **延伸影响写了吗**？ — "被采纳" / "被复用" / "被推荐" 等
