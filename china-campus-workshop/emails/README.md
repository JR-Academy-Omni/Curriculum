## 📋 落地操作（运营 / SendGrid 后台）

1. **8 封模板创建** — 复制以上 Subject + Body 到 SendGrid 模板，变量化 `[名字]` `[Zoom 链接]` `[折扣码]` 等
2. **触发器**：
   - D0 / D-1：购买后 + cron 检测直播日期
   - D+0：直播结束 webhook 触发
   - D+1 / D+3 / D+5 / D+7：D+0 后 N 天定时触发
3. **折扣码后台开**：
   - WS300（D+1，24h 有效 · ¥300 off · 限 8 人）
   - INT500（D+3，48h 有效 · ¥500 off · 限 6 人）
4. **退订机制**：每封邮件底部必须有"不再接收升级邮件"选项（合规）
5. **群运营节奏**：
   - 每天 9am 助教速答（10 min 速答 5 个高频问题）
   - 每周三晚发"本周内推岗位汇总" 5-10 个（Rose 池 + 校友推荐）
   - 每周五晚发"学员 before/after 案例" 1-2 个
   - 每月最后周六 Steven 嘉宾分享 30 min
