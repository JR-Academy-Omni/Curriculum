## 1. 课程定位摘要

> 🧑 目标用户画像见 [./PERSONAS.md](./PERSONAS.md)（2026-06-17，🚧 AI 推断版未验证）

**One-liner**：
- 中文：16 周亲手做 3 个能上线的全栈项目，补齐简历最缺的"本地项目 + 团队协作"
- English：Build 3 deployable full-stack projects in 16 weeks — the portfolio your résumé is missing

**目标人群**（来自 PERSONAS.md，⚠️ 待验证）：
- A · 缺本地经验的新移民开发 — 核心 ~35% ⚠️
- B · 无项目作品集的应届/在校生 — 核心 ~35% ⚠️
- C · 想转码的 Job Changer — 核心 ~20% ⚠️
- ❌ 不是目标：纯零基础小白（去 ai-programming）/ 只想刷 LLM 的人（去 ai-engineer-bootcamp）

**关键卖点**（来自 outline.json highlights / features）：
1. **3 个能上线的全栈项目**：2 个个人项目 + 1 个 Agile 团队商业项目（BA / DevOps / UI Designer 一起协作）→ 写进简历能讲清楚的真东西
2. **一次费用学双方向**：React + .NET 和 React + Node.js 两条线，对应不同公司/岗位的 JD
3. **完整工程链路**：前端 React/Redux → 后端 → 数据库 → AWS 部署 → Git/CI/CD → Agile 敏捷流程，从 0 到上线
4. **求职服务托底**：80+ Career Coaching 自学课 + 12 个月简历内推 + 校友网络
5. **课程成熟**：已开 30 期，FLAG / 澳洲大厂资深专家授课

**主推内容角度**（Type 1 求职 playbook Top 3 × 全栈项目特色，⚠️ persona 待验证后才能定原话）：
1. ⭐⭐⭐⭐⭐ 校友真实 offer + 带项目作品的上岸案例（社会证明 >>> 功能介绍）
2. ⭐⭐⭐⭐⭐ "做出来的真实项目 demo + 上线 URL + 从 0 到部署过程"（破除"我简历没东西可写"的核心痛点）
3. ⭐⭐⭐⭐ "16 周全真团队项目（跟 BA/DevOps/UI 协作）" — 这是新移民 A / 应届 B 简历上最缺、别家速成班给不了的差异点

**承诺红线**：❌ 不写"包就业 / 保 offer / 月入 / 12 月内推保证进面"；✅ 只承诺"做出 3 个能上线的全栈项目 + 完整作品集 + 团队协作经验 + 求职准备度"。
