P21 FAQ · 常见问题
报名前问

报名前
最常被问的 5 个问题

零基础能学吗？
能 — 第 1-3 周就是从 HTML / CSS 开始打底。但建议先熟悉基本编程概念。
没有澳洲身份可以学吗？
可以 — 课程本身没身份要求。但 AI 实习部分需要有合法工作权限（PR / 学签工签都行）。
选哪个后端方向？
零基础选 Node.js（同栈 JS）/ 想进大公司选 .NET / 主攻 AI 选 Python。课上有详细选型咨询。
和网上课程区别？
网上课 = 教程跟做；这里 = 真团队协作（BA/DevOps/UI）+ AI 实习 + 12 个月辅导，不是录播合集。
第 30 期什么时候开？
2026 Q2+，正式大纲筹备中。可以先预习 26/27/28 期 485 节老内容。
