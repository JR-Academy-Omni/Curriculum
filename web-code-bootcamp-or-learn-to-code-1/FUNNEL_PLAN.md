## 3. 主课三档（按"做出/上线什么项目"分级，⚠️ 待 /main-course-tier-design + PERSONAS 验证后定稿）

> **分级铁律（本课特色）**：不按"内容多少 / 周数"分档（16 周内容三档共用），按 **"你最终做出并上线了什么项目、拿到什么级别的作品集 + 求职服务"** 分级。这正好对应 PERSONAS 三类人不同的预算和目标。

| 档位 | 建议定价（AUD / CNY，⚠️ 推断待核）| 一句话承诺（按项目交付分级）| 关键差异点 | 对应 persona |
|------|------|------|------|------|
| **自学版** | ~$1980 AUD / ~¥6700 | "跟着录播 + 社群答疑，**独立做出 2 个个人全栈项目并部署上线**" | 录播全开 + 社群答疑 + Lab/Quest 自动校验，**无 Agile 团队项目、无 1v1 内推** | B 预算紧的学生 / 想先低成本试水的人 |
| **教学版** | ~$3680 AUD / ~¥12500 | "16 周跟着直播做，作业批改 + **完成全部 3 个项目（含 Agile 团队商业项目）**" | 直播 + 作业批改 + 进 Agile 团队项目（与 BA/DevOps/UI 协作）+ 班级群运营 | A 新移民 / C 转行（要团队经验）|
| **陪跑版** | **$5360-5900 AUD**（≈当前定价）/ ~¥18000-20000 | "教学版全部 + **12 个月简历内推 + 1v1 求职辅导，直到拿出能投简历的完整作品集 + 准备好上岸**" | 教学版全部 + 80+ Career Coaching + 12 月内推 + 简历/面试 1v1 + 校友网络 | A/C 盯求职结果、愿为"全套就业服务"付费的人 |

> 🚨 **重要现实**：当前 $5360 单档 ≈ 上表"陪跑版"（含内推 + Career Coaching）。是否真要拆出更便宜的自学版/教学版，取决于：
> 1. PERSONAS 验证后看 B（学生）占比是否真的高到需要一个更低价入口
> 2. 运营 bandwidth（自学版必须有完课率保障，否则口碑塌）
> 3. 别跟 ai-programming（$860-1360）抢低价市场 → 本课定位是高客单价求职服务，自学版别压太低
> **建议：先不急着拆三档，先验证「单档 + 引流课」（§4 选项 B）跑顺，再看要不要加自学/教学档。**

**承诺红线**：❌ 不写"包就业 / 保 offer / 月入 X / 12 月内推保证进面"；✅ 只承诺**过程结果**："做出 3 个能上线的全栈项目 + 完整作品集 + 团队协作经验 + 求职准备度（简历/面试/内推机会）"。

## 4. 引流课（⚠️ 当前无 · 推荐做 · 待 /intro-course-design 填）

### 当前漏斗缺口

```
潜在用户进来
    ↓
看销售页（$5360 AUD 单档高价）
    ↓
❌ 没有低门槛入口 → 大量"先了解再决策"型用户（尤其学生 B / 谨慎的转行 C）直接流失
    ↓
只有少数决心已定的人会直接咨询
```

→ **关键发现**：$5360 + 16 周 = 决策成本极高。**没有 ¥99 / $29 引流课承接"想先试试做个项目"的人**，高价单档把犹豫期用户全挡在门外。

### 推荐引流课（形态 1 · 单引流课）

| 字段 | 内容 |
|------|------|
| 主题 | **《3 小时从 0 做出并上线你的第一个全栈小项目》**（前端 + 一个 API + 部署到云，能拿到一个真实 URL）|
| 为什么能 work | 本课 Phase 0 的 HTML/CSS/JS/React + "GitHub Action CICD 自动部署上线" + "S3 静态托管" 已是现成 lesson，**拆出来 = 0 新内容生产**，1-2 周可上 |
| 时长 | 3 小时（1-2 节直播 + 1 个 Lab + 当晚交付一个上线 URL）|
| 定价 | ¥99 / $29 AUD（多币种成对）|
| Quick win 交付物 | **一个能打开的真实网址**（学完当晚就有，能截图发朋友圈）— 对"没作品集"的 B/C 是强 hook |
| 学完承诺 | "你已经独立部署上线了第一个项目；主课带你做 3 个能写进简历的完整全栈项目" |
| 在主课大纲中的对应位置 | Phase 0（HTML/CSS/JS/React 基础 + GitHub Action 部署 + S3 托管）|
| 转化逻辑 | 引流课让人体验"我也能做出能上线的东西" → 自然导流到"想做更像样、能写简历的 3 个项目" = 主课 |

### 推荐：选项 B（引流课 + 现有单档主课，最小改动）

```
¥99 / $29 引流课《3 小时做出并上线第一个全栈小项目》
       (Phase 0 拆出来，1-2 节直播 + 1 个 Lab，当晚交付一个真实 URL)
                    ↓
            7 天过渡（不推销，每日 1 个校友项目拆解 + 1 个 SOP）
                    ↓
        $5360 AUD 主课（30/31 期）— 3 个项目 + 内推 + Career Coaching
```

**预期数据**（⚠️ 行业基线推测，第 1 期跑完用真实数据替换）：
- 引流课页面 UV → 报名转化率 ~30%
- 引流课报名 → 加微信群 ~60%
- 微信群 → 主课转化率 ~10%（高客单价 $5360，转化率比低价课低）

→ 这套靠的是**把高价课的决策周期拉长成"先 ¥99 体验 → 7 天看见差距 → 再决定 $5360"**，而不是逼人一上来就掏 5000+。

## 5. 7 天过渡序列（引流课 → 主课 · 待 /funnel-handoff-design 填）

> 详细见 `/funnel-handoff-design` 子 skill，本节是上层骨架。

| Day | 内容 | 目的 | 形式 |
|-----|------|------|------|
| D0 | 引流课报名确认 + Day 1 URL + 微信群入口 | 接住 | 邮件 + 群通知 |
| D1-D3 | 引流课正课 + Lab，**当晚做出一个能打开的网址** | 完课体验 + quick win | 平台学习 |
| D4 | 「26-30 期校友做了什么项目」案例拆解（带项目 demo + 进了哪家公司）| 触发想象 | 公众号 / 朋友圈 |
| D5 | 引流课 KEEP / GAP 分析（你做出了一个小项目 / 你简历还缺 3 个能讲清楚的完整项目 + 团队经验）| 让用户看见差距 | 群内分享 + 1v1 答疑 |
| D6 | 「自学 vs 跟着做 vs 全套求职服务」的选择参考 + 下期开课日 + 早鸟剩余名额 | 决策窗口 | 公众号 + 微信群 |
| D7 | 主课早鸟截止（"下期开课前 X 天"压力点）| 临门一脚 | EOI 收单 |

**红线**（参考 funnel-handoff-design）：
- ❌ 不推销 / 不群发课程链接 / 不假装"个人推荐"
- ❌ 不承诺"学完保 offer / 12 月内推保证进面" — 本课承诺是"做出作品集 + 团队经验 + 求职准备度"
- ✅ 只服务（答疑 / 校友项目拆解 / 部署 SOP），让用户自己看见"我现在能做小项目，但简历要的是 3 个完整项目"的差距
