💬 学员高频问题预案（Q&A）
Q1（装机相关）: 我电脑装不上 Claude Desktop App，单位 IT 锁了权限。怎么办？
A: 先承认这是单位策略不是你的问题。然后 3 选 1：(a) 助教带去 Codex CLI fallback 路径，npm 装 codex CLI 走终端跑 prompt — 4 小时下来你能跟上 L03/L04；(b) 用今天的备用电脑（协会会备 2-3 台）；(c) 用你私人手机连热点 + 借助教笔记本演示一遍。三选一都至少能完整经历今天的实操，不让你白来。回去后想在公司机器上跑 Cowork 是你和单位 IT 之间的事，今天我们不介入。
Q2（装机相关）: Cowork 装好了但 Filesystem Connector 一直读不到 sample-data.csv，提示路径无权限。
A: Settings → Connectors → Filesystem → 看授权目录列表。常见 3 错：① 你把 csv 放到了授权目录之外（比如放到 Downloads，但 Connector 只授权了 Documents）— 把 csv 移到授权目录里；② Mac 系统弹窗『允许访问 Documents』那时候点了拒绝 — 系统设置 → 隐私与安全 → 文件夹访问权限重新授权；③ 路径打错（中文目录名 / 空格） — 用纯英文目录名。助教 30 秒到位帮你排查。
Q3（装机相关）: Cowork tab 看不到，左侧导航只有 Chat 和 Projects。
A: 99% 是版本问题。退出 Claude Desktop App，重新去 claude.ai/download 下载最新版（Cowork 是较新功能）。极个别区域 Cowork 还没开放（亚洲个别国家），这种情况助教带去 Codex fallback。
Q4: 我们公司禁止用任何 AI 处理财务数据，今天学的回去也用不了？
A: 三层回应。① 你今天学的方法论 + Prompt 模板 + 5 个 Cowork Skill 永久有效，等政策放开就能用；② 整节课用脱敏 sample-data 演示，方法本身可在脱敏后用 — 怎么脱敏 + 怎么对接 Xero/MYOB/QB 真实数据看《AU 数据准备指南》(au-data-prep.html)；③ Cowork 的 Filesystem Connector 设计本身就比网页版聊天安全（数据在你电脑没上云）。具体到你单位的合规审查规则不在本课范围 — 那是你和单位合规部门 / 你单位 CPA/CA 之间的事。
Q5: 我刚才让 Cowork 分析数据，它给的结论跟我经验不符，到底信不信？
A: 不信。Cowork 是『放大镜』不是『裁判』 — 它把你看到的角度放大，不替你做判断。你的经验告诉你不对，多半是 (a) 你看到了它没看到的口径细节 (b) 它从相关性瞎推因果。回去把你的判断写进 prompt（用 prompt② 异常归因 push back）让它再跑一遍，看它怎么承认。这是模块 4 讲的 3 类典型错误。
Q6: Cowork 用免费档每月够用吗？
A: 看你怎么用。如果只是月底跑一次月度损益分析（用 Skill 调起，5 个 prompt × ~3-5 次对话/月）— 免费档够。如果每天都用（日常对账 / 周报 / 临时查询） — Pro A$30/月（一份咖啡）划算。Pro 升级是你个人 / 单位的决定，不通过协会团购。
Q7: 课后还能找你们答疑吗？有没有学员群？
A: 没有。今天 4 小时讲完就结束 — 没有协会专属群 / 没有持续答疑 / 没有 L2/L3 续作项目 / 没有 D+21 复盘。你拿走的方法 + 5 个 Cowork Skill 永久属于你，永远可用。想继续学免费去 jiangren.com.au 平台 — Prompt Lab / Wiki / Roadmap 全部免费，自己看自己学。我们不卖『长期陪跑』，需要那个的去看 JR Bootcamp 系列（独立产品，跟今天这门课无关）。
Q8: AI 教练你能保证 4 小时学完，我下周一上班就能用吗？
A: 能用，但不会马上熟。今天你拿走的是『装好的 Cowork + 5 个 Skill + 1 份《AU 数据准备指南》 + 7 天行动卡』 — 下周第一次用还是会卡，这是正常的。模块 5 的 7 天行动卡设计就是给你这周的脚手架，AU 数据指南 §8 一页 checklist 就是给你『首次用真实数据』的 SOP。卡在哪里 → 自己 Google / 问 Cowork 自己 / 去 JR Prompt Lab 看相关 lab — 学习是你自己的事。
