Claude Cowork PRIMARY · 主战场

Anthropic 官方桌面 GUI（Mac / Windows）。Claude Desktop App 里的 Cowork tab，能直接开你电脑上的 Excel / 浏览器 / 邮箱替你动手做事 — 不是聊天，是替你完成事。本节课所有实操（L04 / L05）都在 Cowork 里跑：Cowork 通过 Filesystem Connector 直接读你电脑上的 sample-data.csv，输出 Artifact 形式的简报，再把 prompt 存成 Skill 永久复用。

课堂上免费档够用（次数受限）；课后想继续用 Pro A$30/月。Skill / Schedule 功能 Pro 才有，但今天用不到。

OpenAI Codex CLI FALLBACK

极少数学员单位 IT 不让装 Claude Desktop 时的备选。纯命令行，没 GUI，没 Skill 系统，但 prompt 能跑。L02 装机时助教单独带这条路线，确保所有学员能跟上 L04 / L05。

💻 模块 2：Cowork 装机 + 4 步使用流程（全员同步装）

AI 教练主导 · 助教 1:10 救场 00:15–01:10（55m）
4 步装机流程
STEP 1 装 Claude Desktop App claude.ai/download 下载对应系统（Mac arm64 / Mac Intel / Win x64），双击装。10 分钟
STEP 2 登账号 + 切到 Cowork tab 工作邮箱登入，左侧导航切到 Cowork。免费档今天够用，不强求 Pro。8 分钟
STEP 3 配 Filesystem + Web 两个 Connector Settings → Connectors。Filesystem 点选 ~/Documents/AI-Workshop（独立目录，不要给整个家目录）；Web OAuth 一下。Gmail / Drive 今天不配。15 分钟
STEP 4 跑 hello-world 在 Cowork 里说『读 sample-data.csv 第一行，告诉我有哪些列』。看到 6 列 + 144 行 = 装机成功，进 L03。8 分钟
救场预案
  • 助教 1:10 配比 — 协会大班装机必须有救场资源
  • 装好的截屏对照表（成功 vs 卡点）每位助教 1 份
  • 🚨 单位 IT 不让装 Desktop 的学员 → 助教单独带去 Codex CLI fallback（4 分钟），保证 L04 / L05 跟得上
产出
每位学员桌面 Cowork 装好登好 + Filesystem + Web 两个 Connector 配通 + hello-world 跑通。助教抽检通过率 ≥ 95%。
现场用 claude.ai/download ↗ Cowork 产品页 ↗ 样本 CSV ↓ Codex CLI fallback ↗ 🇦🇺 AU 数据准备指南（课后参考）→
