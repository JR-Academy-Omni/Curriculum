📖 怎么用

  1. 找到你要做海报/封面/插图的课程，比如 AI Engineer Bootcamp
  2. 点击卡片上的 📋 复制 AI 出图 Prompt 按钮（已自动复制到剪贴板）
  3. 打开 Claude / ChatGPT / Midjourney / 你常用的 AI 工具，粘贴 prompt
  4. 在 prompt 后面补充你的具体需求，例如「再加一句标题：『12 周从 0 到 AI 工程师』，画幅 3:4 小红书」
  5. AI 会按指定的主题色 + Neo-Brutalism 风格出图，海报放一起品牌识别一致

⚠️ 如果你的输出和品牌色不一致，多半是 AI 没严格遵守 hex 值——把 prompt 里 主色：#XXXXXX 那一行单独再说一次"严格使用 #XXXXXX，不要换其他颜色"通常就能纠正。

✅ 已复制 Prompt — 现在粘贴给 AI 即可
