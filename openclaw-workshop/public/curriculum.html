❓ 常见问题
完全没用过 AI / 不会编程，能来吗？
可以。唯一的前置条件是你用过 ChatGPT、Claude、豆包任意一个 — 知道『输入文字，AI 回你文字』是怎么回事就够了。OpenClaw 本身不需要写代码。
需要带什么？
① 自己的笔记本电脑（macOS / Windows 都行，Windows 需要 WSL2）；② 充电器；③ 一个 LLM API Key（OpenAI / Claude / Gemini 任一，没准备现场会发 JR 共享测试 Key 应急，不会让你空手而归）；④ 一份你想让 AI 帮你处理的真实工作素材（客户邮件、文档、PPT 草稿都行），L07 进阶 Workshop 会用到。
OpenClaw 装上跑起来要另外花钱吗？
OpenClaw 本身是 MIT 开源免费，装在你自己电脑上跑。主要成本是 LLM API Key — 你接 OpenAI / Claude / Gemini 用多少付多少，日常使用通常一个月几美元到十几美元。课上如果你没自备 Key，现场会发共享测试 Key 给你用完这 4 小时；走之后建议自己开通一个（注册入口和费率卡课上会给）。
OpenClaw 靠谱吗？是不是我们自己搞的东西？
OpenClaw 是奥地利开发者 Peter Steinberger 发起的开源项目，2025 年 11 月公开发布，MIT 协议、GitHub 超过 68,000 星、Wikipedia 有条目。不是 JR 自己的产品。JR 只是这个课整合了平台资源 + Tutor 陪跑，帮你更快上手。官方链接在"OpenClaw 官方 & 社区资源"那一栏。
我在 X 行业 / Y 城市，这节课对我有用吗？
Training 大纲是通用的，Live Demo 的行业案例会按本期 Program 定制（本期是什么方向可以看具体开课信息）。即便 Demo 的行业和你不完全一样，核心方法在你的行业里同样适用 — Networking 环节也是跨行业的好机会。
不同城市的 Workshop 内容一样吗？
Training 的 7 个模块结构全球统一。但 L04 Live Demo 和 Workshop 案例会根据当期学员构成定制（例如悉尼某期可能是房地产专场，墨尔本某期可能是保险专场）。不同城市 Program 信息会在报名页明确标注。
课后还有支持吗？
有。① 当场加入 OpenClaw 使用者微信群 — 遇到问题继续答，能看到其他同学踩过的坑；② 平台 9 章 OpenClaw 学习方向 + 16 个 Prompt Lab 持续解锁；③ 不定期 Office Hour 答疑（群内通知）。
有录播吗？错过怎么办？
这节课不录播 — 设计成全互动的，Tutor 陪跑 + Networking 录下来没意义。错过可以等下一期，或者直接看平台的 9 章 OpenClaw 学习方向自学（内容完全覆盖课上的认知部分，免费）。
报名了临时来不了能退 / 转期吗？
线下课容量有限，建议至少提前 48 小时告诉我们 — 可以转给下一期同城学员或你推荐的朋友。具体退改政策看报名链接里的条款，或联系课程顾问。

不能。这节课设计成线下沉浸式的核心原因是 Tutor 陪跑 + 现场 Networking — 这两件事线上做不出效果。如果你想线上学 OpenClaw，平台上有完整的 OpenClaw 学习方向可以自学。
