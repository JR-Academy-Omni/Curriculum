L01📖 小说选择~30 min
AI 漫剧的第一道工序不是开 AI — 是挑对一本小说。挑错了后面所有工序都白做。
📖 你会听到
  • 什么样的小说适合做 AI 漫剧（题材 / 长度 / 节奏 / 画面感）
  • 甜宠 / 玄幻 / 悬疑 / 都市 — 哪类起号最快
  • 公版书 / 自创剧本 / 授权改编的版权红线
  • 看「有没有强情节钩子」比看「文笔」重要 10 倍
  • 怎么从一本 30 万字小说里抠出 1 分钟漫剧的「黄金 30 秒」
🎯 你会带走
  • 5 个挑书维度清单
  • 题材起号速度排序
  • 版权红线避坑指南
L02📝 AI 小说拆分成剧本~40 min
小说几万字，漫剧只有 1 分钟 — 把「故事压缩成戏剧结构」是 AI 干得最漂亮的活。
📖 你会听到
  • 剧本和小说的本质区别：戏剧动作 vs 文字描写
  • 拆小说 Prompt 公式：人物 / 场景 / 冲突 / 反转
  • 1 分钟漫剧标准结构：钩子 5s → 矛盾 30s → 反转 15s → 收尾 10s
  • 对白要写成「AI 能配音」的口语 — 一条铁律
  • ChatGPT / 豆包 / Kimi / Claude 拆剧本横向对比
🎯 你会带走
  • 「小说拆剧本」标准 Prompt 模板
  • 30 分钟拆完一本中长篇
  • 知道哪家 AI 中文剧本最稳
L03🤖 AI 智能体将剧本转分镜剧本~50 min
剧本只有对白和动作 — 分镜剧本要写清楚每镜的镜头 / 景别 / 角色 / 场景 / 时长。用智能体一次跑完。
📖 你会听到
  • 什么是分镜剧本（shooting script）— 一表见全片
  • 搭一个「剧本 → 分镜剧本」智能体（Coze / Dify / Claude Skills 三选一）
  • 智能体自动拆镜：景别（特写 / 中景 / 远景）/ 机位 / 时长 / 转场
  • 分镜剧本 6 列标准模板：镜号 / 景别 / 内容 / 对白 / 时长 / 备注
  • 演示：一段剧本喂进智能体，30 秒输出 12 镜分镜表
🎯 你会带走
  • 能跑完整工序的智能体
  • 分镜剧本模板（CSV / Notion / 飞书三版）
  • 4h → 30min 的工艺提速
L04🎨 AI 角色场景设计~50 min
角色和场景是后面所有分镜图的「资产库」— 这一步出图必须锁死，否则 12 镜会变成 12 个长相不一样的人。
📖 你会听到
  • 主角 + 配角的视觉锁定法（外貌 + 服装 + 标志特征）
  • 多图保持同一个人的 3 种解法：种子 / 参考图 / 描述词冻结
  • 场景库搭建：室内 / 室外 / 夜景 / 雨景 — 一次出 6-8 张备用
  • 风格统一：把「画风 Prompt」抽成模板词，每张图都带
  • 演示：从「古风 18 岁少女」到锁定 5 个角度都能复用的角色集
🎯 你会带走
  • 角色资产库 + 场景资产库
  • 4 套风格 Prompt 模板（古风 / 都市 / 玄幻 / 校园）
  • 一致性 3 种解法
L05🍌 AI 香蕉 (Nano Banana) 分镜图设计~60 min
Nano Banana = 目前角色一致性最能打的图像 AI · 把 L03 的分镜剧本 + L04 的角色场景资产，逐镜出成正式分镜图。
📖 你会听到
  • Nano Banana 平台入口：Google AI Studio / Tapnow 内置 / API
  • 分镜图 Prompt 公式：角色 + 场景 + 动作 + 景别 + 情绪 + 风格
  • 怎么把 L04 锁定的角色喂进去保持长相一致
  • 批量出图工作流：12 镜串行 vs 并行 vs 半人工挑选
  • 失败处理：手指 / 眼神 / 服装错乱的 3 种返工策略
  • 完整演示：12 镜分镜图从头跑到尾
🎯 你会带走
  • 完整一条 12 镜分镜图（含源文件）
  • Nano Banana 分镜图 Prompt 公式表
  • 失败镜局部返工方法
L06🎞 AI 即梦 Seedance 2.0 生成视频~60 min
把 L05 静态分镜图喂进即梦 Seedance 2.0，跑成 3-5 秒动态镜头 — 整套课最关键的一步，也最容易踩坑。
📖 你会听到
  • Seedance 2.0 相比 1.0 / Vidu / Runway 的核心优势（人物一致性 + 运镜）
  • 图生视频 vs 文生视频：漫剧场景为什么必须图生视频
  • 运镜 Prompt 怎么写：推 / 拉 / 摇 / 移 / 跟 + 情绪节奏词
  • 时长控制：3 秒 / 5 秒 / 10 秒 — 漫剧 3-5 秒最稳
  • 队列管理：高峰期排队几小时怎么办（5 个真实对策）
  • 成本测算：一条 1 分钟漫剧大约花多少钱
🎯 你会带走
  • 12 个动态镜头源视频
  • 即梦运镜 Prompt 速查表
  • 1 条片的真实硬成本
L07🎙 AI 声音设计~35 min
漫剧 80% 情绪靠声音 — 不只配旁白，而是角色配音 / BGM / 音效三层都要做。这节用 AI 把三层都跑完。
📖 你会听到
  • 角色配音：声线挑选（少女 / 御姐 / 大叔 / 旁白）+ 情绪调校
  • 配音工具横评：ElevenLabs（情绪最强）/ 海螺 AI（中文最稳）/ 豆包 / 剪映 TTS
  • BGM：用 Suno 出主题音乐 · 5 个免费商用素材站
  • 音效层：脚步 / 风声 / 心跳 / 转场音 — 漫剧专用音效包
  • 三层混音音量比例：人声 0dB / BGM -18dB / SFX -12dB
🎯 你会带走
  • 不像 AI 的角色声线调法
  • 漫剧专用免费商用素材清单
  • 三层混音工业标准
L08✂️ AI 漫剧剪辑包装~35 min
把 L06 视频 + L07 声音 + 字幕 / 封面 / 标题在剪映里拼成成片 — 导出能直接发抖音 / 快手 / 小红书的多版本。
📖 你会听到
  • 剪映 / CapCut 漫剧专用模板（章节卡 / 转场 / 字幕样式）
  • 字幕规范：每行 ≤12 字 / 出现 1.2s 起 / 颜色对比够
  • 封面三秒原则：1 张图说清谁 + 在做什么 + 反转点
  • 三平台标题钩子：抖音 / 小红书 / 视频号 分别怎么写
  • 导出参数：抖音 1080×1920 / 小红书 1080×1350 / 视频号 1080×1920
  • 发布节奏：单条还是连载 · 第一条该不该投流
🎯 你会带走
  • 完整 1 分钟成片（三平台版）
  • 剪辑模板 + 字幕预设 + 封面模板
  • 第一条投流判断逻辑

Q1 · 这套录播课和 2 小时直播工作坊有什么区别？
维度完全不同。2 小时工作坊（ai-comic-workshop）是「俯瞰版」— 讲清楚 AI 漫剧赛道全貌、工具横评、决策框架，看完知道要不要入坑。8 节录播（这套）是「实操版」— 从挑小说到发抖音，一节一节带你做完一条完整漫剧，看完手里有作品。如果时间够 + 预算够，建议先看 2h 工作坊定方向，再用这 8 节录播做出第一条片。
Q2 · 完全零基础能学吗？
能，但要做好心理准备。课程不假设你会任何视频软件 / 编程 / AI 操作，每节都是从开账号开始演示。但 AI 漫剧是个多工具串联的工艺 — 8 个工具来回切，零基础同学要做好「每节看 1 遍 + 跟着做 1 遍」的节奏，整套课大约要花 12-20 小时。不是 2 小时看完就完事的那种课。
Q3 · 课程多长？看一遍要多久？
8 节录播总时长约 6 小时（每节 30-60 分钟不等，L05 / L06 最长）。如果只是「看一遍」大约 6-8 小时；想跟着做一条完整漫剧出来，建议留 12-20 小时（包括工具排队等待、自己出图调整、剪辑迭代）。
Q4 · 工具账号要花多少钱？
全部走免费档可以完整学完所有 8 节，但产出 1-2 条片就会用完免费额度。真要稳定做：
  • 即梦：跑视频是大头，月订阅约 ¥69-199
  • Nano Banana：走 Google AI Studio 免费档日常够用，重度用走 API 按量
  • AI 配音：ElevenLabs 月 $5 起、海螺 AI 国内便宜很多
  • 剪映：免费版够用
一条 1 分钟漫剧的硬成本大约 ¥10-30，每月做 10 条算下来工具开销 ¥200-500。
