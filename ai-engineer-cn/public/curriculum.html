12 周主线：学一节，就在 Dispatch AI 上做一节
周学什么当周在 Dispatch AI 上做什么
W1Python 工程化 / Docker / Git本地 Docker Compose 一键跑通 Dispatch AI，读懂架构
W2国产大模型 API / Prompt / Function Calling把 AI 服务接上 Qwen/DeepSeek，写第一个派单 Function Call
W3Embedding / 向量库选型把调度知识灌进向量库，建检索
W4RAG Pipeline / 混合检索 / 重写压缩给调度决策接 RAG：这一单该派给谁、怎么处理
W5RAG 评测（RAGAS/Langfuse）给 RAG 加评测面板，跑出召回/忠实度指标 ★里程碑
W6Agent 原理 / ReAct / 设计模式把 AI 大脑重写成 ReAct 分诊 Agent
W7MCP 协议 / MCP Server用 MCP 接 Dispatch 后端（查资源/派单/通知）
W8Multi-Agent / 工作流编排升级成多智能体协作调度 ★里程碑
W9微调 LoRA/QLoRA / Llama Factory微调一个调度路由/分类小模型
W10vLLM/SGLang 私有化部署 / 量化 / 合规把微调模型私有化部署进 AI 服务 + 压测 ★里程碑
W11AI Eval / 防注入 / 护栏 / 可观测评测调度决策质量 + 加安全护栏与监控
W12个性化 + Demo + 面试叙事每人加自选领域调度场景 + 独创功能，端到端 Demo ★毕业作品
防简历同质化：W1–W11 全班在同一个 Dispatch AI 上做同样架构（好教好批），W12 强制每人加专属扩展，简历各不相同、抗追问。每周 2 节直播（每节 ≤3 小时），其余录播 / Lab / Quest 自主节奏。
