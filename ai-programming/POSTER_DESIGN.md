## 概述

在 `curriculum/ai-programming/public/` 新增 `poster.html`，提供多渠道 × 多方向的宣传海报组。页面内通过 tab 切换渠道尺寸，子 tab 切换方向角度。每个组合都是一个可截图使用的独立海报。

## 页面结构

```
poster.html
├── 渠道 Tab（顶部主 Tab）
│   ├── 朋友圈版 (1:1, 1080×1080)
│   ├── 小红书版 (3:4, 1080×1440)
│   ├── 公众号头图版 (2.35:1, 900×383)
│   └── 详情长图版 (宽 1080, 高度自适应)
│
└── 方向 Sub-Tab（每个渠道下）
    ├── 🎯 零基础小白版
    ├── 🚀 创业者版
    ├── 💼 职场转型版
    ├── 👩‍👧 宝妈/副业版
    ├── 😤 学编程劝退版
    ├── 💸 花钱请外包版
    ├── 😰 AI 焦虑版
    ├── 🏆 学员作品版
    └── 📊 课程数据版
```

## 技术规范

- **纯 HTML + CSS**，不用 React（跟 curriculum 其他页面一致）
- 复用 `styles.css` 的配色变量和字体
- Neo-Brutalism 风格：黑色粗边框、offset shadow、bold 色块
- Tab 切换用纯 CSS（`:target` 或少量 vanilla JS）
- 每个海报卡片有固定尺寸容器，方便截图
- 移动端响应式（海报本身保持固定比例，容器可滚动）

## 各方向内容设计

### 1. 🎯 零基础小白版

**标题**: 不会写代码？AI 帮你写。

**副标题**: 用中文告诉 Cursor 你要什么，8 周做出一个网站

**核心文案**:
- 从「打开浏览器」教起，不落下任何人
- 不学 for 循环、变量声明、if else
- 你说需求，AI 写代码，你检查结果
- 103 节课 + 29 个互动练习，不是干看视频

**视觉**: 对话气泡风格 — 左边是人说中文需求，右边是 AI 返回代码

**CTA**: 零基础也能做产品 → 了解课程

---

### 2. 🚀 创业者版

**标题**: 有想法？8 周自己做出来。

**副标题**: 不用请开发团队，用 AI 从想法到上线

**核心文案**:
- 请一个开发团队：3 人 × 3 个月 × $8000/月 = $72,000
- 自己用 AI 做：8 周 × 你的时间 = 一个完整产品
- 前端页面 + 后台 CMS + 用户系统 + 数据库 + 部署 + SEO
- 从 PRD 写到产品上线，全流程实操

**视觉**: 成本对比计算器风格 — 左边传统开发（大数字+红色），右边 AI Coding（小数字+绿色）

**CTA**: 省下 10 万开发费 → 了解课程

### 3. 💼 职场转型版

**标题**: AI Coding 工程师：2026 年新岗位

**核心文案**:

## 渠道尺寸适配

### 朋友圈版 (1:1, 1080×1080)

信息密度最低，要求一眼看懂：
- 标题（大号加粗）
- 一句副标题
- 3 个核心数据点
- 课程名 + logo
- 底部 CTA 或二维码位置

### 小红书版 (3:4, 1080×1440)

比朋友圈多一些空间，可以放：
- 标题 + 副标题
- 4-5 个要点（带 icon）
- 课程数据条
- 适合人群标签
- 底部 CTA

### 公众号头图版 (2.35:1, 900×383)

横版，信息最精简：
- 左侧：标题 + 一句话
- 右侧：1-2 个核心数据或视觉元素
- 背景色块区分

### 详情长图版 (宽 1080, 高度自适应)

信息最完整，适合顾问 1v1 转发：
- 标题区
- 痛点描述
- 课程亮点（5-6 个）
- Phase 概览
- 适合人群
- 课程数据
- CTA + 二维码位置

## 交互设计

```
┌─────────────────────────────────────────────┐
│  📱 朋友圈  │  📕 小红书  │  📰 公众号  │  📋 长图  │  ← 渠道 Tab
├─────────────────────────────────────────────┤
│  🎯小白 │ 🚀创业 │ 💼转型 │ 👩‍👧副业 │ 😤劝退 │ ...  │  ← 方向 Sub-Tab
├─────────────────────────────────────────────┤
│                                             │
│         ┌──────────────┐                    │
│         │              │                    │
│         │   海报预览    │  ← 固定尺寸容器    │
│         │  (可截图区域)  │                    │
│         │              │                    │
│         └──────────────┘                    │
│                                             │
│  [ 📋 复制文案 ]  [ 📥 下载提示 ]            │  ← 辅助操作
│                                             │
└─────────────────────────────────────────────┘
```

- 切换 tab 时海报区域平滑过渡
- 海报容器有虚线边框标注截图区域
- 「复制文案」按钮：复制当前方向的配套朋友圈/小红书文案
- 「下载提示」：提示用户用浏览器截图功能（Cmd+Shift+4 / 右键截图）
