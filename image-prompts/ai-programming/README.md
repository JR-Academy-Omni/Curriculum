## ai-programming 专属视觉锚点

每张 prompt 至少用一个：

- ✅ **完成态网页 preview**（屏幕里**绝不显示代码**，避免吓退非技术受众）
- ✅ **Cursor + 中文** = 写代码（核心工具叙事）
- ✅ **5 周 / 8 周** 双轨节奏（基础班 / 进阶班）
- ✅ **想法 → 产品** 时间轴 / 进度环 / track 对比
- ✅ **Live 绿点** 浏览器标签 / 网址 yoursite.com（"上线"的视觉化）
- ✅ **MVP 5 stack badge**（前端页面 / 后台 CMS / 用户系统 / 数据库 / 部署上线）
- ✅ **4 个 Phase 进度环**（0 工具准备 / 1 产品设计 / 2 编程实战 / 3 AI 进阶）
- ✅ **handwritten "shipped" sticky note** / **lunch break 9-5 calendar**（office worker EN 专属）
- ✅ **Engineer Track vs Creator Track** 长短对比（career-changer EN 专属）

## 课程红线

写新 prompt 时必避：

- ❌ **承诺加薪 / 升职 / 找到工作** —— 这是创作者课，不卖就业 outcome
- ❌ **"100% 上岸"/"包就业"** —— 完全不卖 IT 求职（那是 web-code 的事）
- ❌ **真实公司 / 真实软件 logo / 真实截图** —— 法律红线
- ❌ **屏幕里显示真代码** —— 非技术受众看到代码就跑了，永远显示完成态网页 preview
- ❌ **"35 岁 / 996 / 澳洲"** —— 不是这门课受众的焦虑（那是 web-code）
- ❌ **"PR / sponsor"** —— 不卖海外求职
- ❌ **"年薪 N 万"** —— 不卖具体薪资数字
- ✅ **"5 周 ship"** / **"不写代码"** / **"用中文指挥 Cursor"** —— 课程明文卖点
- ✅ **"AI Coding 创造者"** —— 课程对受众的新身份命名
