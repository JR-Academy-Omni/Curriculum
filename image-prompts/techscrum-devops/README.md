## 与 web-code-bootcamp 故意分裂的角度

DevOps 班和全栈班受众接近（都是想进澳洲 IT 的程序员/运维），但是卖点不能复用：

| Combo | web-code-bootcamp | techscrum-devops（本课）|
|-------|-------------------|----------------------|
| programmer × overseas | "4000+ 留学生上岸 + 12 月内推"（桥梁叙事） | "Terraform 比 React 贵 30% / DevOps 上 SOL"（**技能稀缺性**叙事）|
| programmer × age-35 | "35 岁不是终点，澳洲 IT 不歧视年龄" | "8 yrs of ops scars is the resume"（**经验是资产** 叙事）|
| career-changer × overseas | "4000+ 学员澳洲 IT 上岸" | (本课不直接做这条 combo) |
| programmer × side-hustle | (web-code 不做这条) | "6h 部署 → 12m 一键 / A$2K 单价"（**基建可开发票** 叙事）|

绝对不在 DevOps 班用以下全栈班锚点：
- ❌ "三后端语言任选 Node / .NET / Java"（DevOps 不卖语言）
- ❌ "12 月简历内推库 + 3 次/年面对面见招聘官"（这是全栈班的产品力，不绑定 DevOps 课）
- ❌ "P3 商业项目 = 8-10 全栈 + 6-8 DevOps + BA + Designer"（DevOps 班是 7 周独立项目，不是 3 月真团队）

DevOps 班专属锚点：
- ✅ "7 周 1 个真商业项目"（短而硬）
- ✅ "Terraform · EKS · AKS · Prometheus · Grafana"（澳洲企业实际栈）
- ✅ "导师 Dex · Amazon · 9 年 IT · 前 DevOps Team Lead"（导师权威）
- ✅ "AU SOL / A$130K median"（DevOps 在澳洲技能短缺清单数据）
- ✅ "AI 替代不了 prod"（基础设施权责论）
- ✅ "A$1,500-3,000 freelance 单价"（基建副业经济模型）
- ✅ "P0/P1 outage scars"（中年人 ops 经验 = 资产）

## 课程红线（合规）

写新 prompt 时必避：

- ❌ **承诺 100% 上岸 / 包就业** —— 用"行业短缺 / 高就业率 / SOL 名单上"
- ❌ **真实公司名 / Logo**（Amazon / AWS / Azure / HashiCorp / Jenkins 等品牌名只用纯文字 badge，不画 logo）
- ❌ **真实学员姓名 / offer 截图**
- ❌ **真实澳洲国旗带详细国徽** —— 用 generic 蓝红条纹 / 删除国旗
- ❌ **承诺具体薪资数字** —— 用范围 "A$130K median"（公开数据）
- ❌ **真实 ANZSCO 代码 / 真实雇主名 / 真实 visa 类别**（occupation list 文档保持抽象）
- ✅ "上 SOL / 行业短缺 / 9 年 Amazon 导师 / 7 周可写进简历的项目" —— 课程明文卖点

## 三种格式视觉锚点速查

DevOps 课程视觉锚点都围绕"基建静物 / pipeline / dashboard / 发票 / 文档"，避免人脸：

| 锚点 | 适合 combo | 用在哪个 format |
|------|-----------|----------------|
| Pipeline 静物（laptop + Build/Test/Deploy + 绿 ✓） | overseas / side-hustle | hero（Mode A）|
| Terraform plan flatlay（笔记本 + tfstate / vault 标签） | programmer × overseas (ZH) | xhs |
| 6h → 12m + invoice infographic | programmer × side-hustle (ZH) | xhs |
| AI vs Human split + production access ≠ AI access | career-changer × ai-fomo (ZH) | xhs |
| 深夜 dashboard + 蒸汽咖啡 + 城市夜景虚化 | career-changer × ai-fomo (ZH) | hero |

| Manual SSH terminal vs Terraform pipeline | career-changer × side-hustle (EN) | xhs |
| 8 yrs P0/P1 timeline + tool badges | programmer × age-35 (EN) | xhs |
| 老书堆 + dashboard + 眼镜 | programmer × age-35 (EN) | hero |
