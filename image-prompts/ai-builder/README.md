## 与其他 bootcamp 故意分裂的角度

ai-builder 这个课最容易跟 **ai-engineer-bootcamp** 和 **web-code-bootcamp** 重合，所以写之前先把锚点切干净：

| Combo | ai-engineer-bootcamp | web-code-bootcamp（全栈班） | **ai-builder（本课）** |
|-------|--------------------|-----------------------------|----------------------|
| **目标受众** | 转 AI Engineer 的开发者 | 转码到澳洲就业的留学生 | **已会写代码的工程师** |
| programmer × age-35 | "35 岁前端学 RAG"（换技术栈） | "35 岁逃 996 去澳洲"（换国家） | "**35 岁不卷代码量，卷 AI 协作能力**"（换工具层级） |
| programmer × ai-replace | "Don't compete with AI. Engineer with it."（深度 AI 角度） | — | "**AI rewards seniors who orchestrate it**"（orchestration 角度，不是 RAG/Agent SDK） |
| programmer × side-hustle | "$1k MRR project" / 7 production 项目（深度 AI 副业） | AU contract 100-150 AUD/hr（劳务角度） | "**Ship a SaaS this weekend**"（Vibe Coding 加速器视角） |
| pm-manager × ai-fomo | — | — | "**Tech Lead 看不懂下属用 Cursor 写的 PR**"（技术 manager 补课） |
| career-changer × ai-fomo | — | "AI didn't kill coding jobs"（全栈劳动力市场） | "**Junior devs ship before they apply**"（毕业生差距） |

**绝对不在 ai-builder 用以下别 bootcamp 锚点：**
- ❌ Context Engineering / RAG / GraphRAG / Agent SDK / 92% JD coverage（这些是 ai-engineer-bootcamp 的招牌）
- ❌ 4000+ 学员上岸 / 12 月内推库 / Australian IT 不 996（这些是 web-code-bootcamp 的招牌）
- ❌ "4 周 / 33 节课 / 不写代码"（这些是 ai-essentials-bootcamp 的招牌）

**ai-builder 专属锚点（每张至少出现 1 个）：**
- ✅ **113 lessons / 4 phases / 5 实战项目**（深度 marker）
- ✅ **Claude Code · Cursor · Codex CLI**（三工具同框，大师课招牌）
- ✅ **Skills · MCP · Sub-agent · agent.md**（架构层四件套）
- ✅ **PRD → Skill → Agent → Test → Ship**（5-step pipeline）
- ✅ **"大师课"** 不是 "入门班" / 不是 "转码班"（受众已会写代码）
- ✅ **orchestration / 编排** 而不是 "学会用 ChatGPT"（关键词差）

## Prompt 写法规范

每张 prompt 都遵守 [`_shared/gpt-image-2-guide.md`](../_shared/gpt-image-2-guide.md)：

- ✅ **800-1000 字符**（避免长 prompt 被模型放弃）
- ✅ **首 50 词权重**（aspect ratio + subject + 主标题文字前置）
- ✅ **每段中文用 `reading: "..."` 引号锁字**
- ✅ **每段后跟 `Text must be sharp and beautifully composed.`**
- ✅ **直接 hex 配色** `#FF5757` / `#10162F` / `#FFF1E7` / `#FFB800`
- ✅ **xhs Thinking mode** / **wechat-cover & hero Instant**

## 课程红线（合规）

写新 prompt 时必避：

- ❌ **真实品牌名 / Logo** —— 不写 ChatGPT / Anthropic / Cursor / OpenAI / GitHub / Stripe / Vercel / Notion / Atlassian / Figma 的可见 logo（**文字 token 可以**，比如标题里出现 "Claude Code" 这个词没问题，但**画面里不能渲染他们的 logo**）
- ❌ **Ethnic descriptor** —— 不写 "Chinese male" / "Asian young person"
- ❌ **具体薪资数字** —— 海报上不出现具体数字 "$150K" / "5 万" / "10 万"
- ❌ **"100% 上岸 / 包就业 / 学不会退款"** —— 广告法红线
- ❌ **真实人脸正面** —— 三分之一背身 / 物件 flatlay
- ❌ **真实公司 PR 截图 / 工资条** —— 知识产权风险
- ✅ "113 lessons" / "4 phases" / "5 production projects" / "8 years of architecture taste" —— 课程明文卖点
