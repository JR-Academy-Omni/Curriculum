## Prompt 写法规范

每张 prompt 都遵守 [`_shared/gpt-image-2-guide.md`](../_shared/gpt-image-2-guide.md)：

- ✅ **800-1000 字符**（避免长 prompt 被模型放弃）
- ✅ **无真实品牌名**（不写 ChatGPT / Stripe / Cursor / Vercel —— 触发安全过滤）
- ✅ **无 ethnic descriptor**（不写 "Chinese male / Asian young person"）
- ✅ **首 50 词权重**（aspect ratio + subject + 主标题文字前置）
- ✅ **每段中文用 `reading: "..."` 引号锁字**
- ✅ **每段后跟 `Text must be sharp and beautifully composed.`**

## 课程红线（合规）

写新 prompt 时必避：

- ❌ **薪资数字** —— 海报上不出现 "100K / $150K / 30 万" 等具体数字（PRD 时代红线）
- ❌ **"100% 上岸 / 包就业 / 学不会退款"** —— 广告法红线
- ❌ **真实公司 logo / offer 截图** —— 知识产权风险
- ❌ **真实人脸正面**（出图保留三分之一背身 / 物件 flatlay）
- ✅ "92% JD coverage"、"$150K AI Engineer roles"（用统计 / 范围式描述，不承诺个体结果）
