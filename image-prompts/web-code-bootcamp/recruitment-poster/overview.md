- **不要画"程序员形象"**：personas 用 abstract icons not 戴眼镜小人
- **DevTools tabs**：在 [TEXT] 段写完整的 tab 名（Elements / Network / Console），否则模型会编
- **Network waterfall**：明确 `colored horizontal bars stacked vertically, each labeled with a sprint name`
- **不要画"代码截图"**：明确 `DevTools panel STYLE, not screenshots of actual code`
- **"Offer 收割机" 上色**：在 [TEXT] 显式 `colored #FFB800 with yellow highlight tape backing`
