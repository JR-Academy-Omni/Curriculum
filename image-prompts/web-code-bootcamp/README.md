## 与 ai-engineer-bootcamp 故意分裂的角度

同名 "AI Engineer"，但是两个完全不同的课程定位：

| Combo | ai-engineer-bootcamp | 全栈班（本课）|
|-------|--------------------|--------------|
| programmer × age-35 | "35 岁前端开始学 RAG"（**换技术栈**）| "35 岁逃离 996，澳洲 IT 不歧视年龄"（**换国家**）|
| career-changer × overseas | "AI Engineer 是 sponsor 最快通道"（深度 AI 角度）| "4000+ 留学生上岸 + 12 月内推"（桥梁角度）|

绝对不在全栈班用以下 ai-engineer 锚点：
- ❌ Context Engineering / RAG / Agent SDK / 92% JD coverage（受众听不懂、卖点不对）
- ❌ Production Agent / Three SDKs / Eval Pipeline（深度技术）
- ❌ "$150K-$350K production AI Engineer" 薪资范围（全栈班是 80K AUD 起）

全栈班专属锚点：
- ✅ "4000+" 学员上岸（AU/CN/US 已就业，多年口碑）
- ✅ "5-phase 流程" Plan / Develop / Build / Test / Release
- ✅ "三后端语言" badges（Node / .NET / Java 任选）
- ✅ "996 vs 9-5 时钟"（逃离国内 IT 文化）
- ✅ "12 月内推库 + 3 次/年面对面见招聘官"
- ✅ "3 个月真团队"（8-10 全栈 + 6-8 DevOps + BA + Designer）

## 用过的真实卖点（来自课程研究）

每张 prompt 至少用一个真实数据：

- **24 周** / 485 lessons / 2 个人项目 + 1 商业项目（3 月真团队 P3）
- **4000+ 学员**已就业，覆盖 AU / CN / US
- **澳洲 IT 80K AUD 起 / 90% 雇主缺人**（公开数据）
- **澳洲 IT 不 996 / 不 35 岁歧视 / 不性别歧视**（课程描述明文）
- **12 月简历内推库 + 3 次/年面对面见招聘官**（不是冷投）
- **3 后端语言任选** Node.js / .NET / Java
- **Project 2 = MongoDB Weather App** 真大厂面试题
- **Project 3 = 8-10 全栈 + 6-8 DevOps + BA + Designer 真团队架构**

## 课程红线

写新 prompt 时必避：

- ❌ **承诺 100% 上岸 / 包就业** —— 内推 ≠ 保证 offer，写 "进入内推库" / "招聘官主动看你" 而不是"包就业"
- ❌ **真实公司名 / Logo**（Deloitte / Servian 等不能直接画）
- ❌ **真实学员姓名 / 头像** —— 用 abstract silhouettes
- ❌ **真实澳洲国旗带详细国徽** —— 用 generic 蓝红条纹
- ❌ **承诺具体薪资数字**（"年薪 100K"）—— 用范围 "80K AUD 起"
- ✅ "4000+ 学员上岸" / "12 月内推" / "Australian IT 不 996" —— 课程明文卖点
