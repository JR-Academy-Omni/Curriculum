## 出图 tips

- **photocopy grain**：写 `8% opacity speckles, slight horizontal banding` 才像复印
- **stencil 字体**：`stencil display type with photocopy bleed on edges`，否则会画成喷漆涂鸦
- **MIT stamp**：`rotated red rectangle stamp at 12° angle, slightly faded`
- **不要画真 GitHub logo**：用 `octocat-stencil mark` 替代
- **rough deckle edges**：`hand-torn paper edges, NOT clean rectangle borders`
- **"接管" 上色**：在 [TEXT] 段显式 `colored #FFB800 with yellow highlight tape behind`
