# 第三期三维度重新定位

> **核心发现**: 当前课程严重偏向"求职/面试"，"职业提升"内容严重不足  
> **改进方向**: 从三个维度均衡补充资源和内容

---

## 一、三个维度定义

这门课本质上要回答学员三个不同阶段的问题：

```
┌──────────────────┬──────────────────┬──────────────────┐
│  阶段 1：没工作  │  阶段 2：找工作  │  阶段 3：有工作  │
├──────────────────┼──────────────────┼──────────────────┤
│  职业认知/规划   │  求职 + 面试     │  职业成长/提升   │
│                  │                  │                  │
│  我该做什么?     │  怎么拿到 offer? │  怎么升职加薪?   │
│  哪个方向有前途? │  怎么通过面试?   │  怎么不被淘汰?   │
│  我适合什么?     │  怎么谈薪资?     │  怎么长期发展?   │
└──────────────────┴──────────────────┴──────────────────┘
```

---

## 二、当前课程的维度分布

| 维度 | 当前节数 | 占比 | 评价 |
|------|---------|------|------|
| **D1 · 职业认知与规划** | 18 节 | 14% | 基本够 |
| **D2 · 求职与面试** | 95 节 | 74% | **严重过重** |
| **D3 · 职业成长与提升** | 15 节 | 12% | **严重不足** |

**问题：**
- 74% 都是求职/面试内容，这门课更像"求职冲刺营"而不是"职业训练营"
- 职业提升只有 15 节，而且集中在最后（L100-L112），拿到 offer 之前的学员根本看不到价值
- 学员学完就想"找到工作了，这门课跟我没关系了"——**粘性不够**
