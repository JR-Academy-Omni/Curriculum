# 售卖平台对接

> 4 引流课 + 主课三档要在哪些平台卖，怎么对接，谁负责。
> ⚠️ 状态：候选方案已列，待拍板 + 后台配置。

## 平台候选

### 中文学员

| 平台 | 适用 | 优势 | 劣势 |
|---|---|---|---|
| 小鹅通 | 4 引流课 + 主课 | 国内成熟课程平台，接微信支付/支付宝；带学员管理 + 录播分发 + 答疑模块 | 抽佣 5-8%，定制化弱 |
| 课程页直接（自建） | 主课三档 | 0 抽佣 + 完全可控 | 要自建支付（接微信/支付宝官方）+ 课程播放页 |
| 知识星球 | 长尾留客 + 社群答疑 | 社群活跃度高 | 不适合做付费课程主入口 |
| 飞书文档 + 自建落地页 | 4 引流课 | 0 抽佣 + 内容快速迭代 | 学员体验弱（要换平台看课） |

**推荐**：4 引流课走小鹅通（学员心智成熟 + 上线快），主课走自建课程页 + 接微信/支付宝（高客单价省抽佣）。

### 英文学员

| 平台 | 适用 | 优势 | 劣势 |
|---|---|---|---|
| Stripe + 自建落地页 | 全部 | 完全可控，AUD 结算直接 | 要自建落地页 + 课程播放 |
| Teachable / Thinkific | 4 引流课 | 课程平台成熟 | 月费 $39-99/月 + 抽佣 |
| Gumroad | 4 引流课 | 简单上线 | 抽佣高（10%）+ 学员体验弱 |
| Udemy（不推荐） | — | — | 平台定价权 + 价格被压到 $9.99 |

**推荐**：英文走 Stripe + 自建落地页。直接对接 JR 现有 jr-academy-web-zh 课程详情页（已有 Stripe 集成）。

## 售卖配置 checklist（每个引流课）

- [ ] 课程封面图（1080×1080 + 1242×1660 竖版）
- [ ] 课程描述文案（销售文案，禁 AI 味禁金钱承诺）
- [ ] 价格 + 原价（¥99 vs 划线 ¥199）
- [ ] 试看视频（前 5 min 录播片段，可选）
- [ ] 学员评价 / 案例（第 1 期跑完后回填）
- [ ] 优惠券（D7 早鸟 抵扣码）
- [ ] 学员群 / 微信号引导（购买后自动发引流码）
- [ ] 数据埋点：UTM + 转化漏斗

## 数据回流

每个平台的购买数据（学员 ID / 购买时间 / 渠道 / UTM）必须回流到统一 dashboard，用于：
- 计算每个引流课的获客成本（CAC）
- 监控 D7 转化率（引流课 → 主课）
- 长尾留客触达列表（D8-D30）
