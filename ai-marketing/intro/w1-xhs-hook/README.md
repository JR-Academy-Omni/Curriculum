## 主题

**1 小时学会用 AI 写小红书爆款（含 5 篇真实拆解 + 10 个 prompt 模板）**

## Quick win deliverable

- 10 个分行业的小红书爆款 prompt 模板（穿搭 / 美食 / 职场 / 母婴 / 数码各 2 个）
- 1 份「图文结构 SOP」PDF（标题 / 首图 / 钩子 / 正文节奏 / CTA）
- 5 篇真实爆款的 prompt 反推拆解
- 当晚的实战篇（用模板 #3 写一篇关于自己工作的笔记）

## 学完承诺

当晚发出第 1 篇 AI 写的小红书，3 天内回看数据。**不承诺**点赞 / 收藏数 / 涨粉。
