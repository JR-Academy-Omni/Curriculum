## 主题

**1 小时让 ChatGPT / Perplexity / Claude 主动引用你的品牌（GEO 入门）**

GEO = Generative Engine Optimization — 让品牌出现在 AI 搜索引擎答案里，而不是传统 Google 搜索结果第 N 页。

## Quick win deliverable

- 1 套 GEO 优化 prompt（用户视角问 + LLM 视角答的对应规则）
- 5 个真实「被 LLM 引用」案例拆解（截图 + 为什么 work 的分析）
- 1 份 GEO 自查清单（10 项 checklist：网站结构 / 锚文本 / FAQ schema / 提到品牌的高质量第三方内容 / ...）
- 给自己品牌跑一次 GEO 优化 demo

## 学完承诺

1 周内能在 Perplexity / ChatGPT 搜自己品牌相关问题，看到自己品牌出现在答案中（至少 1 次）。**不承诺**搜索量 / 流量 / 订单。
