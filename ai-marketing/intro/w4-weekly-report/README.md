## 主题

**1 小时让 AI 自动帮你出营销周报**

不教 GA4 / Looker / Power BI 怎么用（学员有自己的工具栈），教**怎么把数据丢给 AI、让 AI 出可读的周报**。

## Quick win deliverable

- 1 个万能周报 prompt 模板（投入 / 产出 / 异常 / 假设 / 下周动作 5 段）
- 3 个真实周报案例（含异常解读 + 下周动作）
- 数据源接入示例（GA4 / 飞书多维表 / Klaviyo / 抖音后台等）
- 学员用自己业务数据跑出 1 份真实周报，发给团队 / 老板看

## 学完承诺

本周内用 prompt 给团队 / 老板交 1 份 AI 出的真实周报。**不承诺**周报会让数据变好 / 老板满意。

## Hook 暗线

> "你今天写了 1 份周报。如果你想做的是
> ——北极星指标 + 漏斗 + cohort 留存 + 多触点归因 + 自动 dashboard，让 AI 每周自动出报告
> ——那是完整的数据 SOP。我们 4 周课 W4 会把它建起来。"
