## 1. 漏斗定位

- **主课方向**：AI Marketing（AI 驱动的内容生产 + 投放 + 私域获客）
- **主课时长**：4 周
- **目标用户**：
  - 中文站：0-3 年运营 / 市场 / 内容人，想用 AI 提效或借 AI 转型自由职业
  - 英文站：澳洲本地中小企业 marketer / freelancer，想把 AI 工具链接入日常营销
- **业务模式**：A 完整四档（¥99 / $29 AUD 引流课 + 主课三档）
- **核心承诺**：4 周做出第一个能跑的 AI 内容生产线（小红书 / 抖音 / 长文 + 私域 + 数据）

## 3. 主课三档

| 档位 | 定价 (CNY / AUD) | 一句话承诺 | 关键差异点 |
|---|---|---|---|
| 自学版 | ¥699 / $209 AUD | 跟着录播看完 4 周，社群答疑 + 打卡返现 | 录播 + 大群 + 21 天打卡满返 30% + 完课 50% 解锁高级 prompt pack |
| 教学版 | ¥2480 / $749 AUD | 4 周直播跟做，作业批改 + 小班同学陪伴 | 直播 + 100 人小班群 + 作业批改 + 1对1 答疑券 ×2 |
| 陪跑版 | ¥9800 / $2880 AUD | 60 天内跑通 1 套完整 AI 内容生产线（4 周稳定运行 + 跨平台分发 + 数据复盘报告） | 1对1 教练 + 项目陪跑 + 工具链定制 + 不达标延期 30 天免费陪跑 |

**升级路径**：
- 自学 → 教学：补 (2480 - 699) × 70% = ¥1247 / 约 $377 AUD（开课后 4 周内）
- 教学 → 陪跑：补 (9800 - 2480) × 80% = ¥5856 / 约 $1769 AUD（开课后 6 周内）
- 自学 → 陪跑跳级：补 (9800 - 699) × 75% = ¥6826 / 约 $2061 AUD（开课后 4 周内）

**完课率保障**（自学版必备）：
1. 21 天打卡满返 30%（强约束）
2. 班主任日报 + 每周复盘直播（强社群）
3. 完课 50% 解锁内部 prompt pack（隐性激励）

**陪跑版可衡量承诺**（过程性，不涉及金钱/收入）：
- 60 天内跑通 1 套完整 AI 内容生产线，达标标准：
  - 4 周稳定持续输出（不少于 24 篇）
  - 跨平台分发（小红书 + 至少 2 个其他平台）
  - 数据复盘报告（含曝光 / CTR / 互动率 / 改进项）
  - 工作流文档可复用（prompt pack + SOP + 排期表）
- 不达标条款：延期 30 天免费陪跑（不退款，避免金钱挂钩 + 合规风险）
- 学员义务：每周交作业、参加 80% 直播、不达标视为放弃承诺
- 🚨 **承诺红线**：禁止承诺金钱 / 收入 / 订单 / 月入 N 元（监管 + 合规 + 体面）

## 4. 引流课

- **主题**：1 小时学会用 AI 写小红书爆款（含 5 篇真实拆解 + 10 个 prompt 模板）
- **时长**：85 min
- **定价**：¥99 / $29 AUD
- **Quick win deliverable**：
  - 10 个分行业的小红书爆款 prompt 模板（穿搭 / 美食 / 职场 / 母婴 / 数码）
  - 1 份「图文结构 SOP」PDF（标题 / 首图 / 钩子 / 正文节奏 / CTA）
  - 5 篇真实爆款的 prompt 反推拆解
  - 当晚的实战篇（用模板 #3 写一篇关于自己工作的笔记）
- **学完承诺**：当晚发出第一篇 AI 写的小红书，3 天内回看数据
- **在主课大纲中的对应位置**：W1 「AI 内容生产」的衍生（不是 W1 的删减版）
- **Hook 暗线**：单篇 → 一整条内容线（跨平台 + 数据 A/B + 7×3 节奏 = 主课卖的事）
- **Hook 话术骨架**（85 min 倒数 5 min）：
  > "你今天做的是单篇小红书。如果你想做的是一整条内容线
  > ——一周 7 天每天 3 篇、跨平台分发、还能跑数据 A/B
  > ——那是一个体系化的事情。我们 4 周课会把这条线建起来。
  > 现在你不用决定，先把今天学的发出去，看看数据。"

## 5. 过渡 7 天序列

- **D0（购买后 < 30 min，邮件 + IM）**：交付 deliverable + 引导今晚发布第 1 篇 + 班主任 12 小时内复盘
- **D1（群里）**：发起"昨晚谁发了？" 复盘话题，挑 3 篇真实案例公开点评 + 给每篇 2 个改进 tip
- **D2（群里）**：发一个**别人做的**进阶案例（小红书涨粉 0→1w 的 AI 内容线），拆解为什么 work
- **D3（直播）**：30 min Q&A，提前从群里挑 5 个高频问题，结尾留 5 min "下一步可以怎么放大"——不发链接、不催报名
- **D4（群里）**：发起「7 天小挑战」连发 3 篇看趋势 → 完成的进"实战群"（封闭小群）
- **D5（群里 / 1对1）**：用户开始问"怎么做大"，班主任聊 A/B/C 三条放大路径，自然提及主课走 B 路（不发链接）
- **D6（群里）**：晒 3-5 个完成 7 天挑战的真实优秀同学（**严禁找托**）：输入 → 动作 → 结果数据
- **D7（开放报名）**：主课三档报名页 + 限时 48h 早鸟优惠（引流课用户专属）。结构：共鸣 → 单篇 vs 内容线差距 → 三档 → 早鸟

**反推销红线**：
- D0-D6 不许出现「报名」、「主课」、「立即购买」、「优惠」字眼
- D3 直播尾巴不许发链接
- D7 限时 48h 后早鸟话术全部撤掉，**禁止「再延 24h」**——破坏信任

**转化锚点**：
- D7 报名页基于「单篇 → 一整条内容线」差距叙事
- 早鸟优惠形式：自学版 ¥99 抵 ¥299 / 教学版 ¥99 抵 ¥499 / 陪跑版 ¥99 抵 ¥999
- 健康转化率目标：8-15% 引流课用户买主课（行业平均 5-15%）
