常见问题
课程承诺是什么？会不会包就业 / 接单？
课程承诺只覆盖过程性结果：4 周搭建自己的 AI 营销工具链、4 个模块各跑 1 次完整闭环、产出 1 套作品集 + 1 份 4 周复盘报告。不承诺金钱 / 月入 / 第 N 单 / 包就业 / 拿 offer / 副业 / 接单。陪跑版含 1v1 求职 / 接单方法论辅导，但不保证结果。
需要会写代码吗？需要做过投放或者懂 SQL 吗？
都不需要。需要会用电脑、用过 ChatGPT 或豆包 / Kimi 一两次。课程里所有的 Lab 都不需要写代码，Coze / n8n / Lindy 等工具是 0 代码或低代码。SQL 在 W4 数据课用 Code Interpreter 自然语言出结果，不需要学员手写。
双轨课是怎么走的？我能跨场看吗？
W1（内容） / W4（数据）合班 — 两边学员共享。W2（投放） / W3（私域 agent）中英分轨：中文学员主轴 = 巨量引擎 + 千川 + 聚光 + Coze + 微信私域；英文学员主轴 = Google Ads + Meta + TikTok + n8n + Lindy + HubSpot AI。所有直播都录播，跨场学员可以看回放，但 Clinic 和作品互评只在所属语种场参与。
4 周时间够吗？我每天要花多少时间？
每周直播 2-4 场（含 Clinic）共 ~4-6 小时 + Lab 自主练习 ~3-4 小时 + 自学 ~2-3 小时 + 作业 / Quest ~2 小时。每周总投入约 12-15 小时，相当于业余时间每天 2 小时。如果错过直播可以看录播。每周一个独立闭环，不会因为前一周没完成而完全 block 后一周。
GEO 是什么？跟传统 SEO 有什么区别？为什么要学？
GEO（Generative Engine Optimization）= 让你的品牌 / 内容在 ChatGPT / Perplexity / DeepSeek / 豆包 / Kimi / Claude / Gemini 这些 AI 答案里被引用。传统 SEO 优化 Google 搜索结果排名；GEO 优化 AI 答案的引用频次。2026 Q1 已有 12-18% 英文 informational queries 由 AI 搜索处理（AI Magicx 2026），中文 LLM 增长更快。本课 W2 系统教中英两套 GEO 方法 + 监测脚本。
Coze / n8n / Lindy 哪个更适合我？
简单说：中文学员选 Coze（字节出品、中文社区厚、免费起、对接微信 / 公众号方便）；英文学员选 n8n + Lindy（n8n 自托管 + 全功能；Lindy 偏 SaaS 即开即用，对接 HubSpot / Slack / Gmail 方便）。课程不强迫选哪个 — W3 直播双轨各讲一套主轴，wiki 里有 Zapier / Make / Dify 等其他工具的对比，按团队场景挑 1 个深用。
