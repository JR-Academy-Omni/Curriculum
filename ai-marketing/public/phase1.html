平台规则与流量底层（小红书 vs 抖音 vs 公众号 vs LinkedIn）

Section 2 · Prompt 基础四件套（写文案前必修） L02-L04 · L11-L12 · 5 个复用 Lab
L02 · Lab

Clear Task — 写文案前必修

prompt 30min
📖 教学内容

四要素结构：目标 + 角色 + 受众 + 输出格式。把模糊需求"帮我写个小红书"改造成"帮我写一篇给 25-30 岁职场女性的小红书种草笔记，目标点赞收藏率 8%+，输出 8 个候选钩子标题 + 1 段 200 字正文"。

🔬 Lab 实操

Prompt Lab 内置 6 个模糊 prompt 让学员改造。每改一版 AI 即时反馈缺哪个要素 / 写得弱在哪。最后跑一次自己工作场景的真实 prompt。

🤖 Prompt Lab 自动评分
📋 作业

学员把自己工作中常用的 3 个 prompt 改成四要素版本，贴在学习群。

L03 · Lab

Few-shot — 用 3 条爆款教 AI 写新爆款

prompt 45min
📖 教学内容

Few-shot 原理 — 例子比说明书更有效。从平台抓 3 条真实爆款，提炼共性（标题结构 / 开篇钩子 / 表情符号密度），构造 few-shot prompt 让 AI 写出符合平台调性的新文案。W1 整周的核心方法。

🔬 Lab 实操

学员选定一个垂类（美妆 / 美食 / 职场任选）→ 抓 3 条同垂类爆款 → 喂给 AI → 让 AI 写 5 条新候选。Lab 自动检测 few-shot 例子是否结构一致、是否覆盖足够风格特征。

🤖 例子质量自动校验
📋 作业

用 few-shot 在自己主修平台生成 30 条选题候选，挑 5 条贴到群里。

L04 · Lab

Role-playing — 扮演 30 万粉小红书博主

prompt 30min
📖 教学内容

角色 = 视角 + 语气 + 知识库三个维度。让 AI 化身平台垂类博主：从语气到选题到结尾都按博主人设走，避免通用 AI 味。

🔬 Lab 实操

Lab 内置 5 个博主模板（30w 粉小红书美妆 / 抖音剧情号 / B 站知识区 / LinkedIn thought leader / 公众号深度文）。学员选 1 个 → 让 AI 用该角色写一篇当周热门话题。

🤖 角色一致性自动检测
📋 作业

为自己账号写一份"我是谁"role-playing prompt，下周 L10 用得上。

L11 · Lab

Multimodal Prompting — 截图改文案

prompt 30min
📖 教学内容

多模态输入的关键：图 + 文必须互补，不能重复说一遍。截图 / 海报 / 商品图直接喂给 AI，让它根据视觉信息出文案。适合电商、视觉品牌的提速工具。

🔬 Lab 实操

Lab 提供 3 张商品图（食品 / 服饰 / 电子产品）+ 不同上下文（小红书种草 / 朋友圈分享 / 公众号深度），学员练习 5 套截图 + 文字混合 prompt。

🤖 文案 / 图匹配度评分
📋 作业

用自己账号的 1 张产品图 / 工作场景图，跑一遍多模态 prompt。

L12 · Lab

Output Format — 文案 / 分镜表 JSON 输出

prompt 30min
📖 教学内容

结构化输出 = 对接系统 + 批量处理 + 减少幻觉。让 AI 按 JSON 输出，方便对接飞书 / Notion / 后台系统。批量内容生产的基础。

🔬 Lab 实操

学员让 AI 按预定义 schema 输出 30 条选题（含 title / hook / platform / scheduledDate）。Lab 自动校验 JSON 合法性 + 字段完整性，错时自动让 AI 修正。

🤖 JSON schema 自动校验
📋 作业

把自己 W1 选题用 JSON 导出，贴到飞书 / Notion 用作下周排期。

L05 · Lab

小红书钩子公式（爆款标题 + 封面）

prompt NEW 45min
📖 教学内容

小红书钩子三类 — 数字钩 / 反差钩 / 痛点钩，各举 3 个真实爆款案例。配套封面文案模板（一句话钩子 + 30 字以内结论）。

数字钩反差钩痛点钩
🔬 Lab 实操

用三类钩子公式生成 30 条标题候选 → 平台风格筛选（去掉 AI 感重的）→ 按"完播预期"排序，挑 top 5 试发。

🤖 标题风格自动评分（钩子强度 + AI 味）
📋 作业

给自己一个 W1 主题写 10 条候选标题 + 1 张封面文案，挑 1 条真发到小红书 / 微博。

L07 · Lab

抖音脚本三段式（钩子 / 价值 / 转化）

prompt NEW 45min
📖 教学内容

抖音三段式 — 0-3s 钩子 / 3-15s 价值密度 / 15s+ CTA。每段承担什么任务、最忌讳什么。把一篇产品介绍改造成符合算法的脚本。

🔬 Lab 实操

Lab 提供 3 个原始素材（产品介绍 / 服务说明 / 知识科普）→ 学员让 AI 套三段式改写 → Lab 用规则引擎检测 0-3s 是否有钩子词、3-15s 是否有具体数字 / 案例、结尾是否有 CTA。

🤖 三段结构自动校验
📋 作业

用三段式给自己业务写 1 条 30 秒脚本，可以选拍或不拍，至少完成文字脚本。

L08 · Lab

公众号 / LinkedIn 长文骨架

prompt NEW 45min
📖 教学内容

长文 4 段式 — 钩子段 / 故事段 / 论证段 / 结论段。每段角色、最容易在哪段流失读者。用一个核心观点生成完整长文骨架，再填血肉。

🔬 Lab 实操

学员输入 1 个核心观点（150 字内）→ AI 生成 4 段骨架（每段 100 字 placeholder + 写作要点）→ 学员逐段填充。Lab 检测最终 finished 文章是否每段都达到角色要求。

🤖 4 段角色完整度评分
📋 作业

用 4 段式给自己业务写 1 篇 1200-1500 字长文，发公众号 / LinkedIn / 自己 Notion 公开页。

L09 · Lab

视频脚本 + 分镜表

prompt NEW 45min
📖 教学内容

分镜表元素 — 画面 / 旁白 / 时长 / BGM 提示 / 转场缺一不可。一个剧情想法 → AI 生成 8-12 个分镜，可直接给剪辑或交给 Sora / Runway。

🔬 Lab 实操

学员输入剧情 idea + 目标时长（30s / 60s / 90s）→ AI 输出 JSON 格式分镜表 → Lab 校验时长加和、画面描述是否够具体（能直接交给剪辑）、是否覆盖三段式。

🤖 分镜表完整度自动检测
📋 作业

给自己一个内容想法出 1 份完整分镜表（不必拍出来）。

L10 · Lab

品牌口吻一致性

prompt NEW 30min
📖 教学内容

品牌口吻 = 用词 + 句式 + 禁用词三个维度都要写进 system prompt。禁用词比鼓励词更重要 — AI 知道"不能用'家人们'"比"要用专业语气"更可执行。

🔬 Lab 实操

从品牌过往 5 篇内容（学员真实账号）抽取口吻特征（高频词 / 禁用词 / 句长偏好 / emoji 密度）→ 生成可复用的品牌 system prompt → 跑 3 篇新文测试一致性。

👨‍🏫+🤖 AI 自动评 + 老师抽查
📋 作业

把自己账号的 system prompt 贴到学习群，下周用它产出所有内容。

L16 · Quest 实战 — 在你账号上发出第 1 条 AI 协作内容

Quest 不是 Lab。Lab 在平台 iframe 内做（自动校验），Quest 在真实账号做（AI Tutor 通过对话引导 + 验证）。AI 主动带节奏，学员跟着走。

学习目标：在自己真实账号（小红书 / 抖音 / 公众号 / LinkedIn 任选 1）发布 1 条由 AI 协作产出的内容，并记录发布前 24h 的数据基线截图。

成功标准：账号链接 + 内容截图 + 24h 数据截图（曝光 / 点赞 / 评论 / 收藏 / 完播率）三件齐全。

L16 · Quest 详细流程（AI Tutor 主动引导） 60min · chat 模式

Step 1（10min）— 锁定平台 + 数据基线：AI 引导学员选定 1 个平台（基于现有粉丝 / 内容方向）+ 当前粉丝量 + 历史平均互动数据。

Step 2（25min）— 选题 + 写稿 + 调口吻：用 few-shot（L03）+ role-playing（L04）+ brand-voice（L10）三个 lab 串起来。AI 提示坑点：新人常见错误是想做 100% AI 生成的爆款 → 提醒 60% AI + 40% 人工微调。

Step 3（25min）— 视觉 + 发布 + 数据基线：提示视觉工具选择（Midjourney / Nano Banana / Canva）+ 发布时机（小红书晚 8-10 点高峰 / 公众号工作日早 9 点）+ 24h 后回来收数据基线。

验证方式：(1) 学员说明选哪个平台 + 当前粉丝量 + 历史平均互动数据 (2) 贴出 AI 生成的 3 条选题 + 学员选定的 1 条 (3) 贴出最终文稿全文 (4) 截图视觉素材 (5) 发布后链接 + 截图 (6) 曝光 / 互动数据截图。

W1 周末作业 · 我的 AI 内容 SOP DUE · W2 开课前

交付物：1 份「我的 AI 内容 SOP」(PDF / Notion 公开页 / Google Doc 任选)，包含：

  • (1) 品牌口吻 system prompt（来自 L10）
  • (2) 30 条选题候选表（用 few-shot 生成，标注哪 5 条值得真发）
  • (3) 1 篇完整内容（图 / 文 / 短视频脚本任选 1，必须用上 W1 至少 2 个 lab 的方法）
  • (4) 发布数据基线截图（来自 Quest L16 的 24h 数据）

评分标准：

  • 🤖 AI 评：是否完整覆盖 4 件交付物 / SOP 文档结构是否清晰 / prompt 是否包含三要素
  • 👨‍🏫 老师评（教学版 + 陪跑版）：内容质量 / 角色一致性 / 是否能直接复用
  • 👥 同学互评（Clinic 上）：钩子强度 / 信息密度 / 视觉吸引

承诺红线：作业评估只看「过程性结果」 — 是否完成 SOP / 是否产出 1 篇真实内容 / 是否拿到数据基线。不评估账号涨粉数 / 互动量是否爆款。
