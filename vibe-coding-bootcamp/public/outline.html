Phase 1 — Vibe Coding 入门 & PRD Week 1-2 · 9 节课
L01

什么是 Vibe Coding？未来的开发范式

直播 · 60min

Vibe Coding 不是传统的逐行写代码，而是用自然语言描述你要什么，AI 帮你实现。你负责架构决策和质量把关，AI 负责写代码和调 bug。这节课讲 Vibe Coding 为什么是未来的开发范式、和传统开发的本质区别、以及 AI Coding Agent 正在怎样改变软件工程的工作方式。

L02

安装与配置 Claude、Cursor、Codex CLI

直播 · 120min

三大工具安装配置：Claude Code（Anthropic 官方 CLI，最强的 Agent 编程工具）、Cursor（VS Code 改的 AI 编辑器，补全和对话都好用）、Codex CLI（OpenAI 的命令行工具）。从安装到配置 API Key 到第一次运行，三个工具都跑通。

L03

Claude、Codex、Cursor、Gemini 对比指南

图文 · 120min

Claude Code 擅长大项目多文件编辑和 Agent 式自主完成任务；Cursor 擅长实时补全和快速迭代单文件；Codex CLI 擅长命令行一条指令搞定；Gemini 在 Google 生态和多模态场景有优势。四工具对比实验。

L04

从需求到 PRD 的 AI 产品设计流程

直播 · 90min

PRD（Product Requirement Document）是项目的蓝图。用 AI 辅助 PRD 的完整流程：从一句话需求 → Claude 帮你展开成功能列表 → 拆成模块 → 定义每个模块的输入输出 → 排优先级。

L05

用 Claude 自动生成 PRD 与模块拆解

直播 · 120min

给 Claude 你的产品需求，让它自动生成完整 PRD：功能模块拆解、技术栈建议、数据库 Schema 草案、API Endpoint 列表、前端页面结构。你审核、修改、补充。

L06

PRD 模板与 Claude Prompt 参考

图文 · 120min

6 个经过验证的 PRD/开发 Prompt 模板：需求分析模板、模块拆解模板、代码生成模板、Code Review 模板、Bug Fix 模板、重构模板。每个模板有具体的变量位和示例。

L07

Prompt Engineering 与 Context Engineering 实践

直播 · 90min

Prompt Engineering 管单条指令质量，Context Engineering 管 AI 能看到的所有信息。CLAUDE.md、.cursorrules、项目结构设计。

L08

Claude 协助 Debug：从报错到修复

直播 · 120min

Claude 辅助 Debug 的最佳实践：完整错误堆栈 + 操作上下文 + 先分析原因再给方案。Cursor inline debug vs Claude Code 对比。

L09

Claude Prompt 模板与上下文优化技巧

图文 · 120min

上下文窗口管理策略、大项目的上下文分层策略（全局规范 → 模块规范 → 任务指令）。System Prompt 最佳写法。
