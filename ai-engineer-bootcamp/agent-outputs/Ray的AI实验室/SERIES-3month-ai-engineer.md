## 12 周选题主线（对齐 bootcamp 10 个 phase）

> 每行：周次 · 对应 bootcamp 模块 → 标题 / 痛点角度 / 第一句 hook / 引流钩子（评论区扣关键词）/ 支柱

### Ep.0 · 开篇引流（免费学习路径种草）→ 完整稿见文末
对应：bootcamp Phase 0「LLM API 入门」前置 + free resources
标题：零基础转 AI Engineer 必看！从 0 到 1 入门
支柱：② 入门指南

### W1 · 环境 + LLM API 入门（Phase 1 前半）
- 标题：转 AI 第一周别急着学模型，先搭好这套「工作台」
- 角度：很多人一上来啃理论，结果环境都没配好就放弃
- hook：我带过的转码人，卡在第一周的，八成不是学不会，是工具没装明白
- 引流：评论区扣【工作台】，发你一份《AI Engineer 第一周环境配置清单》
- 支柱：② 入门指南

### W2 · AI/ML/Transformer 基础（Phase 1 后半）
- 标题：转 AI Engineer 要不要先啃数学和深度学习？过来人劝你别
- 角度：被"先学完机器学习"劝退，其实 AI Engineer 用到的基础够用就行
- hook：我转的时候也以为要先补三个月数学，后来发现完全搞反了
- 引流：评论区扣【基础】，发你一份《AI Engineer 真正要会的基础 vs 可以先跳过的》
- 支柱：⑤ 避坑/纠错

### W3 · Prompt + Context Engineering（Phase 2）
- 标题：2026 还在背 prompt 模板？真正值钱的是 Context Engineering
- 角度：prompt 模板满天飞，但工程岗看的是你会不会把上下文当系统设计
- hook：面试官不会问你"写过什么 prompt"，会问你"上下文怎么组织的"
- 引流：评论区扣【Context】，发你一份《从 Prompt 到 Context Engineering 的进阶路线》
- 支柱：④ 技能拆解

### W4 · RAG 入门（Phase 3 前段）
- 标题：RAG 是什么？为什么它是 AI Engineer 的看家本领
- 角度：人人都听过 RAG，但能讲清楚 chunk→embedding→检索→拼 context 的没几个
- hook：把 RAG 拆开看，全是"数据"的活，有数据底子的人偷着乐
- 引流：评论区扣【RAG】，发你一份《RAG 从 0 到跑通的最小路径》
- 支柱：④ 技能拆解

### W5 · RAG 进阶（LangChain / GraphRAG / 向量库，Phase 3 中段）
- 标题：光会切块 embedding 不够，生产级 RAG 还差这几样
- 角度：demo 级 RAG 谁都能搭，但能上线的要会 LangChain、向量库、GraphRAG
- hook：你的 RAG 在 demo 里很准，一上真实数据就拉胯，问题大概率在这
- 引流：评论区扣【生产RAG】，发你一份《生产级 RAG 技术栈清单》
- 支柱：④ 技能拆解

### W6 · RAG 评估 + 部署（Evals / Bedrock，Phase 3 后段）
- 标题：你的 RAG 到底准不准？不会 eval 等于白做
- 角度：最多人跳过、却最值钱的一块——能用数据证明你搭的东西靠谱
- hook：2026 大家都能搭 RAG，但"能证明它靠谱"的人，公司是真的抢
- 引流：评论区扣【eval】，发你一份《RAG 评估工具清单：RAGAS / DeepEval 怎么用》
- 支柱：④ 技能拆解

### W7 · Function Calling + MCP（Phase 4）
- 标题：2026 JD 开始点名要 MCP，这到底是什么、怎么上手
- 角度：MCP 成事实标准，JD 高频出现，但很多人还没搞懂它解决什么
- hook：去年卷 RAG，今年 JD 里冒出来一个词叫 MCP，你得知道它是干嘛的
- 引流：评论区扣【MCP】，发你一份《MCP 从理解到发布第一个 Server 的路线》
- 支柱：④ 技能拆解
- 借势：MCP 月下载量已破亿级、主流厂商全部支持

### W8 · AI Agent 基础（ReAct / Agent SDK，Phase 5）
- 标题：从 RAG 到 Agent，你的 AI 开始会自己「干活」了
- 角度：会调 API 只是起点，能编排多步任务的 agent 才是 AI Engineer 的分水岭
- hook：RAG 让 AI 会查资料，Agent 让它会自己动手——这一步跨过去身价不一样
- 引流：评论区扣【Agent】，发你一份《第一个工具调用 Agent 怎么搭》
- 支柱：④ 技能拆解

### W9 · Multi-Agent + 记忆（LangGraph / A2A / Mem0，Phase 6+7）
- 标题：单个 Agent 不够看，2026 热点是多 Agent 协作
- 角度：单 agent 玩具化，多 agent 编排（LangGraph）+ 长期记忆才是生产形态
- hook：面试问"你做过 agent 吗"已经不够了，现在问"多个 agent 怎么协作、怎么记住上下文"
- 引流：评论区扣【多Agent】，发你一份《Multi-Agent 编排 + Agent Memory 入门》
- 支柱：④ 技能拆解

### W10 · Claude Code / Harness / Skills（Phase 8）
- 标题：会用 Claude Code、还能自己搭 harness，是新的分水岭
- 角度：2026 工程效率新标尺——不只是用 AI 写代码，是懂它怎么跑、能定制
- hook：现在团队招人，悄悄在看一件事：你是会用 Claude Code，还是只会复制它的输出
- 引流：评论区扣【ClaudeCode】，发你一份《Claude Code / Harness / Skills 上手路线》
- 支柱：④ 技能拆解

### W11 · 部署 + 开源模型 + 微调（Docker / Ollama / QLoRA，Phase 9）
- 标题：从 demo 到 production 的最后一公里：部署、Docker、微调
- 角度：很多人停在"跑通了"，但能 Docker 化、上线、按需微调的人才进得了团队
- hook：你简历写"做过 AI 项目"，面试官只会追一句：部署上线了吗、遇到什么坑
- 引流：评论区扣【上线】，发你一份《AI 项目从本地到 production 的部署清单》
- 支柱：④ 技能拆解

### W12 · Evals + 安全 + 求职（P3 / 简历 / 面试，Phase 10）
- 标题：技术学完了，简历怎么改、岗怎么投、面试考什么
- 角度：会做 ≠ 能拿 offer，最后一段是把项目讲成简历、踩准 JD 关键词、过面试
- hook：我见过太多人技术够了却投不动，问题全在简历没 match JD、面试讲不清项目
- 引流：评论区扣【投递】，发你一份《2026 AI 岗关键词清单 + 简历必出现的技术栈词》
- 支柱：① 行业内幕 + ⑤ 避坑/纠错
