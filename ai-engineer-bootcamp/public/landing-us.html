能力差距

会调 API ≠ AI Engineer

2025-2026 招聘方问的不再是「你会不会调 OpenAI」，而是「你能不能交付一套可靠、可观测、控成本、扛得住生产流量的系统」。这中间隔着四层：

1
API demo — 调通一个接口，跑出一段回答。（大多数人停在这）
2
RAG pipeline — chunking、向量检索、混合检索 + rerank，把知识接进模型。
3
Production eval — RAGAS 四指标、LLM-as-a-judge，能量化质量、在上线前抓回归。
4
Observability + 成本控制 — LangSmith / Langfuse trace、延迟、token 成本——这才是 JD 真正要的人。

这门课从第 1 层带你走到第 4 层，外加 Agent / Multi-Agent / MCP / Fine-Tuning 全栈。
