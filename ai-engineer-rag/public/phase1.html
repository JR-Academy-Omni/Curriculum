Part 1 学完目标

这一部分不教代码，也不讲数学。目标是让你看清楚整个 AI Engineer 的地图——知道 GenAI 能做什么、Prompt / RAG / Agent / LLMOps 各自在解决什么问题。

学完 Part 1 你应该能：

  • 识别一个真实业务需求该用 Prompt Engineering / RAG / Fine-tune / Agent 中的哪个
  • 用四要素框架（角色 + 任务 + 约束 + 例子）写出专业级 Prompt
  • 理解企业级 LLM 应用的整体架构和 LLMOps 核心能力
  • 搭好开发环境，跑通第一个 OpenAI API 请求
