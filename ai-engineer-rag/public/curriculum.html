这门课解决什么问题？

市面上讲"怎么用 ChatGPT"的课很多，讲"企业级 RAG 怎么落地"的课很少。更少的课会把 LLM 底层原理（Transformer / Embedding / Tokenizer）+ Prompt 工程 + RAG 架构 + 生产部署一次讲透。

这门课是 JR Academy AI Engineer Bootcamp 的前置入门课。49 节视频、3 个渐进式 Project——从 GPT Store 发布开始，过渡到 Python 从零搭 RAG，最终部署一个能处理 PDF 上传的生产级 RAG 应用。

谁适合学？

软件工程师、后端 / 全栈、DevOps / Cloud、数据工程师，想转型 AI Engineer 或想搞懂 RAG 架构的技术从业者

前置要求

基础 Python（能写函数、调 API、用 pip）。不需要 ML / DL 背景，ML 章节是普及向，帮你看懂 LLM 原理

学完交付物

✅ 发布的 GPT（GPT Store 链接）
✅ 从零写的 Python RAG 仓库
✅ 可上线的 PDF RAG Streamlit 应用

常见问题
没学过机器学习 / 深度学习能学吗？
完全可以。课程的 ML/DL/NLP 章节（L09-L17）是普及向，目的是帮你看懂 LLM 的原理，不需要你训练模型或写反向传播。
需要多强的 Python 基础？
能写函数、用 pip 装包、调用 HTTP API 即可。后半段 RAG 项目会用到 Python，但不需要深入的框架或异步编程经验。
这门课和 AI Engineer Bootcamp 什么区别？
本课程是 4 周视频入门课（前置知识体系），AI Engineer Bootcamp 是 12 周深度实战营（含直播、P3 项目、求职服务）。学完这门课再进 Bootcamp 不会掉队。
学完能做什么？
能独立搭建基础 RAG 原型、看懂企业级 RAG 架构图、理解 LLM 能力边界、为 Agent / 微调 / MCP 等进阶主题做好准备。不教你训大模型。
RAG 项目需要自己买服务器 / GPU 吗？
不需要。三个项目分别跑在 GPT Store、本地 Python 和 Streamlit + Azure OpenAI，全程调云 API，无需本地 GPU。
视频更新频率？
课程随技术演进持续更新。重大模型 / 框架变动（如新一代 embedding 模型、LangChain LCEL 改动）会补充增补视频。
