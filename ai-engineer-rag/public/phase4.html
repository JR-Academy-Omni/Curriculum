Part 4 学完目标

Part 4 是整个课程的压轴戏，也是最长的一部分（20 节课）。之前所有的铺垫——LLM 原理、Embedding、Prompt 技巧——都是为了让你能真正动手构建一个可以上线的 RAG 应用。

这部分会讲透 RAG 工程的方方面面：架构图 → 嵌入模型选型 → 向量数据库 → 朴素 RAG 的坑 → Advanced / Agentic / Contextual RAG → PDF 解析 → Rate Limit → 成本 → Azure 和 AWS 的云端部署。

学完 Part 4 你应该能：

  • 画出一个企业级 RAG 系统的架构图并解释每个组件
  • 不用 LangChain / LlamaIndex，纯 Python 从零实现 RAG
  • 部署一个能处理 PDF 上传的 Streamlit + Azure OpenAI RAG 应用
  • 知道怎么算 Token 成本、怎么处理 Rate Limit、怎么选向量库
  • 看懂 AWS Bedrock Knowledge Base 的全托管 RAG 方案
