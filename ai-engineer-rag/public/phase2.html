为什么要学这些"理论"？

很多人学 LLM 直接从 Prompt / RAG 开始，遇到问题就调包。这门课不这样——Part 2 是 8 小时的底层原理速通，让你知道 Transformer 为什么替代了 RNN、Embedding 到底是什么、为什么上下文窗口不能无限扩大。

学完 Part 2 你应该能：

  • 看懂机器学习 / 深度学习 / NLP 的基本术语，不再是黑盒
  • 解释 Self-Attention 为什么能并行计算、为什么能抓长距离依赖
  • 理解 Token / Embedding / Positional Encoding 的关系，知道 context window 的本质
  • 说清楚 Encoder-only（BERT）/ Decoder-only（GPT）/ Encoder-Decoder（T5）的差异

⚠️ 给没学过 ML 的朋友：这部分是普及向，只讲"直观理解 + 核心术语"，不推公式、不写反向传播。有 Python 基础就能听懂。
