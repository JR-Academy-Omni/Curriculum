常见问题

报名前看一眼 · 还有问题给我们邮箱：hello@jiangren.com.au

我完全不懂技术，没用过 ChatGPT，能跟得上吗？

能。这个 workshop 就是为零技术基础的小老板和自由职业者设计的。Cowork 全程图形界面，没有终端、没有命令行、不写代码 — 你只要会打字会用 Excel 就能跟上。每一步老师等所有人完成才往下走，1:4 Tutor 巡场救场。课前如果你完全没碰过 AI 工具，可以提前 5 分钟玩一下 ChatGPT 或 claude.ai 找下『跟 AI 说话』的手感。

Claude Cowork 必须付费吗？我不付费能上这节课吗？

Cowork 必须 Claude Pro / Max / Team / Enterprise 付费档（A$30/月起），免费档没有 Cowork tab。课上没付费档的可以用 JR 共享测试号过一遍 4 小时，但回家继续用必须自己订。说实话 A$30/月一份咖啡钱换一个不睡觉的助理 — 课上做完报价单 + 邮件批量回复 + 一周内容那一刻，你大概率自己就订了。

我业务数据很敏感（客户身份证、合同金额、内部财务），能用 Cowork 吗？

Cowork 干每件事前都会让你 Approve（除非你在 Settings 关了 — 不建议关）。文件读写默认只在你指定的 working folder 内，跨文件夹要单独授权。Connector 走 OAuth 权限细到 scope。但敏感数据（客户身份证 / 银行账户 / 未公开合同金额）上传前还是建议用 [客户A] [金额X] 打码 — 这是好习惯不是 Cowork 的锅。如果你是律师 / 会计 / 医疗背景需要 100% 数据本地，那个场景应该看 OpenClaw Workshop。

Mac / Win 都行吗？iPad 行不行？

Mac (macOS 11+) 和 Win 10+ 都行。Linux 暂时没 Cowork 桌面 App。iPad / 手机不行 — 主战场必须是桌面（手机 Claude App 只能做 Dispatch 派活儿用）。Win 用户务必下 MSIX 安装包，老的 .exe 没 Cowork tab — 这是踩坑率最高的点。

课前要准备什么？

6 件事：① 自带笔记本（Mac 11+ 或 Win 10+，不能 iPad）② 注册 Anthropic 账号 + Claude Pro 订阅 ③ 桌面建空文件夹 ~/Desktop/cowork-workshop ④ 准备一个个人 Gmail ⑤ 带你常用的报价/发票/合同模板（Excel 或 Word）⑥ 想清楚一个真实业务场景。详见上面课前清单。

/schedule 我配好周一早 8 点跑周报，那时候我电脑关机了怎么办？

🚨 跑不动。Cowork Scheduled Task 不是云端 daemon — 必须电脑 awake + Cowork app 开着才跑。睡眠期间触发的会跳过，电脑醒了 + Cowork 重新打开会自动补跑一次（你会收通知）。3 个解决方案：① 笔记本就让它常开（Mac sleep + display off 算 awake，看 Settings → Energy）② 装家里 / 公司常开的台式机 ③ 不需要每周必跑的就接受『偶尔跳一次』。

我做餐饮（不是 IT 行业），4 小时学这个划算吗？

划算。我们办过几期 SME workshop，餐饮老板回家做的事：① 用 my-quote 给周一企业团餐自动报价（之前 30 分钟手填模板，现在 3 分钟）② 用 my-inbox-triage 清外卖平台投诉邮件（一周省 2 小时）③ 用 weekly-content 出小红书 / 抖音文案（之前外包月 A$500）④ 用 weekly-report 看销售数据。一个月省下时间 + 省下外包费，A$129 课程费 + A$30/月 Pro 大概一周回本。

课后有支持吗？录播？答疑群？

3 件兜底：① 课程录播开课后 3 天内发学员（中文字幕）② 老师 5 个 Skill 模板 + 4 个 Schedule 配置打包发学员 ③ JR Academy 平台 30 天试用（27+ Prompt Lab 互动练习 + Vibe Coding Hub）。课后 30 天遇到卡点可以邮件给老师，30 天后想要长期社群转 AI Adoption Bootcamp 学员社群。

我已经在用 ChatGPT Plus 了，再订 Claude Pro 是不是重复花钱？

不重复。两个产品定位不一样：ChatGPT 主要『跟 AI 聊天 / 问问题 / 写文字』 — 你打开网页跟它对话。Cowork 是『让 AI 替你动鼠标干电脑上的事』 — 它真的会开你 Excel / 改你 PPT / 抓网页 / 发邮件 / 定时跑。本节课 4 个 case 全部是 ChatGPT 做不了的（ChatGPT 没有 Computer Use + 没有桌面 App + 没有 Schedule）。
