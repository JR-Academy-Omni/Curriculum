🎯 Social Media 阵地：小红书 — OPC 最大单一中文获客渠道

2026 年小红书月活已经 3 亿+，澳洲华人专业人士小红书渗透率 60%+。但小红书的内容逻辑跟公众号、X、LinkedIn 完全不同：必须竖图、必须 7 字以内的钩子标题、必须前 3 秒 hook、必须留私信钩子（公开评论限流）。课上拆 3-5 个澳洲华人 indie 在小红书的真实账号，逆向他们的爆款公式。

[Live] AI 视频制作：从脚本到成片，一键生成中英文短视频

🎯 视频是 2026 年所有平台最大流量入口 — AI 让你绕过剪辑技能

视频号 / 抖音 / 小红书短视频 / X 视频帖 / LinkedIn video / Podcast 切片，全是视频。但 OPC 学员一个人没时间学剪辑。这节课教你用 AI 把"脚本 → 画面 → 配音 → 数字人 → 字幕 → 多平台适配"全流程自动化，30 分钟出一条 60s 成片。

2026 主流 AI 视频工具栈实操：①Sora 2（OpenAI 文生视频，60s 高质量）+ Veo 3（Google）+ Runway Gen-4（风格化）+ Kling 2.0（快手中文生态）+ Pika 2.0+ Luma Dream Machine（影视级运镜）；②Heygen / D-ID AI 数字人 — 一个虚拟形象多语言对口型（英文 LinkedIn / 中文视频号 / 日语小红书一次出三版）；③ElevenLabs 配音 + Suno BGM；④CapCut + AI 自动剪辑 + 智能竖屏裁剪（视频号 9:16 / X 16:9 / 小红书 3:4）。

现场产出 4 类视频：①30s 产品 demo（给 Stripe checkout 演示）；②60s founder story（给 X / LinkedIn build in public）；③15s hook 视频（给小红书 / 视频号 / 抖音）；④数字人讲解（给 Podcast pitching / cold outreach）。每类配中英文双版本。

[Live] LinkedIn outreach + ProductHunt 冷启动

🎯 LinkedIn 是 B2B SaaS 最大冷启动渠道

但你不能直接 spam。LinkedIn outreach 的高转化公式 = 个人化第一句（提到对方最近的发帖 / 公司 / 文章）+ 你能给他什么具体价值（不是要他什么）+ 不要求会议、给一个低门槛 ask（"看 30 秒 demo 视频"）。课上演示用 Claude 批量生成 100 条个人化 outreach 的全流程。ProductHunt 冷启动是另一个杠杆：发布日的前 24 小时决定 90% 排名，要在墨尔本时间凌晨 6 点 launch（对应 PT 12:01 AM），动员 5-10 个朋友 upvote。

[Self] Reddit / IndieHackers 社区策略

🎯 不要发 spam，要发"价值帖"

Reddit / IndieHackers 社区对 self-promotion 极度敏感（一发被删 + ban），但对"我用 X 工具解决了 Y 问题"这种价值帖欢迎度高。公式：90% 价值 + 10% 软提产品。课上发 20 个真实"价值帖 → 转化 100+ 用户"案例分析。

🎯 一节课讲透中英文圈所有线下渠道 — AI 帮你从策划到执行全包

线上内容只是入口，OPC 学员被低估的非对称优势是线下。这节课按渠道类型把 6 个一次讲完（不按语言切，不同语言的同类渠道打法是通的）：

② Community / 私域裂变：微信群（公众号软文 + 邀请奖励 + KOL 转发，一周 0 → 200 精准用户）+ Discord/Slack 行业社群（潜伏 3 周再发，长尾流量）。

③ Press / Podcast / Newsletter feature：澳洲华人媒体 PR（澳洲日报 / 微悉尼 / 今日墨尔本 / 澳洲新闻网投稿）+ Podcast pitching（IndieHackers / Lenny / Solopreneur 类）+ Founder feature（The Hustle / IndieHackers Newsletter / BetaList / Tools for Humans 投稿）。

AI 在线下营销的 6 个具体动作（一套打 6 渠道）：①AI 找渠道（meetup / podcast / 媒体记者列表抓取 + 匹配度排序）；②AI 写 pitch（个人化邮件 + 演讲提案 + 投稿稿）；③AI 准备（演讲稿 + 模拟主持人采访 30min Q&A 训练）；④AI 出 asset（海报 + 朋友圈文案 + 引流落地页）；⑤AI 跟进（活动后 follow-up + 加好友 message + 长期 nurture）；⑥AI 反哺线上（一次活动拆成 5 篇小红书 + 1 篇公众号 + 5 条 X + 1 篇 LinkedIn 长帖 + 1 篇 SEO 博客）。

🎯 W6 SEO 周：传统 SEO + AI 搜索时代的 GEO 双线

传统 Google SEO 还有效，但 ChatGPT search / Perplexity / Claude search 已经分走 15-20% 搜索流量。两套优化逻辑：①传统 SEO 强 long-tail keyword + technical SEO + backlink；②GEO（Generative Engine Optimization）强 cite-worthy 内容 + 结构化数据 + entity authority。课上拆 jiangren.com.au 真实 GSC 数据演示如何同时做这两件事。
