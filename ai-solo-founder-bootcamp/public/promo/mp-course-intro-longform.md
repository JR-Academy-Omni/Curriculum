# 我们在墨尔本、悉尼、布里斯班同时开了一门课：用 AI 做一人公司，11 周后毕业你手里至少有 ABN + 一笔真实付款

如果你最近用过 Cursor 写代码、用过 Claude Code 跑完整项目、看过 Lovable 一句话生出 SaaS，你大概率有过这个念头：

**"既然一个人能做出过去一个团队的活儿，那我为什么还在打工？"**

这不是焦虑，是数据。Pieter Levels（@levelsio）一个人 12 个产品在跑、年化 $3M+；Marc Lou 21 个产品在跑、月入 $5 万美金；澳洲本地不少程序员、PM、设计师已经在偷偷 ship——他们没辞职，但 Stripe 后台已经在跳数字了。

可是大部分人卡在同一个点：**工具会用、想法有、行业洞察够，但从 0 到第一笔真实付款这条路上，没人一对一带过你走完一遍。**

我们做了一门课专门解决这件事。叫**AI 一人创业营**——澳洲华人专属、三城线下、11 周打包，从选品到 Demo Day 一条龙带你跑完一次。

下面我把这门课的来龙去脉拆给你看，包括为什么 2026 年才能做、它到底有什么是别的课没有的、不收谁、怎么报名。

## 一、为什么是这门课，为什么是现在

匠人学院过去几年开了 5 门主力 Bootcamp：AI Engineer / AI Adoption / OpenClaw / Vibe Coding / AI Essentials。我们扫过一遍，发现**所有这些课的出口都是"继续在某家公司里干活"**——升职、换工作、内部转型、个人提效，再不济也是技能升级。

**没有一条出口是"自己开公司、卖产品、拿 Grant"。**

这不是匠人独有的盲点。看全球：

| 同类产品 | 定价 | 差异点 |
|---------|------|-------|
| Reforge / On Deck | USD $4-7k | 英文 / 远程 / 不教 AI 工具栈 / 不含澳洲 Grant |
| YC Startup School | 免费 | 万人级科普 / 非 cohort / 没有落地陪跑 |
| 生财有术 | RMB 3,000-5,000/年 | 社群型内容 / 不教 AI 产品 / 内容碎片 |
| Antler / Startmate | 拿股份 | VC track only / 英文 / 不教 OPC（一人公司） |

**没有一门课同时具备**：用中文教、教澳洲华人专业人士、教 AI 工具栈做一人公司、教拿澳洲 Grant、三城线下 networking。

这是我们做这门课的硬条件。三件事同时成立：

1. **技术拐点真实** — 2024 工具还不够稳定，2025-2026 Cursor / Claude Code / Lovable / v0 / Sora 这一波真的让"非工程师一周 ship 一个 SaaS"成为常态。窗口期内你是 founder，窗口关后你是 follower。
2. **焦虑供给侧真实** — 澳洲华人 35-45 岁拿 150-250k 薪水的中产专业人士，天花板焦虑不是表面的"被 AI 替代"，是"我打工的尽头在哪、能不能不打工"。
3. **澳洲合规真实** — RDTI 43.5% 退税、EMDG 出口补贴、Sole trader vs Pty Ltd 决策——这些是真金白银，但本地华人没有中文课讲怎么操作。

## 二、5 大亮点：每一条都查得到出处

### 亮点 1 · 三城线下 Hub-and-Spoke · 不是 Zoom 大水课

这门课不是录播、不是 Zoom 群讲，是**三城同步线下**：

- **墨尔本**：主场教室（嘉宾物理出席）
- **悉尼**：卫星教室（直播 + 本地 Tutor + 本地校友 networking 晚宴）
- **布里斯班**：卫星教室（同上）

嘉宾飞主场城市，其他两城听直播 + 本地 Tutor 现场陪练。**Demo Day 三城联合直播 + 各城本地 networking 晚宴**——这是匠人 2020 年前就跑通过的模式，不是凭空设计的。

为什么必须线下？因为 **cohort 文化是这门课的核心资产**，下面亮点 5 会展开。线上同学只是同学，线下同学才是 5 年后澳洲华人 AI 圈的核心节点。

### 亮点 2 · 10 类 Faculty 顾问团 · 单独咨询市场价 $43-176k AUD / 年

这门课的师资不是"一个老师讲到底"，是 **10 类领域专家联合授课**：

1. 麦肯锡 / 战略顾问
2. SEO / GEO 专家
3. Growth / Marketing 专家
4. Senior eng / 微软 / 大厂资深工程师
5. AI 工程 / Stripe / 自动化
6. 澳洲华人 CPA / CA
7. Grant consultant（5+ 客户拿过 RDTI / EMDG）
8. 投资人 / Local VC
9. 已退出 founder 校友
10. AI Engineer / LLM 实战

如果你单独找这 10 类顾问做 1-3 个月咨询，市场价 **$43-176k AUD / 年**。这门课里他们轮番站台，部分还在 W11 Demo Day 现场做评审。

**实话**：首期我们还在锁定具体讲师，不会假装 Logo 已经全部到位。但顾问团**类别**已经定，部分人选已经签确认函（包括 Roger 等已实名嘉宾，详见课程详情页 §08 顾问团区块）。

### 亮点 3 · Phase 3 澳洲 RDTI 实战 · 全球独家 3 周

这是其他课**真的没有**的模块。我们在 Phase 3（W7-W9）整整三周教澳洲合规：

- **W7 财务自动化**：Xero + Notion ERP，把 P&L / 现金流跑起来
- **W8 澳洲会计**：澳洲华人 CPA 嘉宾**现场陪你注册 ABN**（毕业硬指标 #3）。Sole trader vs Pty Ltd 决策树、GST / BAS 实操、不是 PPT 念条款是现场操作。
- **W9 Grant 实战**：Grant consultant 嘉宾教 **RDTI 43.5% 退税申请实操**。RDTI 是澳洲政府针对 R&D 支出的退税政策，符合资格的项目可以拿 43.5% 退税（具体能拿到多少看项目和审计，不承诺金额）。这门课的毕业硬指标 #4 是 **Grant 申请提交**——不是教完就走，是真的提交一次。

国外没有 RDTI（地理隔离），国内不教（用不到），生财有术 / 出海帮也不讲（不针对澳洲）。这 3 周是全球独家。

### 亮点 4 · 6 项毕业硬指标 · 不是结业证，是手里有真东西

很多 Bootcamp 毕业 = 一张 PDF 证书。这门课毕业 = 你手里同时具备这 6 样：

1. **$1+ 真实付费** — 不接受朋友友情付，必须是陌生用户的真实付款（Stripe / PayPal 后台可查）
2. **ABN 注册完成**
3. **Grant 申请提交**（拿到与否不一定，但必须提交一次）
4. **产品 URL 上线** — 公网可访问，不是 Figma 静态页
5. **10+ 内容资产** — 公众号 / 小红书 / X / LinkedIn 任选，做 SEO + 个人品牌
6. **Demo Day 出席** — 三城联合直播 + 现场评审

**11 周后你手里是产品 + 真实付款截图 + ABN + Grant 申请回执，不是一张毕业证**。这是匠人对首期 case study 的承诺。

### 亮点 5 · Cohort 经济 · 24 个同学不是"同学"，是 5 年商业网络

首期 cohort 招 18-25 人，外加跨期校友。这群人的潜在网络交易额我们做过测算：

| 互动模式 | 单次价值 | 24 人 × 5 年估算 |
|---------|---------|-----------------|
| 互为早期客户 | $500-5,000 | 你的 SaaS 卖给其他同学 |
| 互为服务采购 | $2k-20k | 律师同学帮你 review TOS / 设计师同学帮你做品牌 |
| 互为天使投资 | $5-50k | angel check 互投 |
| 互为客户引荐 | $5-50k value | 一次引荐 = 一笔大单 |

总潜在交易额 **$240k-1.9M 之间**（按 24 人互动密度估算）。

这就是为什么我们坚持 **18-25 人小班** + **三城线下 networking 晚宴** + **6 个月校友 Office Hour**——同学之间能不能"想起对方做什么"决定了这门课 5 年后的真实价值。

学校里大家都说"同学网络是最大的财富"，但你毕业后大概率没再联系过 90% 的同学。这门课的 cohort 设计**强制建立交易关系**——W11 有"我能给的 / 我要的"卡片互换机制，落地了再走。

## 三、11 周课程结构速览

```
PHASE 1 · AI Build & Sell（W0-W3）
└─ 选品调研 → MVP 上线 → 接入 Stripe → 跑第一笔真实付费
   工具：Cursor / Claude Code / Lovable / v0 / Stripe / Notion

PHASE 2 · AI Marketing OS（W4-W6）
└─ 中英文内容机器 + SEO / GEO + 社媒矩阵
   工具：公众号 / 小红书 / X / LinkedIn + AI Marketing 自动化栈

PHASE 3 · AI × 澳洲合规（W7-W9）🇦🇺 全球独家
└─ Xero + Notion ERP + 注册 ABN（CPA 陪写）+ RDTI 申请（Grant consultant 陪写）

PHASE 4 · AI Demo Day（W10-W11）
├─ W10 选修：VC fireside chat（本地 VC 2 位嘉宾，OPC vs VC 路线决策）
└─ W11 Demo Day 三城联合直播 + 嘉宾评审 + 校友 networking 晚宴
```

共 **49 节课 · 26 直播 · 92 小时** · 8 个 Lab 互动实验。

## 四、谁该来，谁绝对不该来

申请制 + 单价 $2,400-$3,800 AUD 已经筛掉两道。还有几条硬红线：

**该来的人**：

- **焦虑型 senior 程序员**（45%）：35-45 岁、Atlassian / Canva / Telstra / Optus / Airwallex / 四大行拿 150-250k+、已用 Cursor / Claude Code 在内部干活、看到 AI 替代趋势、手里有 6-12 个月生活费缓冲敢辞职或转兼职过渡
- **高薪 PM / 咨询 / Designer**（35%）：40-50 岁、在 Legal / Real Estate / Healthcare / EdTech / Fintech 某垂直泡了 10-15 年、看完 Lovable + Cursor 演示后第一次相信"我可以绕过工程团队"、有强行业洞察
- **副业冲动专业人士**（20%）：律师 / 会计师 / 医生 / 资深咨询、本职收入稳定不打算辞职、想用 AI 做行业 niche SaaS / 知识产品、在乎澳洲税务 + Grant + 合规

**不收的人**（明确排除）：

- ❌ 没积蓄的应届生 / 在校生 — 撑不到第一笔付费就为下个月房租焦虑
- ❌ 没辞职勇气的纯打工者 — 不投入时间，cohort 文化被污染
- ❌ 想拍短视频割韭菜的 — 申请关筛掉
- ❌ 想做币圈 / Token 项目的 — 不在课程范围
- ❌ 不在澳洲且不打算来澳洲的 — Grant + 会计模块对你无用
- ❌ 想做 VC fundable startup 的 — 推荐 Antler / Startmate / Y Combinator

**申请流程**：5 分钟填表 → 30 分钟 1:1 视频面试 → 录取通知。

## 六、为什么我们敢做这门课，为什么你应该慎重决定

**坦白讲**，这门课首期不挣钱。18-25 人 × ~$2,500 均价 ≈ $50-60k 收入，刚好覆盖嘉宾费 + 三城场地 + 运营。我们用首期换 5-8 个真实赚钱的毕业 case study + 完整 SOP，目的是把第二期、第三期、海外复制（加拿大 / 英国 / 新加坡首期）的飞轮转起来。

所以**首期是给真正想做这件事的人留的**。如果你只是看到 AI 热想跟一下、想多个证书、想"了解一下"，请把这个位置让给真正在准备辞职 / 已经在 ship 的人。

**如果你符合下面任意一条**，我建议你今晚就去 [https://jiangren.com.au/curriculum/ai-solo-founder-bootcamp/landing.html](https://jiangren.com.au/curriculum/ai-solo-founder-bootcamp/landing.html) 看完整内容然后申请：

- ✓ 你在大厂 / 咨询 / 律所 / 会计师事务所工作 5 年以上，已经拿到 senior 或 principal level
- ✓ 你过去 6 个月想过"自己做点什么"但每次卡在第一步
- ✓ 你已经在用 Cursor / Claude Code / Lovable 写东西，只是没人带你走完商业化闭环
- ✓ 你有 6-12 个月生活费缓冲，敢真的辞职或转兼职过渡

申请链接在文末。这一期 25 个位置，前 15 个走预售价 $2,400，目前还在开放——但**满 15 即关**，下一档涨到 $2,800。
