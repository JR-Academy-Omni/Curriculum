### 1. 甲方提供（匠人学院）

#### 1.1 联合 Workshop（如适用）

- **场次**：1 场，[日期] [时间] [城市] [线下场地 / 线上 Zoom]
- **时长**：[60 / 90] 分钟（含 [10] 分钟课程介绍 + [50-80] 分钟实战内容 + [10] 分钟 Q&A）
- **讲师**：[主讲老师姓名]，匠人学院 [职位]

- **录像 / Recap**：workshop 录像由甲方剪辑后 [7] 天内发乙方使用（双方品牌共同出现）

#### 1.2 会员专属折扣码（如适用）

- **折扣码**：`[ORG-CODE]-300`（如 `ACBC-300`），仅限 [协会名] 会员使用
- **会员优惠**：报名直减 AUD $[300]（基于 Standard 价 $4,800，约 6.25% off）
- **乙方返佣**：每成交一单返 [10-15%] = 每人 AUD $[480-720]（按 Standard 价）
- **返佣结算周期**：每月 [1] 号，按上月成交单数结算，30 天内转账到乙方指定账户
- **Attribution tracking**：甲方后台记录每个使用折扣码的报名 → 付款 → 录取 → 入学全链路，每月 1 号给乙方发 attribution report

> ⚠️ **NFP 协会特别处理**：如乙方章程不允许商业返佣，转为「协会运营经费赞助」形式 — 甲方按招生数发给乙方 invoice，乙方按 NFP 流程接收。具体由乙方财务确认走哪条路径。

#### 1.3 保留席位 + 创始人体验（如适用 · 方案 C）

- **Reserved seat**：每期 cohort 保留 [2-3] 个名额给 [协会名] 推荐的会员
- **录取保证 / 优先**：被推荐会员享 **优先审核**（24h 内回复，不用排队 — 首期无视频面试，纯后台 review 申请单 + LinkedIn）+ 申请单带 referral code 字段（审核同事知晓推荐来源）— **但仍走申请制，不保证录取**。第二期起加视频面试 + 严筛，到时候 reserved seat 价值更高
- **主理人体验权益**（两个 variant 二选一，**按协会画像**）：
  - **Variant A · 商会 / 行业协会**（律师 / 会计 / 医生 / 行业 senior）：协会会长 / 创始人本人享免费 [4 次 × 30min] 1:1 Office Hour（主讲老师），课程结束后 6 个月内有效
  - **Variant B · Founder community / 创业协会 / Venture Club**：**不**给 1:1 Office Hour — 创业的人喜欢跟创业的人聊，不喜欢听讲师讲。改成 **Founder Network Access**：① Demo Day Networking 晚宴 [2-3] reserved seats（跟 cohort + 校友 + VC + 嘉宾同桌） ② 每季度 [3-5] 个 JR 校友 founder warm intro（双方都同意才约） ③ W10 VC fireside 旁听 + 提问优先 ④ 主理人本人 W10/W11 嘉宾席（画像合适时）
- **会长免费体验**：会长本人可以选择是否参加课程作为旁听（不占 reserved seat 名额，不进 cohort 计数）

#### 1.4 Demo Day 评审席 + Networking（如适用 · 方案 C）

- **评审席**：协会派 [1] 名明星会员 / 资深 founder 担任 W11 Demo Day 评审

### 3. 红线 / 双方都不做的事

- **甲方**：不向乙方任何会员承诺金钱 / 收入结果 · 不承诺录取 · 不接受乙方付 sponsor fee（合作是互利不是甲乙关系） · workshop 不变硬广（课程介绍占比 ≤ 15%）
- **乙方**：不向甲方索取 sponsor fee / 入会费 / 内容审核费 · 不要求独家合作 · 不要求保证录取 N 人 · 不向会员宣传时使用"匠人学院承诺 X 收入 / X 退税" 等违反承诺红线的话术

---

### 4. 数据 / 隐私

- 折扣码 attribution report 数据仅含「使用折扣码的报名数 / 付款数 / 录取数 / 返佣金额」， **不含任何会员个人信息**
- workshop 现场扫码进群的会员，其个人信息归属甲方（用于后续课程 nurture），乙方不获取
- 双方均遵守 Australia Privacy Act 1988

---

### 5. 退出 / 终止

- 任一方提前 [30] 天书面通知，可终止本 MOU
- 终止后：已成交订单的返佣按合同照付到底 · 未发的 workshop 不再举办 · 折扣码自终止日 [7] 天后失效
- 任何一方违反承诺红线 / 数据隐私 / 品牌使用授权 → 另一方有权立即终止并保留追究权利
