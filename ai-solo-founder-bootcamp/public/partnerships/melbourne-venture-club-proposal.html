vs 其他创业课

JR Academy × MVC 08 · 3 Tiers
09 · WORKSHOP AGENDA
09 / 12
90 MIN WORKSHOP · 设计已敲定

Workshop 议程 · 10% 介绍 + 90% 实战

不变成硬广 · MVC 主理人有权监督 · 课程介绍 ≤ 10 分钟

TIMESEGMENT内容
0-5min MVC 主理人开场 介绍这场 workshop 为什么对会员有用 + 介绍讲师背景。这 5 分钟是 MVC 主场，不是讲师 talk over。
5-15min 破冰 + 痛点扫描 3 个举手问题：① 谁已有 ABN？② 谁尝试过 AI 替代员工但没跑通？③ 谁听说过 RDTI 但没申请过？— 现场画像决定后续内容权重。
15-50min ⭐ 实战 1 · 30 分钟 ship AI 产品 现场用 Lovable / Cursor / Claude Code 从需求 → landing → Stripe test 收款 — 屏幕共享，会员同时跟做。出 Stripe test 交易截图证明可跑通。
50-80min ⭐ 实战 2 · RDTI 退税 AI Prompt 用 Claude / GPT 起草一份 R&D activity description（占 RDTI 申请书 80% 工作量）。讲解 4 个 prompt template · 会员用自己业务跑一遍。不承诺拿到，只承诺写一份合规可提交的初稿。
80-90min Q&A + lead magnet 现场发 现场答疑 + 3 个 lead magnet 二维码扫码（OPC 自评 / 30min ship demo / RDTI Prompt 模板）。
90-100min 课程介绍（仅 10 分钟） "如果你想把今天学的扩展成 11 周完整旅程" — 课程一页纸 + MVC 专属折扣码 + 申请入口。10 分钟硬上限。

FOR MVC

MVC 拿到什么

① 免费高质量内容供给（90min 实战 workshop · 不收讲师费 / 场地费）
② 会员实打实福利（每人省 $300 折扣 + 18 人 cohort = MVC 累计拿 $5,400-8,640 持续被动收入）
③ Club 品牌曝光（出现在 workshop / Demo Day 物料 / 社媒回放 / 毕业证书）
④ 会员关系 +1（送 $300 折扣 ≈ 一年 club membership 价值）
⑤ 方案 C：Founder Network Access（晚宴 + 校友 intro + VC fireside）

FOR 匠人

匠人拿到什么

① 一次触达 40-60 个高匹配 lead（vs C2C 文案需要数月积累）
② 主理人开场 = 课程信任度建立（5min 比我们自吹强 10 倍）
③ 差异化定价合理（不同渠道、不同折扣 · 不影响 C2C 公平）
④ 跑通后第二期 / 第三期 cohort 续约自动延续 · 不用每次重新 outreach

我们不抽成 — 任何会员通过 workshop 后请讲师 / Faculty 做 1:1 advisory / consulting，学员直接跟讲师对接，匠人不参与不抽佣。这是匠人对所有讲师 + 合作机构的透明承诺红线。
