02 · 为什么是现在

这批 founder 正在涌现 —— 但还没人把他们聚到一起

三件事同时成立，让"澳洲华人 AI solo-founder"第一次成为一个看得见的早期池子。

01

技术拐点

Cursor / Claude Code / Lovable / v0 让一个人做出过去一个团队的产品。Pieter Levels / Marc Lou / Tony Dinh 这类 indie hacker 从孤例变可复制 playbook。（公开行业观点，非我们的数据）

02

人群在动起来

澳洲拿 150–250k+ 的华人专业人士（senior dev / PM / 咨询 / 律师 / 会计）天花板焦虑，开始认真做 AI 一人公司 —— 这是 founder 的供给侧。

03

窗口全球独家

"澳洲华人 AI 一人公司 + Grant + 三城线下"全球独一份。这个早期池子现在还没人系统接触。

这批 founder 正在涌现，但还没人把他们聚到一起 —— 现在是认识他们的最早窗口。
