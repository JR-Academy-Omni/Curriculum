PHASE 1 · W0-W3

PHASE 1 让 AI 真正进入你的业务里跑起来 — 起新生意 / 升级老业务 / 自动化日常都算

25 节课 · 13 直播 · 5 Lab
📍 这门课不是必须从零做新产品。很多学员来的时候已经有自己的生意（律师事务所 / 会计事务所 / 小公司 / 自由职业）— Phase 1 教你的不是『再起一个新公司』，是让 AI 真正在你的业务里跑起来，4 条路径任选其一：
① 升级现有业务（小公司老板把客服 / 报价 / 跟进的某个环节让 AI 接管）
② 把咨询服务做成 AI 化的标准套餐（律师 / 会计 / 顾问把『1 对 1 咨询』变成『AI 出初稿 + 你审』模式，毛利更高）
③ 起一个新生意（程序员 / 产品经理 / 设计师做软件产品或服务）
④ 验证扩展方向（已经有产品想申请 YC / Antler 的 founder）
每周课程给 4 条路径不同的具体走法。
W0 PRE-WORK
开课前的预备
把要等几天才批下来的事提前跑起来，开课第一天你不会卡。具体 3 件事（已经有公司的学员可跳过 ①）：
① 注册商业实体（如果还没有）— 首期墨尔本 cohort = 澳洲 ABN（澳洲商业号码，线上 5 分钟免费申请，2-7 天审批）；未来 UK / Canada / NZ / Singapore 版换成对应国家的实体注册（Companies House / CRA / NZBN / ACRA）
② 注册 LLM API Key（Claude / OpenAI / Gemini 任一，月预算 $30-100）— 这是你的 AI 员工要用的
③ 注册 Stripe 账号（先 test mode 不用 KYC，立刻能用；W3 课上换 live mode）

这些慢节奏行政事如果拖到 W3 才办，会卡住你『第一笔真实付费』那一周。
W1
定 AI 进入点
+ 找用户验证
W1 主线：决定 AI 在你的业务里具体做什么 + 跟 5 个真实用户/客户聊过、确认这件事真的有人愿意付钱（或现有客户愿意为升级后的服务多付）。

W1 末你拿到这 3 样：
① 一句话 positioning（具体到『我用 AI 帮 [谁] 解决 [什么]』 — 比如『帮我现有的会计客户出月度税务摘要，每月加收 $49』，或者『帮澳洲华人保险经纪写客户跟进邮件，每月 $49』）
② 5 个目标用户/客户的真实联系方式（这是 W3 收第一笔钱的 seed user 名单）
③ 一份明确的 pricing 方案（5 种 business model 选一个：one-time / subscription / freemium / B2B / productized service）

4 条路径在 W1 走什么：
🏢 小公司老板 / 自由职业（已有生意） → 给现有业务做一次 AI 接管 audit，找出最高 ROI 的 1 个环节（客服 / 报价 / 跟进 / 合同审 / 月报…），W2 把它 AI 化
⚖️ 律师 / 会计 / 医生 / 资深 consultant → 选你最熟的 1 个咨询场景，把它从『1 对 1 实时咨询』productize 成『AI 出初稿 + 你审 30 分钟』的标准套餐
🧑‍💻 程序员 / PM / Designer（要起新生意） → 做 SaaS 软件产品（W2 自己写代码或用 Lovable/v0）
🚀 想申请 YC / Antler 的 founder（已有产品） → 验证扩展方向 + 找 traction 数据

7 节课走完什么：第一节讲 4 条路径分别的真实 case（含小老板 AI 化客服 / 律师 AI 套餐 / 程序员起新生意 / YC 申请者扩展验证 各 1-2 个走通的华人 indie hacker）/ 教你用 AI 在 Reddit / X / 小红书 / 牛客上 mining 真实用户痛点 / 5 个候选 idea 用决策表砍到 1 个 / 跟 5 个目标用户每人 30 分钟 interview（用 Mom Test 提问法 — 不要直接问『你愿意付费吗』，要问『你过去一个月遇到这个问题几次 / 上次怎么解决 / 为这个问题花过多少钱』）/ 最后一节选 pricing model。
W2
装 AI 员工
+ 把 AI 跑进
你的业务
⭐ 最 WOW 的一周
整门课最让你『WOW，原来一个人能做这么多』的一周。这周做这 4 件事（互相关联，并行进行）：

① 给你装第一个 AI 员工 — 你的『AI 秘书』（晚上你睡着、它在帮你跑事的虚拟员工） — 5 种方案对比 + 选一个装上（Claude Cowork 默认推荐 / OpenClaw 隐私敏感行业 / Hermes 跨 IM / Lindy.ai 零代码 / 自建），课上现场 ship 7 个员工任务：每天 8 点 inbox 总结 push 到 Telegram / 自动整理会议笔记 / 客户 follow-up queue / 每月 1 号 Stripe+Xero 财务月报 / 每周日 weekly report… 下课后它每天自动跑。
不论你已经有公司还是从零起步，这一项都是同样的 AI 员工配置。

② 把 AI 真正跑进你的业务里（4 条路径各走各的）：
　🏢 小公司老板（已有业务） → W1 选好的那 1 个环节这周让 AI 接管：客服自动回邮件（OpenClaw / n8n 接 Gmail）/ 报价单 AI 出初稿 + 你 review / 客户跟进自动排队 / 合同 AI 风险审 — 选 1 个跑通，下周再扩第 2 个
　⚖️ 律师 / 会计 / consultant → 用 Notion + AI 把咨询服务 productize：客户填表 → AI 出初稿（合同/税务摘要/法律意见 等） → 你 review 30 分钟 → deliver。一对一 90 分钟咨询变成 30 分钟审稿，毛利从 $200/小时变 $400/小时
　🧑‍💻 程序员（要起新生意） → 用 Cursor / Claude Code 一周 ship 出 SaaS，部署到 Vercel / Cloudflare 拿公网 URL
　🎨 PM / Designer（要起新生意） → 用 Lovable / v0 不写代码生成完整 app

③ 写 1 份 source of truth，AI 一次喂出全套对外 asset — 你只写 1 份 markdown（产品定位 + USP + pricing + use case + FAQ），AI 一次喂 6 个工具：Lovable 生成 landing + Gamma 生成 pitch deck + Canva 生成 brochure + 一页纸 PDF + 公众号长文 + 小红书爆款标题。以后改一个字，所有 asset 重新生成。
已有业务的学员：这份 source of truth 可以是你升级后业务的新对外说法，旧网站 / 旧 deck 一并刷新。

④ Agent Schedule 工作坊（让 AI 员工的任务每天自动触发）— 现场配 5 个真实定时任务：每天 7am 拉竞品价格 / 每周一 9am AI 写 SEO 周报 / 每月 1 号 8am 财务月报 / 每周日 6pm 自动周报 / 每天 22pm 当日总结邮件。明天起每天自动跑。
W3 ⭐ 关键
让公司能收钱
+ 合规
毕业硬指标 #1
W3 让你 W2 搭起来的那个东西能合规收钱。这周拿到的是做这件事最珍贵的一张截图 — Stripe 后台一笔真实交易，付款人是不认识你的人（或者是你现有客户为升级后的新服务付的钱）。不接受朋友友情付费，不接受亲戚『支持下』。

课上 4 件事：
① Stripe / Lemonsqueezy 接入实战（含 webhook + 退款 + subscription + one-time + 服务报价）— 不论你卖的是 SaaS、AI 化套餐、还是接管现有业务环节，都跑通收款（已有 Stripe 账号的学员直接 extend 到新业务，不用重新注册）
② 域名 + Cloudflare DNS + Sentry 错误监控配齐（月成本 $0-15）
③ $1 真实付费挑战（毕业硬指标 #1）— W1 留下的 5 个目标用户里至少 1 个真的付钱（小老板 / 律师 / 会计可以是现有客户为新升级的服务付钱）
④ 5 份法律文件（Privacy Policy / TOS / Refund Policy / Cookie Policy / Disclaimer）挂 footer — 雇真人澳洲律师起草这套要 $3-8k，AI 生成 + AI 律师审核路径 $0 + 2 小时（Termly 模板生成 + Claude 扮演『澳洲科技律师』逐条 review，列出高/中/低风险三档）

W3 末你拿到：能合规收钱 / 第一笔真实付费截图 / AI 员工每天自动跑 / W2 搭起来的那个东西已经上线（不论是新产品、AI 化套餐、还是现有业务的某个环节升级）。

陪你坐下来

FAQ 常见问题
▸ OPC 一人公司 vs 可融资 startup，这门课是哪个？
主轴 OPC 一人公司——目标 $1k → $10k MRR 的自由，不雇人、不融资、用 AI 替代员工。但这门课对 YC / Antler / Startmate 申请准备者也极适合：accelerator 最看重的就是 traction 数据，11 周后你有能跑能卖的产品 + 真实 Stripe 流水 + 第一批付费用户，就是最强的申请材料。W11 还有 Antler 墨尔本 / Startmate 悉尼 / Blackbird VC 的 fireside chat，毕业后 apply 直接对接。把这门课当 pre-accelerator 用刚好。
▸ 我不在澳洲（或不打算来澳洲）能报吗？
可以，但 W7-W9 澳洲会计 + Grant 三周（占总课时 27%）跟你无关，价格不变。W1-W6 + W10-W11 全球通用，对其他国家学员也有价值，但 ROI 不如澳洲学员。强烈建议你等加拿大/英国/新加坡版开（计划在首期跑通后启动）。
▸ 首期为什么定 $3,800-$4,800 这么便宜？第二期就贵？
首期是 case study 价。我们需要 18-25 个学员的真实毕业 case 喂养第二期招生。这种"低开高走"是 cohort-based course 的标准玩法（Reforge / On Deck 都这么做）。第二期起 Standard $5,800、Premium $9,800（含 1:1 mentorship）。
▸ 录取率 30-50% 是不是噱头？
不是。OPC 课最大资产是同期同学的网络——5 年后这批人是澳洲华人 AI 圈的核心节点，互相投资、雇佣、收购。一个心智不对的学员（想拍短视频割韭菜 / 想做币圈 / 没辞职勇气的纯打工心态）会污染整个 cohort 文化。申请表 + 1:1 面试就是筛这个。
▸ 三城同时开课，悉尼和布里斯班是不是被当二等公民？
不是。墨尔本主场老师讲，悉尼/布里斯班线下教室一起看大屏直播 + 本地 Tutor 1:8 现场答疑 + 同城同学线下讨论。这跟纯线上完全不同——你在卫星教室能跟身边的人吃饭、聊产品、搞 networking。匠人 2020 年前就用这个模式跑过多期，已验证。Demo Day 三城联合，每城本地 networking 晚宴。
▸ 毕业硬指标 6 项，万一我达不到怎么办？
不达标不发结业证书。但课程内容你已经全部学完，材料、模板、嘉宾联系方式都在你手里。我们会在毕业 90 天内继续提供 Office Hour 帮你把指标补齐。这不是惩罚，是对认真完课的学员公平——把"上完了课"和"做出了东西"严格区分。
▸ 我已经上过 AI Engineer / OpenClaw Bootcamp，这门课重复吗？
不重复。AI Engineer 教你"AI 工程技能"（找工作用），OpenClaw 教你"AI 接管你的 daily work"（个人提效），这门课教你"用前两门课学到的东西自己开公司、卖产品、合规收钱"。三门课构成一个完整的 AI 时代职业升级阶梯。老学员升级有专属优惠（待商务确认）。
这门课不教你创业。教你做出第一笔真实付费、注册 ABN、提交 Grant 申请——11 周后你不一定成功，但你会有一个能跑、能卖、能合规的 AI 一人创业的完整框架，和 18-25 个跟你一起走过这条路的同期同学。
