PHASE 1 · W0-W3

PHASE 1 拿到：能卖的真产品 + 第一笔真实付费 + AI 秘书每天自动跑 + 5 份合规法律文件

25 节课 · 13 直播 · 5 Lab
W0 PRE-WORK
开课第一天不卡。 ABN 申请、LLM API Key、Stripe 实名 — 这些慢节奏行政流程要等几天到几周，W0 全部预启动。第一节课你跟同学一起做产品调研，不是跟 myGov 客服打电话。未来 UK / NZ / 新加坡 cohort 替换为本国对应实体（Companies House / NZBN / ACRA）。
W1
选品 + 验证
W1 末你拿到的是：1 个被 5 位真实用户访谈过的产品定位 + 一份明确的定价方案。不是『我大概想做 SaaS』那种模糊念头。Reddit / X / 小红书 / 牛客挖痛点的方法学一遍，5 ideas → 1 用决策矩阵砍掉，5 个目标用户 30min interview（联系方式留下 — 这就是 W3 收第一笔钱的种子用户名单）。最后一节课定型『一次性 / 订阅 / Freemium / B2B / 服务+软件混合』里你走哪个 — 选错路径整个 W2 build 都白做。
W2 MVP +
企业大脑
⭐ 最 WOW 的一周
整门课最让你『WOW，原来一个人能做这么多』的一周。三件事并行：

① 一周 ship 出真产品 — 不是 demo 截图。Cursor / Claude Code / Lovable / v0 工具地图选你顺手那个，部署到 Vercel / Cloudflare 拿到公网 URL，朋友能打开。

② 装好你的 AI 秘书 — 那个晚上你睡着、它在帮你跑事的虚拟员工。5 方案选型（Claude Cowork 桌面 GUI 默认推 / OpenClaw 隐私敏感行业 / Hermes 跨 IM / Lindy 零代码 / 自建），课上现场 ship 7 个秘书任务：每天早上 8 点收件箱总结 push 到 Telegram、会议笔记自动整理、客户跟进队列、每月 1 号 Stripe+Xero 财务月报、每周日周报... 下课后它每天自动跑。

③ Source of Truth + AI 一稿出全套 — 写 1 份 markdown『你的产品定位 / USP / pricing / FAQ』，AI 一次喂 6 个工具：Lovable 出网站 + Gamma 出 pitch deck + Canva 出 brochure + 一页纸 PDF + 公众号长文 + 小红书爆款标题。以后改一个字所有素材重生成 — 你不再陷『反复修文案』黑洞。

④ Agent Schedule 工作坊 — 现场配 5 个真实定时任务（每天竞品价格 / 每周 GSC SEO 报告 / 每月财务月报 / 每周周报 / 每天 git review），跑完离场，明天起每天自动触发。Cowork /schedule / Codex / Claude Code cron / OpenClaw / n8n / GitHub Actions 跨平台对照清楚，知道哪个适合你的工作模式。
W3 ⭐ 关键
毕业硬指标 #1
这周拿到的是你做这件事最珍贵的一张截图 — Stripe 后台一笔真实交易，付款人是不认识你的人。不接受朋友友情付费，不接受亲戚『支持下』。课上跑通 Stripe / Lemonsqueezy 接入（含 webhook + 退款 + 订阅），域名 + Cloudflare DNS + Sentry 错误监控配齐（月成本 $0-15）。同时把5 份法律文件（Privacy / TOS / Refund / Cookie / Disclaimer）挂上 footer — 雇真人澳洲律师起草这套要 $3-8k，AI 生成 + AI 律师审核路径 $0 + 2 小时（Termly + Claude『澳洲科技律师』角色 review，列出高/中/低风险三档逐条修）。W3 末你的产品有：能收钱、能合规、有第一笔真实付费。

PHASE 3 · W7-W9

🇦🇺 PHASE 3 拿到：ABN/Pty Ltd/GST 全套合规 + RDTI Grant 申请提交（全球独家 3 周）

11 节课 · 6 直播 · 4 嘉宾
W7
财务自动化
W7 末你不再每周花 5h 处理财务 + 客服。Notion 个人 ERP 把 CRM + 收入 + 任务一体化到一个 workspace；AI 客服自动看邮件 + 草拟回复（OpenClaw / n8n / Zapier 三方案现场对比，按你隐私和技术能力选）；Stripe 每笔交易自动同步到 Xero — 月底不用对账。从 W7 起每周帮你省下的时间转去做产品和营销。
W8 嘉宾周
澳洲 CPA
陪你坐下来
嘉宾 = 澳洲华人 CPA / CA。这一小时决定你接下来 5 年是 sole trader 多交 47% 个税、还是 Pty Ltd 拿 25% 公司税 + Grant 申请资格。课上嘉宾用真实客户案例讲 5 个临界场景，陪你现场注册 ABN（W8 末每个学员都有 ABN — 毕业硬指标 #3）。PSI rules / GST / BAS / CGT 这些自己看一年都搞不清的事，3 节直播加 1:1 邮件 review 一次到位 — 雇澳洲华人 CPA 起步价 $5-10k，这周拿到的咨询深度不输他们的 onboarding。
W9 嘉宾周
RDTI
⭐ 最高 ROI 一周
这周很可能是整门课 ROI 最高的一周。嘉宾 = 帮 5+ 客户拿过 RDTI 退税的 Grant Consultant。RDTI 是澳洲最大创业补贴：年营收 < $20M 的小公司，符合条件的 R&D 支出可以拿 43.5% 现金退税。AI 创业的 R&D 支出范围比想象的宽，但 ATO 审计严格 — 这节课嘉宾发 1500 行 Grant 申请 prompt 模板（已经在真实 RDTI 客户上验证过）+ 评估你的 R&D 支出。W9 末提交至少 1 份 Grant 申请（毕业硬指标 #4）。其他 Grant（EMDG / 各州 / 联邦 AC）以视频 + wiki 形式自学，不浪费课时。

FAQ 常见问题
▸ OPC 一人公司 vs 可融资 startup，这门课是哪个？
主轴 OPC 一人公司——目标 $1k → $10k MRR 的自由，不雇人、不融资、用 AI 替代员工。但这门课对 YC / Antler / Startmate 申请准备者也极适合：accelerator 最看重的就是 traction 数据，11 周后你有能跑能卖的产品 + 真实 Stripe 流水 + 第一批付费用户，就是最强的申请材料。W11 还有 Antler 墨尔本 / Startmate 悉尼 / Blackbird VC 的 fireside chat，毕业后 apply 直接对接。把这门课当 pre-accelerator 用刚好。
▸ 我不在澳洲（或不打算来澳洲）能报吗？
可以，但 W7-W9 澳洲会计 + Grant 三周（占总课时 27%）跟你无关，价格不变。W1-W6 + W10-W11 全球通用，对其他国家学员也有价值，但 ROI 不如澳洲学员。强烈建议你等加拿大/英国/新加坡版开（计划在首期跑通后启动）。
▸ 首期为什么定 $3,800-$4,800 这么便宜？第二期就贵？
首期是 case study 价。我们需要 18-25 个学员的真实毕业 case 喂养第二期招生。这种"低开高走"是 cohort-based course 的标准玩法（Reforge / On Deck 都这么做）。第二期起 Standard $5,800、Premium $9,800（含 1:1 mentorship）。
▸ 录取率 30-50% 是不是噱头？
不是。OPC 课最大资产是同期同学的网络——5 年后这批人是澳洲华人 AI 圈的核心节点，互相投资、雇佣、收购。一个心智不对的学员（想拍短视频割韭菜 / 想做币圈 / 没辞职勇气的纯打工心态）会污染整个 cohort 文化。申请表 + 1:1 面试就是筛这个。
▸ 三城同时开课，悉尼和布里斯班是不是被当二等公民？
不是。墨尔本主场老师讲，悉尼/布里斯班线下教室一起看大屏直播 + 本地 Tutor 1:8 现场答疑 + 同城同学线下讨论。这跟纯线上完全不同——你在卫星教室能跟身边的人吃饭、聊产品、搞 networking。匠人 2020 年前就用这个模式跑过多期，已验证。Demo Day 三城联合，每城本地 networking 晚宴。
▸ 毕业硬指标 6 项，万一我达不到怎么办？
不达标不发结业证书。但课程内容你已经全部学完，材料、模板、嘉宾联系方式都在你手里。我们会在毕业 90 天内继续提供 Office Hour 帮你把指标补齐。这不是惩罚，是对认真完课的学员公平——把"上完了课"和"做出了东西"严格区分。
▸ 我已经上过 AI Engineer / OpenClaw Bootcamp，这门课重复吗？
不重复。AI Engineer 教你"AI 工程技能"（找工作用），OpenClaw 教你"AI 接管你的 daily work"（个人提效），这门课教你"用前两门课学到的东西自己开公司、卖产品、合规收钱"。三门课构成一个完整的 AI 时代职业升级阶梯。老学员升级有专属优惠（待商务确认）。
这门课不教你创业。教你做出第一笔真实付费、注册 ABN、提交 Grant 申请——11 周后你不一定成功，但你会有一个能跑、能卖、能合规的 AI 一人创业的完整框架，和 18-25 个跟你一起走过这条路的同期同学。
