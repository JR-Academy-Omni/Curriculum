课程时间线

30 分钟分秒安排（跟着做就能 ship）

0-3min

① 介绍 + 今天 ship 什么

看 OPC 心智 + 今天目标：一个 micro-SaaS landing page + 一个 working demo + Stripe checkout（test mode），所有都在公网可访问。

产出：一句话产品定位（"AI 帮小红书博主写爆款标题，每月 $9"）
3-10min

② 用 Lovable 一句话生成 landing page

打开 lovable.dev，写一句 prompt：『做一个 AI 小红书爆款标题生成器的 landing page，hero 标题 + 痛点 + 演示 + pricing 三档 + FAQ』。Lovable 30-60 秒生成完整页面，含响应式 + Tailwind 样式 + 工作的表单。

产出：一个真的 landing page，能在 Lovable 预览 URL 直接打开
10-17min

③ 加 Stripe Checkout（test mode）

注册 Stripe（澳洲 ABN 不必须，先 test mode）→ Stripe Dashboard 创一个 product（$9/month subscription）→ 复制 Checkout link → 贴到 landing page 的『立刻订阅』按钮。注意 test mode 用 4242 4242 4242 4242 测试卡。

产出：landing 上的『订阅』按钮真的能跳到 Stripe checkout，能完成 test mode 交易
17-24min

④ 部署到 Vercel 拿公网 URL

Lovable 内置 GitHub 集成 → 1 click 推到 GitHub → Vercel import GitHub repo → 30 秒部署完成 → 拿到 your-product.vercel.app 公网 URL。这一步以前要懂 git/CI/DNS，现在 3 个 click 完成。

产出：公网可访问的 URL，朋友能点开
24-28min

⑤ 测试一笔交易（test mode）

用 4242 测试卡走完整 checkout → Stripe Dashboard 看到 test 交易记录 → landing 上加一个『最近付费用户』wall（社交证明），先用 test 数据填，进 live mode 后自动切换。

产出：Stripe Dashboard 一笔 test 交易截图，可作为 build in public 第一帖素材
28-30min

⑥ 收尾 + 7 个进阶方向

30 分钟完成，但你的产品才刚起步。下一步 7 个方向：① 真的 AI 后端（连 Claude API / OpenAI）② 用户系统（Lovable 内置 Clerk 集成）③ Stripe live mode（要 ABN）④ 自定义域名 ⑤ 法律文件（Privacy / TOS / Refund）⑥ Analytics（Plausible / Posthog）⑦ 第一笔真实付费的获客（去哪发？发什么？）

GOTCHAS

常见踩坑（提前看少 30 分钟排错）

① Lovable 生成失败 / 卡住：刷新重试一次；prompt 太长拆短。免费档每天有限额，跑多次会被限速。
② Stripe Checkout link 跳 404：Stripe 要求 product 必须激活（active=true）；如果是 subscription 必须有定期 billing 周期。test mode 也要全配齐。
③ Vercel 部署 build 失败：通常是 environment variable 缺失（Stripe API key 等）；在 Vercel project settings 加上重新部署。
④ 测试卡 4242 不工作：必须用 Stripe test mode（看 Dashboard 顶部 toggle），live mode 必须真卡。
⑤ 域名问题：本视频用 vercel.app 默认子域，自定义域名（如 your-product.com）需要 namecheap 买域名 + Vercel DNS 配置，本视频不教。
