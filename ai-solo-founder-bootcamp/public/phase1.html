W1 · WEEK 1 选品 + 验证 线下 4h · 自学 4h · 6 节课
L01
LIVE · 120min
[Live] OPC 心智 + 澳洲华人 indie 案例 3-5 个

🎯 第一节课定调：OPC 不是创业，是另一种自由

OPC 跟传统创业最大的区别是：拒绝融资、永远不雇人、用 AI 替代员工、目标 $1k-$10k MRR 的自由而不是 $1B 估值的雄心。课上拆 3-5 个澳洲华人 indie 真实案例，嘉宾来 1-2 个本地 indie hacker（$3k-$30k MRR 范围）讲产品、用户、定价、营销、踩过的坑。课后发"OPC 适配评估表"，低分学员 W2 末有最后一次退款机会。

课上 4 个 step：①OPC vs 传统创业 vs VC startup 三条路对比 ②嘉宾分享 1（墨尔本 indie $3-10k MRR） ③嘉宾分享 2（华人 micro-SaaS $10-30k MRR） ④OPC 适配 5 维度自评。

L02
LIVE · 120min
[Live] AI 用户调研：从 Reddit / X / 小红书 / 牛客挖痛点

🎯 选品的核心是找到一个真实存在、有人愿意付费、AI 能解决的痛点

不是凭空 brainstorming。课上演示用 Cursor + Claude 一小时挖完 200+ 条 Reddit 真实抱怨、用 perplexity 验证市场规模、用小红书评论提取中文用户的具体语言。这是选品最关键的输入数据。

现场操作 4 步：Reddit API + Claude 抓高赞抱怨贴 / X 搜 "I wish there was" 句式 / 小红书爬虫 + AI 聚类 / 输出"痛点 × 频次 × 付费意愿"三维矩阵。

L03
LAB · 30min
Lab: AI 用户调研模板（Prompt Lab）
Prompt Lab 互动练习：给你一个目标行业，写出能完整跑通"找数据源 → 抓数据 → AI 分析 → 输出报告"的 4 段 Prompt 链。系统用真实数据集验证你的输出质量。
L04
SELF · 60min
[Self] 5 个 idea → 1 个：选品决策矩阵

🎯 W1 末的 deliverable：从 50 个候选痛点砍到 1 个 idea

用课上发的"选品决策矩阵"（5 维度 × 加权评分），把 L02 你挖到的 50 个痛点筛到 5 个，再用 AI 模拟用户访谈的方式砍到 1 个。这 1 个 idea 就是 W2 MVP 周要 ship 的产品。

验收：5 个候选 idea 各写 100 字 PRD + 5 维度评分（市场规模 / 付费意愿 / AI 可行性 / 不公平优势 / 6 个月做出 $1k MRR）+ 最后选 1 个写为什么。

L05
SELF · 90min
[Self] 跟 5 个目标用户做 30min interview

🎯 idea 选完后必须做的事：跟 5 个真实潜在用户聊一次

不是问"你愿意付费吗？"——这是 indie hacker 最容易踩的坑（用户嘴上说愿意，掏钱时全跑）。正确问法是 Mom Test 那一套："你过去一个月遇到这个问题几次？""上次遇到时怎么解决？""为这个问题花过多少钱？"用 AI 模拟用户面试一遍后，去真实用户那聊 5 个。

W2 · WEEK 2 MVP 周 — 一周做出能跑的产品 线下 4h · 自学 8h · 7 节课 · 4 个 Lab
L06
LIVE · 120min
[Live] Cursor / Claude Code / Lovable / v0 工具地图

🎯 W2 第一节：选对工具，一周 ship 产品

4 个工具不是同一个东西：Cursor = IDE（写代码）、Claude Code = agentic CLI（自主写整个项目）、Lovable = 无代码 SaaS 生成器、v0 = UI 组件生成器。课上 demo 同一个 idea 用 4 个工具各做一遍，对比时间、质量、限制，让你知道哪个适合你的能力和 idea 类型。

L07
LIVE · 120min
[Live] 一周做一个 SaaS 的 PRD → MVP playbook

🎯 不是教你写完美 PRD，是教你写"AI 能直接 ship 出来"的 PRD

传统 PRD 写给工程师看，AI 时代 PRD 写给 Claude / Cursor 看。结构、详细度、约束都不一样：要明确技术栈、要给 example、要标 must-have vs nice-to-have、要预留 AI 修改余地。课上拆 3 个真实 PRD 模板（SaaS / 工具 / API 服务），学员当场写自己的。

用 Cursor 在浏览器内完成第一个 AI 协作编码任务，理解 Cmd+K / Cmd+L / Composer 三种交互模式的差异。
L09
LAB · 45min
Lab: Vibe Coding — Claude Code agentic 工作流
用 Claude Code CLI 完成一个完整的功能（PRD → 代码 → 测试 → commit），体验 agentic 自主编码 vs Cursor 半人工的差异。

LAB · 45min
Lab: Lovable — 无代码 ship 第一个 SaaS
用 Lovable 在 30 分钟内 ship 一个能跑、能 deploy、能登录的小型 SaaS，体验 no-code 路径。

复用 jiangren.com.au /learn/vibe-coding/hub Phase 2 章节（约 12 章 AI 编码进阶）。建议在 W2 末完成至少 8 章，跟你的 MVP 实操配合。
L11a
LIVE · 120min
[Live] Source of Truth + AI 一稿出全套（Website / Pitch / Brochure / 一页纸 / 公众号）

🎯 W2 末核心动作：一份 source of truth → AI 一次喂出全套对外 asset

OPC 学员最大的时间黑洞是反复写产品文案——网站、pitch deck、brochure、公众号、小红书、Email pitch、媒体稿，每一份都从头写。这节课教你做对的方式：先写 1 份 markdown source of truth（产品定位 / USP / pricing / target audience / FAQ / 5 个 use case / 对比同类产品），再用 AI 一次性喂出所有对外 asset。

实操产出（同一份 source 跑 6 个工具）：① Lovable / v0 → landing page；② Gamma / Beautiful.ai / Pitch.com → pitch deck（投 VC + 找客户两版）；③ Canva AI / Adobe Express → A4 brochure + 一页纸 PDF；④ Notion AI → 客户 onboarding 文档；⑤ 公众号长文 + 小红书爆款笔记标题；⑥ X / LinkedIn 个人简介 + bio。

核心约束：source 改一个字，所有 asset 重新生成。绝不在某一份 asset 里手改文案——改了就回去改 source。这是 AI 时代 OPC 的唯一可持续做法。

L11b
LIVE · 120min
[Live] 企业秘书：Agent OS 选型 + 7 个秘书任务 ship

🎯 让你有"企业秘书 / 大脑"的感觉 — OPC 的非对称杠杆

没雇人的 OPC 怎么有"有秘书帮我打理"的感觉？答案是 Agent OS。这节课对比 4 个主流方案 + 现场 ship 7 个秘书任务。

4 方案选型对比：① OpenClaw Hermes（JR 自家，跟 Bootcamp 学员深度集成 + 专属优惠）；② Claude Cowork（Anthropic 官方，工作流强、隐私好）；③ Lindy.ai（第三方 workflow agent，集成多但贵）；④ 自建（Crew AI / LangGraph，灵活但要写代码）。课上发 4 维决策表（隐私 / 成本 / 集成度 / 学习曲线）。

7 个秘书任务现场 ship：① 每天早上 8 点收件箱总结（重要 + 待回 + 噪音三档）；② 会议笔记自动整理（Otter.ai / Granola → 决策清单）；③ 日程冲突解决 + 客户预约协调；④ 客户跟进队列；⑤ 合同审查（用 AI 律师角色，跟 W3 L16 联动）；⑥ 财务月报（Stripe + Xero → 自动生成）；⑦ 周报生成。Demo 用 Hermes 接 Gmail + Calendar + Drive 跑通收件箱总结。

L11c
SELF · 90min
[Self] 企业超级记忆系统：Email/Drive/Notion/Slack/微信记录变成 AI 可查的记忆库

🎯 给"企业大脑"装上长期记忆（跟 L11b 配套）

L11b 的 Agent 是"执行层"，这节课装的是"记忆层"。RAG over 个人/公司全部数据 — 让 AI 能回答你 6 个月前邮件聊过什么、上周 standup 决定了什么、客户上次提的折扣是多少。

数据源接入：Gmail / Outlook + Google Drive / OneDrive + Notion / Confluence + Slack / Teams + 微信聊天记录（导出后入库）+ WhatsApp / Telegram + 飞书。

工具栈选型：① Claude Cowork built-in memory；② OpenClaw Hermes 记忆模块（跟 L11b 同栈）；③ Mem.ai / Reflect（消费级）；④ 自建（Pinecone / Weaviate / Supabase pgvector，月成本 $0-30）。

5 个典型问答：① "上次跟谁说过 X idea"；② "客户 Y 邮件里的折扣是多少"；③ "上周 standup 决定 Z"；④ "代码里 X 是怎么实现的"；⑤ "Q3 财务数据"。关键约束：本地 RAG 优先（隐私），微信/WhatsApp 记录绝不上云。

SELF · 90min
[Self] 部署到 Vercel / Cloudflare / Railway

🎯 W2 末必须有一个公网可访问的 URL

不是 localhost 截图。买个 .com 域名（namecheap $10/年）+ 选 Vercel / Cloudflare Pages / Railway 之一部署。课程发部署模板（Next.js / Remix / Astro 三选一），学员选一个跟着做。

W3 · WEEK 3 ⭐ 关键 上线 + 收第一笔钱（毕业硬指标 #1 里程碑） 线下 4h · 自学 4h · 4 节课
L13
LIVE · 120min
[Live] Stripe / Lemonsqueezy 接入实战（澳洲版）

🎯 W3 第一节：把"产品"变成"能收钱的东西"

Stripe 在澳洲的限制：开账号需要 ABN（W0 已预备）、需要银行账号验证、首次提现要 7 天。Lemonsqueezy 是 Merchant of Record 模式（替你处理 GST / VAT），对 W3 暂未注册公司的学员是 fallback。课上演示两套接入流程，含 webhook、退款、订阅、One-time payment。

L14
LIVE · 90min
[Live] 域名 + 部署 + 监控基础

🎯 不是教你成为 DevOps，是教 OPC 必须的最小运维栈

域名 + Cloudflare DNS、SSL（Let's Encrypt 自动）、Sentry 错误监控（free tier 够用）、Plausible / Posthog 流量分析。这套配齐月成本 $0-15，是 OPC 标配。

L15
⭐ 硬指标 #1
[Self] $1 真实付费挑战（毕业硬指标 #1）

⭐ W3 末的核心 deliverable：拿到第一笔真金白银 ⭐

$1 也算（很多 indie hacker 第一笔就是朋友 PayPal 转的 $5）。这个挑战的目的不是钱，是验证"你做的东西有人愿意为它付一次费用"。这一步是 OPC 路上最难的——从 0 到 1 比从 1 到 100 难得多。

验收：Stripe / Lemonsqueezy / PayPal 后台截图，含交易时间、金额、买家。不接受朋友亲属"友情付费"——必须是不认识你的人。

L16
SELF · 60min
[Self] 法律文件：Privacy Policy / TOS / Disclaimer (澳洲版)

🎯 收钱前必须挂的 3 份法律文件

澳洲消费者法 (ACL) 比美国宽松但有具体要求：必须有清晰的 Privacy Policy（GDPR + APP 双合规）、Terms of Service、Refund Policy。课程发模板（已经过澳洲律师 review），学员填空 + 挂上线。这件事 5% 的 indie hacker 会偷懒不做，剩下 95% 用 ChatGPT 一键生成 + 人工 review。
