W1 · WEEK 1 选品 + 验证 + 定价模型 线下 4h · 自学 4h · 7 节课
L01
LIVE · 120min
[Live] OPC 心智 + 澳洲华人 indie 案例 3-5 个

🎯 第一节课定调：OPC 不是创业，是另一种自由

OPC 跟传统创业最大的区别是：拒绝融资、永远不雇人、用 AI 替代员工、目标 $1k-$10k MRR 的自由而不是 $1B 估值的雄心。课上拆 3-5 个澳洲华人 indie 真实案例，嘉宾来 1-2 个本地 indie hacker（$3k-$30k MRR 范围）讲产品、用户、定价、营销、踩过的坑。课后发"OPC 适配评估表"，低分学员 W2 末有最后一次退款机会。

课上 4 个 step：①OPC vs 传统创业 vs VC startup 三条路对比 ②嘉宾分享 1（墨尔本 indie $3-10k MRR） ③嘉宾分享 2（华人 micro-SaaS $10-30k MRR） ④OPC 适配 5 维度自评。

L02
LIVE · 120min
[Live] AI 用户调研：从 Reddit / X / 小红书 / 牛客挖痛点

🎯 选品的核心是找到一个真实存在、有人愿意付费、AI 能解决的痛点

不是凭空 brainstorming。课上演示用 Cursor + Claude 一小时挖完 200+ 条 Reddit 真实抱怨、用 perplexity 验证市场规模、用小红书评论提取中文用户的具体语言。这是选品最关键的输入数据。

现场操作 4 步：Reddit API + Claude 抓高赞抱怨贴 / X 搜 "I wish there was" 句式 / 小红书爬虫 + AI 聚类 / 输出"痛点 × 频次 × 付费意愿"三维矩阵。

🎯 W1 末的 deliverable：从 50 个候选痛点砍到 1 个 idea

用课上发的"选品决策矩阵"（5 维度 × 加权评分），把 L02 你挖到的 50 个痛点筛到 5 个，再用 AI 模拟用户访谈的方式砍到 1 个。这 1 个 idea 就是 W2 MVP 周要 ship 的产品。

L05
SELF · 90min
[Self] 跟 5 个目标用户做 30min interview

🎯 idea 选完后必须做的事：跟 5 个真实潜在用户聊一次

不是问"你愿意付费吗？"——这是 indie hacker 最容易踩的坑（用户嘴上说愿意，掏钱时全跑）。正确问法是 Mom Test 那一套："你过去一个月遇到这个问题几次？""上次遇到时怎么解决？""为这个问题花过多少钱？"用 AI 模拟用户面试一遍后，去真实用户那聊 5 个。

L05a
LIVE · 90min
[Live] 定价模型选型：一次性 / 订阅 / Freemium / B2B / 服务+软件混合

🎯 建产品前必须先定的一件事：你的定价模型

同一个 idea，选不同定价模型 → 产品结构、目标用户、获客渠道、技术架构全不同。不能等到 W3 接 Stripe 时才想——那时候产品已经按错的模型建好了。这节课在 user interview 之后、build 之前，把定价模型选型一次定下来。

5 大定价模型 + 适用场景：

  • ① 一次性买断 (One-time) — 工具类 / 模板类 / 信息产品（Gumroad / Setapp 一次买）。决策门槛低、复购低。例：设计模板包、Notion 模板、AI prompt 包、ebook。
  • ② 订阅 (Subscription) — SaaS 标配。月付 $9-99 / 年付 8 折。MRR 可预测但 churn 是命门。例：Notion / Linear / Cursor / Lovable。
  • ③ Freemium — 免费拉用户 + 付费档转化（典型转化 1-5%）。用户基数大但服务器成本高。例：Notion / Figma / Loom。
  • ④ B2B / Enterprise — Per-seat / per-team / contract。客单价 $500-50k+/年。销售周期 1-6 月。例：Slack / Linear team / Vanta。
  • ⑤ 服务+软件混合 (Productized Service) — 软件做骨架 + 人工服务做交付。AI 简历 + 人工 review / AI 营销 + 人工策略。高毛利 + 高客单价 + 难替代，OPC 一个人最适合的模型之一。

3 维决策框架：① 客户类型（C 端 vs B 端）；② 使用频次（每天 vs 一次性）；③ 创始人时间投入度（纯 SaaS vs 服务+软件）。课上每个学员用 3 维表评估自己 idea，得出推荐定价模型 + 价格 anchor。

价格 anchor 设置：选定模型后，定价不是拍脑袋——竞品调研 (5+ 家) + 价值锚 (替代方案多少钱) + 早鸟折扣 + Annual vs Monthly 折扣。课上发"定价计算器 + 竞品对照表 + 5 行业 anchor 数据"。

跟 W3 L13 联动：定价模型决定 Stripe / Lemonsqueezy 的接入方式（一次性 = Checkout / 订阅 = Subscriptions / B2B = Custom invoicing / 混合 = 多产品组合）。

L11a
LIVE · 120min
[Live] Source of Truth + AI 一稿出全套（Website / Pitch / Brochure / 一页纸 / 公众号）

🎯 W2 末核心动作：一份 source of truth → AI 一次喂出全套对外 asset

OPC 学员最大的时间黑洞是反复写产品文案——网站、pitch deck、brochure、公众号、小红书、Email pitch、媒体稿，每一份都从头写。这节课教你做对的方式：先写 1 份 markdown source of truth（产品定位 / USP / pricing / target audience / FAQ / 5 个 use case / 对比同类产品），再用 AI 一次性喂出所有对外 asset。

实操产出（同一份 source 跑 6 个工具）：① Lovable / v0 → landing page；② Gamma / Beautiful.ai / Pitch.com → pitch deck（投 VC + 找客户两版）；③ Canva AI / Adobe Express → A4 brochure + 一页纸 PDF；④ Notion AI → 客户 onboarding 文档；⑤ 公众号长文 + 小红书爆款笔记标题；⑥ X / LinkedIn 个人简介 + bio。

核心约束：source 改一个字，所有 asset 重新生成。绝不在某一份 asset 里手改文案——改了就回去改 source。这是 AI 时代 OPC 的唯一可持续做法。

L11b
LIVE · 90min
[Live] 装 OpenClaw — 你的 AI 执行手（开源编码 / 自动化 agent）

🎯 W2 末第一节 Agent OS — 装好 OpenClaw 跟它建立第一次对话

OpenClaw 是什么：NousResearch 推出的开源 CLI agent（前身 Moltbot / ClawdBot），定位类 Claude Code —— 能在你电脑上读文件、写文件、跑命令、调 API。memory 层用本地 markdown：MEMORY.md（项目记忆）+ USER.md（你的人设）+ 可选 Mem0 向量插件。不是 JR 的产品，但 JR 的 4 周 OpenClaw Bootcamp 专门教这个——上过 OpenClaw Bootcamp 的学员可以跳过装机部分直接看 OPC 场景配置。

课上 4 段：

  1. 装机（20min）：macOS 直装 / Win 10/11 走 WSL2；OpenClaw CLI + bun + LLM API key 配置
  2. 配人设 USER.md（25min）：写出『你是谁、你的产品、你的写作语调、决策风格、行业术语偏好』——后面所有 agent 行为的根基
  3. 配项目 MEMORY.md（20min）：把 W1 选好的产品 idea + W2 在做的 MVP 写进 MEMORY，让 agent 知道当前在做什么
  4. 第一次对话验证（25min）：让 OpenClaw 跟你 W1 的产品 idea 写一篇小红书笔记。验收：跟你过往笔记盲测，至少 70% 像本人语调（不像就回去改 USER.md）

OpenClaw vs Claude Code 取舍：Claude Code 闭源 + Anthropic 官方支持，OpenClaw 开源 + 社区驱动 + 可深度定制 memory 格式。OPC 学员推荐 OpenClaw 因为后面 L11c 要接 Hermes 记忆层，OpenClaw 的 markdown 格式跟 Hermes 兼容（一个命令就能迁），Claude Code memory 格式不直接兼容。

L11c
LAB · 90min
Lab: 装 Hermes 记忆层 — OpenClaw 跨设备 / 跨 IM 持久记忆

🎯 让 OpenClaw 在 Mac 办公室 / 家 / 手机 / Telegram / Slack 上共享一个大脑

Hermes 是什么：NousResearch 出的开源跨平台持久记忆 agent（不是 JR 的，跟 OpenClaw 同一个组织出的），定位 = 记忆层而非编码 agent。文件存在 ~/.hermes/：FTS5 索引的对话 memory（STM Ring Buffer + LTM）+ 自动生成的 skills + Slack/Telegram/Discord/WhatsApp/Signal/CLI gateway。

关键能力（直接拿来用，不用自己写）：

  • ✅ Markdown + FTS5 双层记忆 — 全文搜索 + 时序压缩
  • ✅ Agent-curated memory + periodic nudges — AI 自己整理你的记忆 + 定期问你『这条还重要吗』
  • ✅ 自主 skill 创建 — 跑过的任务自动变成可复用的 skill 文件
  • ✅ hermes claw migrate 一键把 OpenClaw 的 MEMORY.md + USER.md 迁到 Hermes
  • ✅ 内置 Honcho 用户画像模块

Lab 实操（90min）：① 装 Hermes（pip / docker 任一）；② 跑 hermes claw migrate 把 L11b 配好的 OpenClaw memory 迁过来；③ 验证 1 — FTS5 搜索；④ 验证 2 — 跨设备同步（第二台 Mac 启 daemon）；⑤ 启用 Telegram gateway，手机上跟你的 agent 对话。

跟 JR 内部 PRD 的关系：JR 内部也在演化 Hermes 作为团队共享大脑（docs/HERMES_TEAM_BRAIN_PRD.md）。bootcamp 学员的个人版是 L1（个人 scope），未来 L2/L3 公司 scope 由 JR 内部继续打磨。

L11d
LIVE · 120min
[Live] 接通公司核心数据：Gmail / Calendar / Drive / Notion / Slack 全部喂进 agent

🎯 让你的 agent 真正"知道你的公司在发生什么"

L11b 装好 OpenClaw、L11c 接好 Hermes 记忆层。这节课把公司核心数据源全部接进来 — agent 才有上下文回答 OPC 的真实问题（不接数据 = agent 只能聊天，没法干活）。

5 个数据源 + 接入方式（120min 全演示）：

  1. Gmail / Outlook 邮件（30min） — OpenClaw skill：OAuth → 索引最近 90 天邮件 → Hermes FTS5 入库。本地索引优先，邮件正文不上云
  2. Google Calendar / Outlook Calendar（15min） — Calendar API 读未来 30 天事件 + 历史 1 年画像
  3. Google Drive / OneDrive 文档（25min） — 选择性接：Working / Customer / 财务文件夹，敏感文件夹排除
  4. Notion / Confluence（20min） — Notion API integration token 全 workspace 索引
  5. Slack / Discord / Telegram 群消息（20min） — Hermes gateway 直接接，channel 白名单管理
  6. 整合验证（10min）：现场问"客户 X 最近一次邮件提到什么 + standup 决定了什么"，agent 跨数据源回答

隐私 4 档：① 本地 only（Gmail / WhatsApp 必须）② 自建云（Supabase pgvector）③ zero-retention 端点（Anthropic / OpenAI）④ 完全云端（仅公开内容）。课上发数据源 × 隐私级别决策表。

L11e
LIVE · 120min
[Live] 7 个秘书任务现场 ship — 让你有"企业秘书 / 大脑"的真实感觉

🎯 W2 Agent OS 总收官 — 让 agent 真正接管 OPC 的日常工作

前面 L11b/c/d 把 OpenClaw + Hermes + 数据源都装好了。这节课现场把 agent 配成"真的秘书" —— 不是聊天玩具，是每天 8 点 push 总结、自动跟进客户、出月报的执行机器。

7 个秘书任务（现场至少跑通 4 个，剩下作为 W2 末 deliverable）：

  1. ① 收件箱总结 — 每天 8:00 push 到 Telegram，重要/待回/噪音三档 + 重要邮件回复 draft（OpenClaw skill + cron + Hermes Telegram gateway）
  2. ② 会议笔记自动整理 — Otter.ai / Granola 录音 → Hermes 入库 → AI 抽决策清单 + action items + owner
  3. ③ 日程协调 + 客户预约 — Cal.com / Calendly 接 OpenClaw skill，agent 看日历给客户回时段
  4. ④ 客户跟进队列 — 每周一 7:00 push：最近 7 天没回的客户 + 上次对话上下文 + AI 跟进文案 draft
  5. ⑤ 合同审查 — 联动 W3 L16 AI 律师角色 — agent 看到合同 PDF 自动 review 风险点 + 红线清单
  6. ⑥ 财务月报 — 每月 1 日 8:00 push — Stripe API + Xero API → MRR / churn / GST 准备金 / 现金流月报
  7. ⑦ 周报生成 — 每周日 18:00 — agent 看本周 git commit + 邮件 + 日历 + Hermes memory，自动出周报

课上 demo 顺序：先 ① 收件箱总结（最容易出效果），再 ⑥ 财务月报（OPC 学员最爽）。验收：W2 末每个学员至少跑通 4 个秘书任务（一周的 push 输出截图）。

L11f
SELF · 60min
[Self] 进阶选型 — 自建 vs Mem0 vs Lindy.ai vs 团队 Brain

🎯 OpenClaw + Hermes 之外的方案 — 知道存在 + 何时切换

L11b-e 给的是『推荐方案』（OpenClaw + Hermes），适合 80% OPC 学员。其他 4 个备选方案对比：

  • ① 自建（Crew AI / LangGraph + Pinecone / Weaviate / Supabase pgvector） — 给会写代码 + 隐私洁癖 + 想完全控制的学员，月成本 $0-30 但要写 200-500 行 Python
  • ② Mem0 向量插件 — 不想装 Hermes 跨平台、只想给 OpenClaw 加向量检索的学员，装 1 个 npm 包搞定
  • ③ Lindy.ai — 第三方 workflow agent SaaS，UI 漂亮、集成多，但 $50-300/月 + 数据上他们云
  • ④ 团队 Brain（Hermes 共享 daemon） — 当 OPC 长大到 2-5 人小团队时，Hermes 跑在公司 VPS 上做 shared memory（参考 HERMES_TEAM_BRAIN_PRD.md），OPC 一个人阶段先不动

4 维决策表：① 隐私（本地 ↔ 云）② 成本（$0/月 ↔ $300/月）③ 跨设备 / 跨 IM 能力 ④ 学习曲线。课程发完整决策矩阵 + 5 个 OPC 真实学员选型案例。

⭐ 硬指标 #1
[Self] $1 真实付费挑战（毕业硬指标 #1）

⭐ W3 末的核心 deliverable：拿到第一笔真金白银 ⭐

$1 也算（很多 indie hacker 第一笔就是朋友 PayPal 转的 $5）。这个挑战的目的不是钱，是验证"你做的东西有人愿意为它付一次费用"。这一步是 OPC 路上最难的——从 0 到 1 比从 1 到 100 难得多。

验收：Stripe / Lemonsqueezy / PayPal 后台截图，含交易时间、金额、买家。不接受朋友亲属"友情付费"——必须是不认识你的人。

L16
LIVE · 120min
[Live] AI 生成法律文件 + AI 律师审核（Privacy / TOS / Disclaimer / Refund / Cookie · 澳洲版）

🎯 OPC 不雇律师怎么搞定法律文件 — AI 双层流程

第一笔付费收完之前必须挂的 5 份法律文件：Privacy Policy（澳洲 APP + GDPR 双合规）、Terms of Service、Refund Policy、Cookie Policy、Disclaimer。雇澳洲律师起草这套 = $3-8k；用 AI 生成 + AI 律师审核 = $0 + 2 小时。

第一层：AI 生成（30min）：Termly / iubenda 模板 + Claude / GPT 个性化填充（产品类型、收款方式、数据处理位置、用户地区）。课上发"澳洲科技产品 5 文件 prompt 包"，学员现场跑出 5 份文件初稿。

第二层：AI 律师审核（45min）：把生成的文件喂回 Claude / GPT，用专门的"澳洲科技律师"角色 prompt review — 检查 ACL 条款、ATO 税务表述、GDPR 跨境数据传输条款、消费者保障漏洞、SaaS 退款政策合规性。AI 律师会列出"高风险 / 中风险 / 低风险"三档清单，学员逐条修。

什么时候必须找真人律师（30min）：① 投资协议 / SAFE notes / 股权分配；② 首位员工劳动合同（澳洲 Fair Work 极复杂）；③ 重大客户合同（B2B 单笔 $20k+）；④ 跨境出海（美国 Delaware C-corp / 中国数据出境）；⑤ 被人 sue 或威胁 sue。课上发"澳洲华人科技律师推荐名单"（3 位 fixed-fee 律师，跟 Bootcamp 学员合作）。

毕业 deliverable：5 份文件挂在你的产品 footer，AI 律师审核报告作为附件。
