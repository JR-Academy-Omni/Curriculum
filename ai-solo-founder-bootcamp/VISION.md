## 0. 一句话北极星

## 1. 为什么 networking 不是卖点，是这门课的本体

一般 cohort 课卖"完课率 / 知识点"。OPC 创业营真正卖的是——**你 11 周后认识的这 18-25 个人**。

OPC founder 最缺的从来不是知识（网上、AI 里都有，而且越来越多越来越便宜）。最缺的是一个**高密度、同阶段、互相信任**的圈子，因为一个一人公司 founder 真正过不去的几关，全都要靠人：

| 过不去的关 | 圈子怎么解 |
|---|---|
| 没有第一批客户 | 同期同学**互为客户** —— 第一笔 $1 真实付费常常来自身边人 |
| 一个人扛不下所有事 | 圈子里找**合伙人 / 服务采购**（你做产品、他做设计、她做增长）|
| 没人给你背书 / 转介 | 校友 + Faculty 成为你的**推荐源**，把你接到更大的机会 |
| 孤独期撑不住 | 一群**同路人**，知道你在熬什么，这一条决定了谁能活过第 6 个月 |

> 这就是 landing 上「5 类商业角色：互为客户 / 合伙人 / 天使 / 服务采购 / 推荐源」的本质。

**核心判断**：知识会贬值（AI 让内容半衰期越来越短），**网络会增值**——圈子里每多一个 founder 做成，整个网络的价值就上升一档。在 AI 把一切信息商品化的时代，**一个高信任的同阶段 founder 网络，是少数几个抗通缩、还在升值的资产之一**。所以我们把它放在产品的正中央，不是当附属卖点。

## 2. Networking 飞轮（复利护城河）

```
   每期 cohort 跑完
        │  产出：① 一批高质量 founder 节点  ② 一批真实毕业 case
        ▼
   网络更密 + 有真实战绩
        │  → 网络本身成为这门课最强的卖点
        ▼
   吸引更优质的下一期学员 + 更想合作的机构 + 更愿意来的嘉宾/Faculty/投资人
        │
        ▼
   网络再密一层 ──────────────────────────┐
        ▲                                  │
        └──────────────  循环  ────────────┘
```

飞轮的三个关键性质：

1. **复利（compounding）**：网络价值不是线性增长，是随节点数 × 节点质量 × 信任密度复利增长。
2. **先发锁定**：早期节点 = 未来的核心节点。"**5 年后澳洲华人 AI 圈的核心节点，就是首期这批 cohort 同学**"（PRD / landing why-now #4）。后来者买不到"最早进圈"这件事。
3. **为什么首期是 case study 价**：首期低价不是促销，是**用价格换飞轮的初始动能**——换最早一批高质量节点 + 一批真实毕业 case。这两样东西是飞轮转起来的第一推力，比首期那点学费收入重要得多（FUNNEL_PLAN：首期定位为 case study 引流，不是利润中心）。

> 💡 **奖学金是飞轮的定向燃料**：每期评选 1 名「一人奖学金」得主（学完达标 6 项硬指标才全额返学费，见 [FUNNEL_PLAN.md](./FUNNEL_PLAN.md) §10）——只在"已经证明模型成立"的人身上花钱，把那一份学费精准换成下一期最硬的 hero case。冠名赞助还能把外圈合作方 / 投资人直接绑进飞轮。

## 3. 圈子的结构 —— 三层同心圆

```
        ┌─────────────────────────────────────┐
        │  外圈 · 合作机构                       │
        │  混沌澳洲 / 极客公园 / 协会 / 商会      │
        │   ┌─────────────────────────────┐    │
        │   │  中圈 · Faculty + 嘉宾 + 校友  │    │
        │   │   ┌─────────────────────┐    │    │
        │   │   │ 内圈 · cohort 同学    │    │    │
        │   │   │ 同阶段 / 最强信任      │    │    │
        │   │   └─────────────────────┘    │    │
        │   │  跨期经验 / 资源 / 背书        │    │
        │   └─────────────────────────────┘    │
        │  接到更大的精准人群池                   │
        └─────────────────────────────────────┘
```

- **内圈 · cohort 同学** —— 同阶段 founder，信任最强，互为客户 / 合伙人的主战场。
- **中圈 · Faculty + 嘉宾 + 校友** —— 跨期资源、经验、背书；上一期的优秀学员成为下一期的嘉宾。
- **外圈 · 合作机构** —— 把圈子接到更大的精准人群（混沌会员、极客公园读者、协会成员）。**[PARTNERSHIP_PROGRAM_PLAN.md](./PARTNERSHIP_PROGRAM_PLAN.md) 就是外圈扩张引擎**。

三层是**流动**的：校友变嘉宾、嘉宾变投资人、合作机构的会员变学员、学员的产品成为彼此的客户。流动越多，网络越活。

## 4. 未来定位演进：课程 → 网络 → 生态

| 阶段 | 形态 | 北极星动作 |
|---|---|---|
| **Now** | 一门课（首期 case study · 澳洲三城）| 跑出第一批真实毕业 case + 锁定核心节点 |
| **Next** | 校友网络 | 持续 Office Hour + 互为客户日 + Demo Day 做成**澳洲华人 AI founder 的年度节点** |
| **复制**（anchor PRD §1.2 ladder）| 多国本地化 | 澳洲二期 → 加拿大 / 英国 / 新加坡首期（同一核心 80% + 当地化模块 20%）|
| **终局** | 生态 / 圈子 | **全球华人 AI Solo Founder 校友网络** —— 想做 AI 一人公司的华人，第一个想到的入口 |

**诚实的类比**：cohort 网络密度学 On Deck / Reforge，华人社群厚度学生财有术，但**聚焦点是别人没占的 niche**——AI × OPC × 全球华人，加上澳洲 Grant / 合规这块全球独家的本地化。我们不做大而全的"创业大学"，**做深这一个 niche 的网络**——niche 里做到第一，比品类里做到第十值钱得多。

## 5. 宏大志向（north-star metric · 方向不编死数字）

衡量我们成不成，**不看招了多少学员，看培育了多少能自立的华人 OPC founder、以及这个网络内部长出了多少真实合作**。

志向性指标（directional · 具体目标待商务定，本文件不写死、不编造）：

- 🌏 覆盖 **[N] 个城市 / [N] 个国家**的华人 AI founder cohort
- 🧑‍🚀 累计培育 **[N] 个**完成 6 项硬指标的毕业 founder
- 🔁 网络内部年度真实合作 / 成交 **[待跟踪]**（互为客户 / 合伙 / 转介）
- ♻️ 校友复投 + 转介率 **[待跟踪]**（飞轮是否真在转的核心体感指标）

> ⚠️ 这些是**志向，不是承诺**。landing 上「$240k-1.9M cohort 网络潜在交易额」是测算口径，必须标注为**潜在 / 测算**，不是对任何学员的收入承诺。

## 6. 这跟商业模式 / 合作 / 融资怎么扣上

networking 飞轮一旦转起来，同时解决一家教育公司最难的三件事：

- **获客** —— 网络口碑 + 校友转介，是最便宜也最高信任的获客（降低 Type 3 创业课最贵的"信任成本"）。
- **护城河** —— 先发锁定核心节点 + 复利密度，后来者用钱买不到"最早进圈 + 最厚的信任"。
- **LTV** —— 校友复投（进阶课 / 二次 cohort）+ 转介 + 变嘉宾 / 投资人，单个 founder 的终身价值远超一次学费。

所以三份下游文档其实是同一个飞轮的三个切面：

| 文档 | 它讲飞轮的哪一面 |
|---|---|
| [PARTNERSHIP_PROGRAM_PLAN.md](./PARTNERSHIP_PROGRAM_PLAN.md) | **外圈扩张引擎** —— 怎么把圈子接到更大的人群池 |
| [PRD.md](./PRD.md) / [FUNNEL_PLAN.md](./FUNNEL_PLAN.md) | **内圈质量** —— 怎么筛出对的人、保证真产出 |
| `INVESTOR_DECK_PRD.md`（针对投资人）| **把飞轮讲成一个可投资的复利资产** |

## 7. 红线

1. **志向 ≠ 承诺** —— 本文件所有未来数字是方向，不对学员构成任何承诺。
2. **networking 是机会不是保证** —— 课程提供高密度同阶段圈子，不保证一定带来客户 / 合伙 / 收入。
3. **不夸大海外复制的确定性** —— 多国 ladder 是志向路线，不是已发生的事实，对外措辞用"计划 / 志在"，不用"已布局 / 已落地"。
4. **不承诺金钱** —— 沿用 CLAUDE.md + FUNNEL_PLAN 顶级红线，网络潜在交易额一律标"测算"。
5. **不编造规模数字** —— 城市数 / founder 数 / 成交额未跟踪到真实数据前用 `[待定]` / `[待跟踪]`。
