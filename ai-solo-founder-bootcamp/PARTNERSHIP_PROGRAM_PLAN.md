## 0. 为什么这门课特别需要机构合作

OPC 创业营是 **Type 3 创业孵化课**：客单价 $2,800-6,800 AUD、决策周期 4-8 周、**靠信任链路不靠流量冲动**（见 FUNNEL_PLAN §1）。这类课最稀缺的不是曝光，是**信任背书 + 精准人群池**：

- 一个有 6-12 个月储蓄、纠结 4-8 周的 senior 工程师，不会因为刷到一条小红书当晚报名 $3,800 的课。
- 但如果是**他已经信任的机构**（他付费上过的创新课、他关注的科技媒体、他所在的专业协会）出面背书 + 联名办一场 workshop，信任门槛一次性降到位。

所以机构合作对这门课不是"加个渠道"，是**漏斗信任段的核心杠杆**。合作方越多、越对，首期 20±5 人的招生越稳，二期复制越快。这就是"资源越多大家越一起做"的真实机制。

## 1. 合作方分层（4 层 · 按"他们能给什么"分）

| 层 | 典型对象 | 他们的核心资产 | 我们最想要的 | 试点状态 |
|---|---|---|---|---|
| **T1 · 媒体 / 内容平台** | 极客公园 / 36氪 / AI 自媒体矩阵 / 科技公众号 | 大流量 + 科技人群信任背书 | 联名内容 + 背书曝光 + 独家访谈位 | 待启动（招募 deck 主打这层）|
| **T2 · 创新教育社区** | **混沌澳洲** / 湖畔 / 黑马 / 各创业营校友会 | **精准付费人群池**（已被教育过、认知到位、敢为成长付费）| 会员专属价 + 联名 workshop + co-branded landing | 🚀 **混沌澳洲 = 首发试点**（本次产出）|
| **T3 · 行业协会 / 商会** | Melbourne Venture Club / 各华人专业协会 / 商会 | 本地线下人群 + 活动场地 | 线下 workshop + 会员折扣 + 协会返佣 | 已有 `mvc-proposal/`（MVC 提案）|
| **T4 · KOL / 个人** | 校友 / 嘉宾 / 讲师 / 个人 IP | 个人信任 + 垂直小众精准 | 分销 / affiliate 返佣 | 待启动 |

**为什么混沌澳洲是 T2 首发试点**：混沌学园的人群 = 已经为"创新认知"付过费、相信"第一性原理 / 创新方法论"的中产专业人士。这正是 Persona A/B/C 的高浓度版本——他们缺的不是创业意愿，是**把认知落地成一家真公司的 11 周实战路径**。OPC 创业营正好接住这个缺口（"混沌教你怎么想，匠人陪你做出来"）。人群匹配度是 4 层里最高的，所以先打。

## 2. 合作形式菜单（每种形式给什么拿什么）

> 不是每个合作方都用全套，按层级和意愿组合。所有数字 / 比例落地前**商务确认**，不在物料里写死。

> **三大合作类（下面 7 种形式归这三类，对外物料先说这三句）**：
> ① **联合推广** —— 合作方用自己渠道把课程推给会员/粉丝（联名 landing + 联名内容/EDM + EOI 导流，素材我们供）。
> ② **联合销售 / 分销** —— 合作方作为渠道带成交：会员专属价（折扣码）+ **成交返佣**（按 attribution 结算，比例 [商务确认]，不签独家）。
> ③ **资源共建** —— 联名 workshop / fireside 嘉宾席 / 冠名奖学金（品牌共建 + 学员网络，非直接卖课）。

| 合作形式 | 合作方给 | 合作方拿 | 归类 / 适用层 | 红线 |
|---|---|---|---|---|
| **联名 landing page** | 把链接发给自己会员 / 粉丝 | co-branded 页面（双 logo）+ 会员专属价 | T2 / T3 | 不夸大、offer 与主 landing 一致 |
| **会员专属折扣码** | 渠道分发 | 折扣 + 渠道返佣（按成交）| T2 / T3 / T4 | 返佣比例 [商务确认]，不签独家 |
| **联名 workshop**（线上/线下 60-90min）| 召集会员 + 场地 | 免费给会员的实战内容 + 品牌共建 | T2 / T3 | **workshop 不变硬广**（先给价值再提课）|
| **fireside / 嘉宾席位** | 派人来 W10 fireside 分享 | 曝光 + 内容素材 + 学员网络 | T1 / T2 | 不承诺学员转化为对方客户 |
| **联名内容 / 独家访谈** | 发布在自己渠道 | 优质内容素材 + 流量 | T1 | 不承诺金钱 / 收入数字 |
| **EOI 导流 + attribution** | 投放联名页 | 数据透明 + 按效果返佣 | 全层 | 学员 PII 不外流、合规 |
| **冠名奖学金** | 出资赞助 1 个奖学金名额 | 冠名曝光（「[合作方]奖学金」）+ 与得主毕业故事绑定 | T1 / T2 / 投资人 | 见 [FUNNEL_PLAN.md](./FUNNEL_PLAN.md) §10：学完达标才发、不承诺金钱、税务措辞商务确认 |

## 3. 物料工具箱（"想想好要什么" 的答案）

合作越多越需要**可复用的标准件**，每来一个合作方不重新造轮子。这是整个计划的核心交付物清单：

| 物料 | 作用 | 模板 / 范本 | 状态 |
|---|---|---|---|
| **招募合作方 deck** | 说服一个机构"为什么跟我们一起做" | → `PARTNER_RECRUITMENT_DECK_PRD.md`（本次 PRD，单独 agent 写）| 🟡 PRD 进行中 |
| **co-branded landing 模板** | 合作方拿去发给会员 | → **混沌澳洲页是首个范本**：`public/partnerships/hundun-au/` | 🟢 本次产出 |
| **课程一页纸 (one-pager)** | cold outreach 附件 | `public/curriculum.html` 现成 | ✅ 已有 |
| **BD outreach 邮件话术** | 5 套 cold/follow-up | `public/promo/partnership-outreach-emails.md` | ✅ 已有 |
| **MOU 合作备忘草稿** | 谈定后签约模板 | `public/promo/partnership-mou-draft.md` | ✅ 已有 |
| **折扣码 + attribution 机制** | 渠道追踪 + 返佣结算 | 复用 TrackingLink 模块（[商务确认]接入）| 🔴 待接 |
| **联名 workshop SOP** | 线上/线下 workshop 执行 | `offline-event-sop` skill + `speaker-proposal/` | ✅ 框架已有 |
| **联名海报** | 渠道分发视觉 | brand v5 + `xhs-poster` skill | 🔴 按合作方出 |

**结论**：招募 deck（说服机构）+ co-branded landing（机构发给会员）是两个最缺的标准件 —— 这次一次性把两个都立起来（deck 先出 PRD，landing 直接出混沌范本）。

## 5. 红线（CLAUDE.md + 现有合作物料统一口径）

1. **不承诺金钱 / 收入** —— 不说"月入 X / 包赚钱 / 保证拿到 Grant 退税金额"。
2. **不承诺录取** —— 申请制，合作渠道来的也要走 LinkedIn 审核。
3. **不签独家** —— 不跟任何单一合作方签排他，渠道越多越好。
4. **workshop 不变硬广** —— 联名 workshop 先交付真实价值，课程是结尾的自然出口。
5. **所有数字落地前商务确认** —— 返佣比例 / 折扣值 / 会员规模 / 案例数据一律 `[商务确认]`，物料里不写死、不编造。
6. **不编造合作方信息** —— 合作方 logo / 人群规模 / 历史案例没核实就用占位符。
7. **学员 PII 合规** —— attribution 只追踪来源渠道，不外流学员个人信息。
8. **禁用"副业"作承诺词** —— 见 FUNNEL_PLAN §3，框架用"起一条 AI 业务线 / 第二条现金流"，不用"副业"做收入承诺。
