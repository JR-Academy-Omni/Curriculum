## 1 · 问题陈述 + 与其他三份 deck 的差异

### 1.1 这份 deck 是干什么的

读者是**正在决定要不要投钱进来的投资人**。ask = **投资**。这份 deck 要在 3 分钟内回答一个问题：

> **"为什么这是一个值得投的生意 —— 市场多大、护城河是什么、单位经济能不能成立、为什么是这个团队、退出/回报想象空间在哪？"**

核心论点（anchor 全篇，来自 VISION.md）：**我们卖的不是一门 $3,800 的课，是在建「全球华人 AI 一人公司 founder 的网络 / 操作系统」**。从一个别人没占的澳洲 niche 切入，每一期 cohort 都让网络更密、更值钱（先发锁定核心节点 + 复利密度 = 后来者买不到的护城河），网络反过来驱动便宜的口碑获客 + 高 LTV（复投 / 转介 / 校友变嘉宾）。**课程 = wedge（入口），网络 / 生态 = venture（生意本体）。**

### 1.2 与其他三份 deck 的差异（显式表格）

`curriculum/ai-solo-founder-bootcamp/` 下已有 / 已规划三份 deck，**没有一份是给投资人看的**，逻辑链、主语、ask 全不同，不能复用：

| Deck | 给谁看 | 它回答的核心问题 | ASK | 主语 | 为什么不能拿来给投资人 |
|---|---|---|---|---|---|
| `COURSE_INTRO_DECK_PRD.md`（`partnerships/course-intro/`） | **潜在学员**（35-45 澳洲华人专业人士，自己掏 $2.8k-6.8k） | "这门课是给我的吗 / 我能赚什么钱 / 为什么花 11 周" | **报名**（enroll） | 学员（"你"） | 它是 to-C 销售逻辑（副业现金流 / 主业增收 / RDTI 退税）。投资人不关心自己能不能起副业，关心**这门生意能不能 scale、能不能赚钱、护城河在哪** |
| `PARTNER_RECRUITMENT_DECK_PRD.md`（`partnerships/partner-recruitment/`） | **合作方机构 BD / 决策人**（极客公园 / 混沌 / 协会 / KOL） | "我机构为什么跟你一起推、能拿什么、怎么落地、红线" | **合作**（co-promote） | 合作方机构 | 它是 B2B 渠道分发逻辑（曝光 / 联名 / 返佣 / 会员专属价）。投资人不是来借渠道的，是来**买股权 / 买未来回报**的 |
| `mvc-proposal/`（已上线 10 page deck） | **单一合作方**（Melbourne Venture Club） | 针对 MVC 一家的定制合作提案 | **合作**（一对一） | MVC | 写死单一机构，且仍是合作逻辑，不是融资逻辑 |
| **本 deck（`investor-deck/`）** ← 本 PRD | **投资人**（天使 / 早期 VC / 战略·教育产业方） | "为什么这是值得投的生意：市场 / 护城河 / 单位经济 / 团队 / 回报" | **融资**（fund us） | 这门生意 / 公司 | — |

**一句话区分**：course-intro 卖**课**给学员，partner-recruitment 借**渠道**给机构，本 deck 卖**生意**给投资人。前两份讲"课"，本 deck 讲"以课为 wedge 的网络型 venture"。

### 1.3 这份 deck 的特殊性：pre-traction thesis deck

`outline.json` 的 `cohortStatus = PLANNING` —— **首期还没开，0 真实 cohort、0 收入、0 毕业 case**。所以这不是一份"看我跑出了什么数据"的 traction deck，而是一份**早期 / pre-traction 的 thesis deck**：靠的是**市场时机（why-now）+ wedge 的锋利度 + 团队既有资产 + 早期信号（waitlist / EOI / partner LOI 若有）+ 飞轮逻辑的可信度**，而不是已实现的财务表现。

因此全篇所有 traction / 财务 / 预测 / TAM / CAC / LTV 数字 = `[商务确认]` / `[待填]` 占位。Deck 的说服力来自**逻辑链 + 团队信誉 + 诚实**，不来自编出来的漂亮数字。

## 2 · 目标读者（investor 类型）

读者统一特征：**不是学员、不是合作方**，是**决定要不要把钱投进来换取未来回报的人**。共同顾虑 = 「这门生意能不能 scale、护城河经不经得起后来者复制、钱投进去多久能看到里程碑、退出路径是什么、这个团队能不能执行」。

| 投资人类型 | 他在乎什么 | 让他点头的关键 | deck 哪几页主攻他 |
|---|---|---|---|
| **天使 / 个人投资人**（可能是华人圈成功 founder / JR 校友里的高净值） | 团队靠不靠谱、赛道对不对、早期能不能进、跟创始人信任关系 | 团队既有资产（32 课 + 平台 + 校友鱼塘 + 澳洲本地）+ 飞轮先发逻辑 + 早期信号 | Team / Wedge / Moat / Early Signal |
| **早期 VC（pre-seed / seed）** | 市场够不够大、单位经济能不能成立、护城河复利不复利、能不能跑出指数曲线、退出想象空间 | networking 飞轮复利 = 真护城河 + niche-first 制胜 + LTV 来自网络（非一次性学费）+ 海外复制 ladder | Why-now / 真正的生意 / Moat / 经济模型 / 财务预测 / Ask |
| **战略 / 教育产业方**（教育集团 / 出海创投平台 / AI 公司战略部） | 跟自身业务的协同、能不能并入生态、品类卡位、品牌 | "全球华人 AI founder 网络入口"的品类卡位 + 三层同心圆生态 + 海外复制可对接其资源 | 真正的生意 / 海外复制 ladder / GTM 分发 / Vision 收尾 |

> ⚠️ **三类读者的侧重不同，但本 deck 先做一份通用版**（不为每类单独出一版），用同一套 13-16 页覆盖。哪类投资人优先、要不要拆版 = §8 开放问题之一。

## 3 · 目标 / Non-goals

### ✅ 目标
1. 任一类投资人 **3 分钟**翻完能回答 §1.1 的 5 件事（市场 / 护城河 / 单位经济 / 团队 / 回报）。
2. **把课程降级为 wedge，把网络/生态升级为 venture** —— 全篇主语是"这门生意"，不是"这门课"。
3. **护城河讲透**：networking 飞轮的复利 + 先发锁节点逻辑（VISION §2），让投资人相信"后来者用钱买不到最早进圈 + 最厚信任"。
4. **诚实的早期定位**：明确标 pre-launch，用 why-now / wedge / team / 飞轮逻辑撑起论点，不假装有 traction。
5. **所有数字占位**：TAM / 收入 / CAC / LTV / 增长率 / 融资额一律 `[商务确认]`，给**口径和逻辑**（怎么算的），不给**编造的数**。
6. **投资风险如实披露**（pre-traction、嘉宾供应链依赖、单一 niche 集中度、海外复制不确定）—— 主动亮风险反而建立信任。

### ❌ Non-goals
1. ❌ **不对学员承诺金钱 / 收入**（CLAUDE.md + FUNNEL §3 顶级红线）—— 即使是给投资人看 LTV，也用"过程指标 + 测算口径"，不出现"保证学员赚 X"。
2. ❌ **不编造任何 traction / 财务 / TAM 数字 / 用户数 / 增长率 / CAC / LTV**（cohortStatus=PLANNING，0 真实数据）。
3. ❌ **不夸大海外复制的确定性**（VISION §7.3）—— 加拿大 / 英国 / 新加坡用"计划 / 志在"，不用"已布局 / 已落地"。
4. ❌ **不卖课给投资人**（投资人不是学员）—— 课程内容 only 占 1 页 wedge 介绍，不展开 11 周 syllabus。
5. ❌ **不把这份 deck 当成"已经决定融资"的既成事实**（§8 #1 是首要开放问题）—— PRD 层面明确这是 conditional deliverable。
6. ❌ **不抹平哲学张力**（§8 #2）—— 不能既对学员讲"拒绝融资=自由"，又对投资人讲"我们要融资 scale"而装作无事；必须正面处理，且文案要 Lightman sign-off。

## 4 · 叙事主线（投资人视角的 chain）

注意这条链跟另两份 deck 完全不同：course-intro 是「学员 pain → 学员 outcome」，partner-recruitment 是「机构 pain/机会 → 机构 value」，本 deck 是「**市场窗口 → wedge → 真正的生意 → 护城河 → 经济模型 → 团队 → ask**」—— 一条标准的早期 venture pitch 链，但护城河锚定在 networking 飞轮。

```
① 钩子/Vision  一句话：我们在建「全球华人 AI 一人公司 founder 的网络入口」（VISION §0）
② 问题         OPC founder 最缺的不是知识（AI 让知识贬值），是高密度同阶段信任网络（VISION §1）
③ Why-now      AI 拐点 + 白领天花板焦虑 + 澳洲华人 founder 圈空窗，三件事同时成立（PRD §1.2）
④ Wedge        这门 11 周澳洲 niche 课 = 锋利入口（筛人 + 锁节点 + 产真实 case）
⑤ 真正的生意   网络/生态终局：课程→校友网络→海外复制→全球华人 OPC 网络（VISION §3/§4）
⑥ Moat         networking 飞轮 = 复利护城河（节点×质量×信任密度复利 + 先发锁节点，VISION §2）
⑦ 市场/TAM     给口径不给编造数（华人专业人士 × AI founder 意愿 × 客单价，全占位）
⑧ 商业模式+单位经济  pricing 真值（$2.8k/$3.8k/$6.8k）+ LTV 来自复投/转介 + CAC 靠网络口碑（数字占位）
⑨ GTM/分发     4 partner tier = 网络外圈扩张引擎（PARTNERSHIP_PROGRAM_PLAN）
⑩ 竞品定位     vs Reforge/On Deck/生财有术/Antler（PRD §2.1）+ 为什么 niche-first 制胜
⑪ Traction/早期信号  诚实标 pre-launch，列 waitlist/EOI/partner LOI（若有，全占位）
⑫ Team         为什么是匠人：32 课 + 平台 + 校友鱼塘 + 澳洲本地（既有资产 = 别人没有的起跑线）
⑬ 财务预测     假设驱动模型（首期 case study ~$50-60k → 二期 → 海外，全占位标"假设"）
⑭ Ask          融资额 / 用途 / 里程碑（全 [商务确认]）
⑮ Vision 收尾  回到北极星：全球华人想做 AI 一人公司，第一个想到的入口
```

> 🚨 通用红线（每页都适用）：① 财务/用户/增长/TAM/CAC/LTV/融资额 = `[商务确认]` 占位；② 不对学员承诺金钱；③ 海外复制用"计划/志在"；④ 不编造嘉宾/校友/合作机构真实数据 —— 没核实就占位。

### Page 01 · COVER · Vision one-liner
- **目的**：3 秒让投资人知道这不是"卖课 deck"，是一个 venture pitch。
- **核心信息**：北极星一句话 + 这是融资提案。
- **必含要素**：
  - 大标题（VISION §0 原话精炼）：`我们在建「全球华人 AI 一人公司 founder 的网络入口」`
  - 副标：`课程是入口 · cohort 是种子 · 网络是终局`（VISION §0）
  - 角标：`投资人提案 · [商务确认 日期] · 匠人学院 Jiangren Academy · [融资轮次 · 商务确认]`
  - 匠人 logo（封面深色用 `jr-academy-brand/assets/logo/logo-zh-white.png`，浅底用 `logo-zh-full.svg`）
- **数据来源**：VISION §0。**不放价格 / syllabus / 课程 feature 抢焦点 —— 这是 venture 不是 course。**

### Page 02 · 问题 · OPC founder 最缺的是网络，不是知识
- **目的**：让投资人理解这门生意的根本 insight —— 为什么"网络"才是值钱的东西。
- **核心信息**：AI 让知识无限贬值、网络反而增值；OPC founder 过不去的关全靠人。
- **必含要素**：
  - 核心判断（VISION §1）：`知识会贬值（AI 让内容半衰期越来越短），网络会增值 —— 一个高信任的同阶段 founder 网络，是少数抗通缩、还在升值的资产之一`
  - "过不去的关 × 圈子怎么解"4 行表（VISION §1 原表）：没有第一批客户→同期互为客户 / 一个人扛不下→圈里找合伙人服务采购 / 没人背书→校友+Faculty 推荐源 / 孤独期撑不住→同路人。
  - 一句给投资人的话：`所以这门生意的核心资产不是课件，是网络 —— 这是为什么它能成为一个 venture，而不只是一门课`
- **数据来源**：VISION §1。**不编 founder 数量 / 网络规模**。

### Page 03 · Why now · 三件事同时成立的窗口
- **目的**：给投资人"为什么是现在、为什么这个窗口不等人"的紧迫感。
- **核心信息**：技术拐点 + 白领天花板焦虑 + 澳洲华人 founder 圈空窗，三件事同时成立（PRD §1.2）。
- **必含要素**（直接抄 PRD §1.2，投资人视角）：
  1. **技术拐点真实**：Cursor / Claude Code / Lovable / v0 让一个人做出过去一个团队的产品；Pieter Levels / Marc Lou / Tony Dinh 类 indie hacker 从孤例变可复制 playbook；Sam Altman 公开预测"一人 billion-dollar 公司"（**标注为公开行业观点，不引为我们的数据**）。
  2. **焦虑供给侧真实**：澳洲拿 150-250k+ 的中产专业人士天花板焦虑（PRD §1.2.2）—— 这是付费意愿的供给侧。
  3. **澳洲华人 founder 圈空窗**：全球华人 AI 创业课没差异化（生财有术 / 出海帮在做），但"澳洲华人 AI 一人公司 + Grant + 三城线下"全球独一份（PRD §1.2.3 / §2.1）—— 空窗 = 先发机会。
  - 一句给投资人的话：`这三件事的交集只在这个时间点出现 —— 早 2 年技术不够，晚 2 年圈子被别人占了`
- **数据来源**：PRD §1.2 / §2.1。indie hacker 名字 + Sam Altman 预测标"公开行业观点"，不编 MRR。

### Page 04 · Wedge · 这门课是锋利的入口（不是生意本身）
- **目的**：让投资人理解课程的真实角色 —— 不是收入产品，是切入网络的 wedge。
- **核心信息**：11 周澳洲 niche 课 = 筛人 + 锁早期核心节点 + 产真实毕业 case 的入口。
- **必含要素**：
  - 一句话课程定位（FUNNEL §1）：`用中文教澳洲华人专业人士用 AI 做一人公司：选品 → MVP → 收第一笔钱 → 全渠道营销 → 澳洲会计/Grant → Demo Day`
  - 极简属性条：`11 周三城线下（墨/悉/布）· 申请制（录取率目标 30-50%）· 首期 18-25 人 · 不录播`（PRD §3.3 / §7.5）。
  - **为什么是 wedge 不是 venture**（这页的核心）：
    - 申请制 = 筛出对的早期节点（不是泛流量）；
    - 首期低价是**用价格换飞轮初始动能**（换最早一批高质量节点 + 真实 case），不是促销，不是利润中心（VISION §2.3 / PRD §1.3 张力3 / FUNNEL §3）；
    - 毕业 6 项硬指标（$1+ 真实付费 / 内容产出 / ABN / Grant 提交 / 产品 URL / Demo Day 出席）= 产出"真实战绩"喂飞轮（PRD §4.2）。
  - 一句给投资人的话：`首期收入 ~$50-60k（[商务确认]）我们不当利润看，当成买入网络第一批核心节点的成本`
- **数据来源**：FUNNEL §1/§3、PRD §3.3/§4.2/§7.5、VISION §2.3。
- **⚠️ 红线**：6 项指标里 Grant 写"提交申请"，不写"拿到退税"；不写"包做出赚钱产品"。首期收入数字标 `[商务确认]`（PRD §7.5 给的是 ~$50-60k 区间，标注来源）。

### Page 05 · 真正的生意 · 网络 / 生态终局
- **目的**：把视角从"一门课"拉升到"一个网络型 venture"—— 这是投资人真正在买的东西。
- **核心信息**：课程 → 校友网络 → 海外复制 → 全球华人 OPC 网络的四阶段演进（VISION §4）。
- **必含要素**：
  - 四阶段 ladder（VISION §4 原表）：**Now**（一门课·首期 case study·澳洲三城）→ **Next**（校友网络·年度节点）→ **复制**（多国本地化·同核心 80%+当地化 20%）→ **终局**（全球华人 AI Solo Founder 校友网络）。
  - **三层同心圆**（VISION §3）：内圈 cohort 同学（同阶段最强信任）/ 中圈 Faculty+嘉宾+校友（跨期资源背书）/ 外圈 合作机构（接更大精准人群池）；标"三层是流动的"（校友变嘉宾、嘉宾变投资人、合作机构会员变学员）。

## 7 · 红线（实现 deck 时一字不破）

2. **不编造 traction / 财务 / TAM / 用户数 / 增长率 / CAC / LTV / 融资额** —— cohortStatus=PLANNING，0 真实数据。全部 `[商务确认]` / `[待填]`，给口径不给编造数（§5 各页已逐条标）。唯一可写死的真数 = pricing（FUNNEL §3 已锁的 $2,800/$3,800/$6,800）。
3. **不夸大海外复制的确定性**（VISION §7.3）—— 加拿大/英国/新加坡用"计划 / 志在 / 假设情景"，不用"已布局 / 已落地"。
4. **投资风险如实披露** —— page 11 诚实标 pre-launch；deck 应包含（或在 page 11/13 内）风险提示：pre-traction、嘉宾供应链依赖（PRD §11 P0 风险）、单一 niche 集中度、海外复制不确定、首期失败定义（PRD §11.3）。**主动亮风险 = 建立信任，不是减分。**
5. **不用代码生成 pptx / docx**（最高级铁律）—— deck = **HTML multi-page**，禁止 python-pptx / pptxgenjs / 任何 .pptx/.docx 产物。要给投资人 PDF 就用浏览器单页 print。

## 9 · 验收标准

实现的 deck（且仅在 §8 #1/#2/#3 拿到 Lightman sign-off 后）上线前，逐条 check：

1. **3 分钟测试**：找一个不懂这门生意的人（扮演投资人）翻完 deck 3 分钟，能复述 §1.1 的 5 件事（市场多大 / 护城河是什么 / 单位经济怎么成立 / 为什么是这个团队 / 回报想象）。4/5 答对 → pass。
2. **wedge vs venture 分得清**：读者能讲清"课程是入口，网络/生态是真正的生意"，而不是以为这是在卖一门课。
3. **护城河讲透**：读者能复述 networking 飞轮的复利 + 先发锁节点逻辑（不是"因为我们课好"）。
4. **数字全占位**：grep deck **无写死的 TAM / 收入 / 用户数 / 增长率 / CAC / LTV / 融资额** —— 全部 `[商务确认]` / `[待填]` 或区间+标来源。唯一真数 = pricing（$2,800/$3,800/$6,800）。
5. **pre-traction 诚实**：page 11 明确标 PLANNING / 0 真实数据；无编造的 waitlist / case / 财务数。
6. **海外复制不夸大**：grep 无"已布局 / 已落地"，全用"计划 / 志在 / 假设情景"。
7. **风险已披露**：deck 含 pre-traction / 嘉宾供应链 / niche 集中度 / 海外不确定 / 失败定义的风险提示。
8. **不对学员承诺金钱**：grep 无"保证赚 X / 月入 / 包做出赚钱产品"；Grant=提交申请不保通过。
9. **哲学张力已处理 + 已 sign-off**：deck 对"公司融资 vs 学员拒绝融资"的处理经 Lightman 过目签字。
