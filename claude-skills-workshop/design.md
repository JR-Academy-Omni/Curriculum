# 设计风格文档 · Design Style Guide

> 风格代号：**Mars Research Lab / 火星研究所 Zine 风**
> 灵感来源：复古科技杂志 + 工程笔记 + 现代信息图（Editorial × Technical × Brutalist）

---

## 1. 整体气质（Mood & Tone）

一种"硬核技术拆解杂志"的视觉语言。像一本由极客在火星实验室印刷出来的工程周报：

- 信息密度高，但层级清晰
- 字体大胆、对比强烈，像海报又像说明书
- 复古印刷感（牛皮纸底色、网格、技术标注）
- 内容严肃，但通过插画与排版增加趣味性
- 中英文混排，强调"研究所感"与"档案感"

关键词：**Editorial（编辑设计）、Brutalist（粗野主义）、Technical Document（技术档案）、Risograph（油印质感）**。

---

## 2. 配色系统（Color Palette）

主色板非常克制，仅用 4 个核心颜色：

| 角色 | 颜色 | 色值（参考） | 使用场景 |
|---|---|---|---|
| 背景 Base | 米白 / 牛皮纸 | `#F2EFE6` ~ `#EDE8DC` | 整体底色，模拟纸张 |
| 主文字 Ink | 近黑 | `#0F0F0F` ~ `#111111` | 标题、正文、边框 |
| 高亮 Accent | 鲜亮黄 | `#FFD83D` ~ `#FFE14D` | 重点词、金句框、高亮背景 |
| 警示 Signal | 朱红 | `#E63329` ~ `#E5302A` | 标注点、装饰图形、强调 |

辅助色：浅灰网格线 `#D9D5C8`（用于背景网格与分隔线）。

**使用原则**：黑 + 米白 = 信息骨架；黄 = 强调（一页最多 2~3 处大面积黄）；红 = 点缀（不超过整页面积的 5%）。

---

## 3. 字体系统（Typography）

中英文混排是这套风格的灵魂。

**中文标题**：超粗黑体（如思源黑体 Heavy / 阿里巴巴普惠体 Bold / Misans Heavy），字号巨大、字重极高，常用黄色色块作为底高亮。

**英文标题/副标题**：
- 衬线斜体（Serif Italic，如 Playfair Display Italic、Tiempos Italic），用于诗意化的英文 Slogan
- 等宽字体（Monospace，如 IBM Plex Mono、JetBrains Mono），用于技术标注、版本号、坐标、日期

**正文**：现代无衬线（Sans-serif），中等字重，行距宽松。

**层级示例**：
1. 巨型中文主标题（占页面 1/3 高度）
2. 英文斜体副标题（说明性 Slogan）
3. 黄色高亮的关键句（一句话金句）
4. 普通段落正文
5. 等宽小字标注（Tag、Meta、坐标）

---

## 4. 版式与网格（Layout & Grid）

- **底层网格**：浅色方格纸背景（grid paper），强化"实验室笔记"感
- **页眉**：左侧黑底白字标签（栏目名 + 期数）+ 右侧日期（等宽字体）
- **页脚**：四列数据带（如 STARS / FORKS / LANGUAGE / LICENSE），用细线分隔，类似仪表盘
- **黑色边框块**：右上角常见，用作"信息卡片"，包含项目元数据
- **大量留白 + 大字标题** 的对比节奏：先用巨型字震一下，再用小字密集铺信息

页边距相对宽松，但内部信息块紧凑。

---

## 5. 图形元素（Graphic Elements）

- **黄色高亮色块**：模仿马克笔涂抹的效果，包裹关键词
- **红点 + 虚线引线**：技术图纸式的标注（如 "FIG.01 SIGNAL BEACON"）
- **极简插画**：扁平化、几何风的小房子/建筑/卫星/雷达，黑色描边 + 黄红点缀
- **坐标尺**：插画下方常带 `0  3m  6m  9m` 这种工程刻度
- **小图标 + 编号**：如 `LAB 01`、`FIG.01`、`#003`，营造档案感
- **加号十字标记 `+`**：散布在画面中，作为定位点装饰

---

## 6. 文案语气（Voice & Copy）

- 中文短促有力，常用陈述句 + 金句结构（例："架构比价格更值得抄。"）
- 英文小写斜体，写法接近编辑性副标题（例："how to rebuild a $24,000 terminal as an open-source desktop app."）
- 重要数字用加粗 + 单位分离的方式呈现（例：**7.4** k stars）
- 使用引号、破折号、顿号丰富节奏感
- 有"栏目化"语感：本期金句、本期不讲X，只讲Y

---

## 7. 组件清单（Reusable Components）

| 组件 | 描述 |
|---|---|
| 期刊头条（Masthead） | 黑底白字栏目名 + 期数 + 研究所名 + 日期 |
| 金句框（Highlight Box） | 黄色背景 + 黑色粗体短句，作为视觉锚点 |
| 项目元信息卡（Meta Card） | 黑色细边框 + 项目名 + 类型 + 技术栈 + License |
| 巨型主标题（Hero Headline） | 中英混排，部分文字带黄色 highlight 背景 |
| 技术插画（Technical Illustration） | 扁平几何风格 + 工程标注 + 比例尺 |
| 数据带（Stats Bar） | 页脚 4 列，分别为关键指标，等宽字号大数字 |
| 作者署名（Byline） | 圆形头像 + 作者名 + handle，左下角 |

---

## 8. 应用建议（Usage Tips）

- 适合用于：技术内容拆解、Newsletter 封面、产品复盘海报、开源项目介绍、知识星球封面图
- **避免**：渐变、阴影、玻璃拟态、写实摄影 —— 都会破坏"印刷品"质感
- 黄色和黑色比例约为 **1 : 6**，红色作为"提神"的极少量点缀
- 中文标题字号必须够大（通常 >120pt 印刷尺寸），气势是这套风格的命门
- 始终保留网格底纹与等宽字体的小标注，"实验室档案感"由它们承担

---

**一句话总结**：用印刷品的克制 × 编辑设计的节奏 × 工程图纸的细节，把硬核内容做成可以贴在墙上的研究所档案。
