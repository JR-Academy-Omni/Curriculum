L01

W1.1 AI Agent 全景 — 不是 ChatGPT 那种 AI

30 min 老师主讲
📖 你会听到

跟 4h Workshop 的 L01 不同 — Workshop 讲 Agent 跟 ChatGPT 差别。这节讲 4 周后你的 Agent 会长什么样，整个升级路径今天先看到全景。

🎯 你会带走

Agent 5 个能力维度（调工具/记忆/主动/多步推理/跨平台），每周升一个。把课前列的 5 件事按『AI 接管度』分成 ABC 三类。

💡 行业相关

本期 Program 行业方向（律师/地产/会计/内容）4 类典型工作流的 AI 接管路径，老师对着你的素材集解释。

L02

W1.2 Quest: 装好 OpenClaw + 跑通第一条指令

45 min 🌸 Quest Tutor backup
🌸 AI 小花一对一带

装机继承自 Workshop 的 6 步 Quest（系统 → Node ≥22 → 装 → 验证 → 配 LLM Key → 第一条指令）。每步贴终端输出当场验证。

🆕 Bootcamp 加固

装完立刻 mkdir -p ~/openclaw/{skills,configs,data} 立工作目录。后面 W2-W4 所有配置文件归档到这里，避免 4 周后乱成一团。

🛟 守护规则

Win 用 PowerShell → 立刻劝退换 WSL2；Node < 22 → 升级；Bun → 换 pnpm。AI 5 分钟卡住 Tutor 接手。

L03

W1.3 配你的 Agent 人设 — 签名 / 语调 / 表达习惯

30 min 老师主讲
📦 现场写好这 4 件事

① 身份卡（你是谁、做什么、服务谁）
② 语调样本（开头 5 种 / 结尾 5 种 / 签名）
③ 禁忌词（绝不会说的词）
④ 风格标签（简洁/友好/专业/幽默 选 2-3 个）

💡 老师当场写样板

老师以本期行业拿真实案例现场写 SystemPrompt 给你看，你照着改成你自己的。看老师怎么思考的。

📂 产出物

~/openclaw/configs/persona-v1.md — W2-W4 持续迭代成 v2、v3。

L04

W1.4 Live Demo — 老师用配好人设的 Agent 跑本行业典型任务

30 min Live Demo
🎬 你看的不是 PPT

从老师真实素材输入开始（脱敏过的客户案例），看 Agent 怎么读、怎么想、怎么写。

🎯 行业场景（按本期）

律师：从 3 封客户邮件起草跟进 / 地产：从备忘录起草项目跟进 / 会计：从一周流水生成对账单初稿 / 内容：从选题生成三平台稿。

📋 课后 5 分钟

从你素材集里挑一件跟 Demo 同类型的事 — L05 Workshop 你就跑这件事。

L05

W1.5 Workshop — 用你自己的素材跑一遍

60 min Tutor 1:8 陪练
🛠 Tutor 帮你判断

第一次跑大概率不完美 — Tutor 帮你看是 Prompt 没写清楚 / SystemPrompt 没调到位 / 还是任务本身不适合 Agent 做。

📦 3 个产出物

① 跑通的产出文件（电脑里留下能复用的）
② 调好的 persona-v1.md
③ 时间数据：手动 vs Agent

📊 时间数据怎么记

『同样这件事我以前手动做要 X 分钟，Agent 跑用 Y 分钟』。W4 Demo Day 用这数据。

L06

W1.6 Networking + 周作业布置

30 min 茶歇
☕ 4-5 人小组

互相看对方的 SystemPrompt 和产出。跨行业经常借到意想不到的思路 — 律师从地产学客户跟进、内容创作从会计学结构化思维。

📋 W1 课后作业

① 真实工作里 Agent 跑 3 次（记成功/失败）
② 看 L07-L08 自学
③ 完成 L09-L10 两个 Lab
④ Slack 群贴一份脱敏产出

🎯 W2 现场前准备

列出你想接哪 5 个工具（邮箱/日历/CRM/Notion/消息平台），W2 现场就接这些。

L09

Lab: 明确任务 — 让 Agent 准确理解你的需求

30 min Prompt Lab
🧪 实操

把模糊需求改成四要素 Prompt（任务/受众/格式/约束）。在浏览器内即时验证。

💡 为什么 W1 课后做

W2 接工具前先把『把需求写清楚』功底打好 — 接好的工具会被烂 Prompt 浪费。

L10

Lab: 输出格式 — 按你要的格式回答

30 min Prompt Lab
🧪 实操

让 AI 按指定格式输出（Markdown 表 / JSON / 邮件结构 / Bullet 列表）。在浏览器内完成。

💡 W2 用得上

W2 接 Notion / CRM 时大量用到结构化输出，Lab 里练熟，到时候不卡。
