❓ 常见问题
没上过 4h Workshop 能直接报 Bootcamp 吗？
可以。Bootcamp 包含 W1.2 完整 Quest 装机环节（跟 Workshop 同套路）+ 课前预习包补齐基础。但如果你预算紧张又想试水，先报 4h Workshop 看是否适合你 — 上过觉得『装上了不够』再报 Bootcamp，凭 Workshop 报名号有首期减免。
4 周抽不出每周 4h 现场 + 2-3h 课后怎么办？
那就不要报。这门课 dropout 80% 来自周中没时间消化。Bootcamp 价格不便宜 + 错过现场课跟不上节奏 + 没做完每周作业下一周课等于白上。先把 4 周时间在日历上锁死再报。如果只是某一周特殊情况，可以申请延期到下一期补课。
OpenClaw 跑起来一个月成本多少？
OpenClaw 本身 MIT 开源免费。主要成本是 LLM API Key — Bootcamp 4 周 + 课后高频用 + 跑自动化任务，月成本通常 10-30 美元（比 ChatGPT Plus 便宜，效果完全不同档次）。L44 Wiki 专门讲怎么避免月费突然冲到 200 美元。
本期 Program 是什么行业方向？
本期是律师专场（首期）。后续期次轮换：地产 / Mortgage 专场、会计 / 财税专场、内容 / 自媒体专场。每期赠送对应行业 Wiki 包。报名前问清楚课程顾问本期是什么方向 + 下一期是什么方向。
非律师能参加律师专场吗？
可以，Training 主体（4 周路径）全球统一，跨行业学员也能正常吸收。但赠送的行业 Wiki 包跟你的行业不对口 — 律师 Wiki 包对地产顾问帮助有限。建议等到本行业专场再报。如果实在等不及，可以参加，但行业 Wiki 包对你价值低。
能不能线上参加？
不能。这门课只开线下。跟 4h Workshop 同样的设计原因 — Quest + Tutor 1:8 陪跑、Workshop 现场调试、Networking、Demo Day 公开展示，线上做不出效果。报名前先确认你能稳定到现场上完 4 周。开课城市每期轮换（悉尼 / 墨尔本 / 布里斯班 / 黄金海岸），看本期 Program 选离你近的那期。
有录播吗？错过一节怎么办？
不录播。线下沉浸式课程录播没意义 — Quest 装机 / Workshop 调试 / Networking / Demo Day 录下来你看不出价值。错过一节现场课需要联系课程顾问申请延期到下一期同主题课补上。这也是为什么报名前必须确认 4 周能稳定到场。
开课城市在哪？
每期轮换悉尼 / 墨尔本 / 布里斯班 / 黄金海岸，少数期会去 Adelaide / Perth。报名前看本期 Program 在哪个城市开。住外地的学员通常这周飞过来，4 周连续上完（多数学员是本城本州的）。
跟 AI Engineer Bootcamp / AI Adoption Bootcamp 啥区别？
OpenClaw Bootcamp（这门课）— 4 周让你用好 OpenClaw 这一个具体工具，AI 接管你工作。
AI Adoption Bootcamp（8 周）— 给所有商业岗位的 AI 通识 + 多个工具入门。
AI Engineer Bootcamp（12 周）— 给开发者的，写 Skill / 接 MCP / 跑 RAG / Fine-tune，是技术深度路径。
这门课定位最聚焦：非技术 + 已用过 ChatGPT + 想真把 AI 用起来。
报名了临时来不了能退 / 转期吗？
开课前 7 天免费转下期或全额退款。开课后 W1 上完前可转下期或退 80%；W2 之后只能转下期；Demo Day 后不退不转。具体条款见报名链接。
