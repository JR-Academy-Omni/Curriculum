L11

W2.1 工具接入 = AI 真有用的分水岭

30 min 老师主讲
📖 为什么 W1 装好后多数人 Agent 闲置

不是 Agent 不行 — 是没接你常用工具。等于雇了秘书但没给她办公室钥匙、不让她碰电脑。

🔧 接工具的 3 种姿势

① OAuth（Gmail/Notion/Slack）— 最安全
② API Key（自家 CRM/数据库）— 公司给 Key
③ MCP Server（开源生态）— 一行配置接 50+ 工具

📋 本期行业 ROI 排序

律师：Outlook + Notion + 合同库 / 地产：Gmail + Calendar + Domain / 会计：Xero + Outlook + Excel / 内容：公众号 + Notion + Buffer

L12

W2.2 Quest: 接邮箱 — Gmail / Outlook 二选一

45 min 🌸 Quest Tutor backup
🌸 5 步 OAuth Quest

选邮箱 → Google Cloud / Azure 建 OAuth Client → 在 OpenClaw 配置 → 浏览器授权 → 测试读 5 封 + 起草草稿。每步贴截图。

🛟 守护规则

① 绝不接受贴邮箱密码（必须 OAuth）
② 公司邮箱第一次先用 Gmail 个人账号测
③ Outlook 学员要确认有 Azure App 注册权限

✅ 接完 Agent 能做

读过去 30 天邮件 / 按规则分类 / 起草回复（草稿箱不直接发） / 提取行动项

L13

W2.3 Quest: 接日历 + 消息平台

45 min 🌸 Quest
📅 日历部分

Google Calendar / Outlook Calendar 用同一个 OAuth Client / Azure App。15 分钟搞定。

💬 消息平台三选一

WhatsApp / Slack / 飞书。WhatsApp Business API 复杂（公司账号），多数学员先用 WhatsApp Web Bridge。Slack / 飞书相对简单。

✅ 接完 Agent 能做

『今天后面有什么会』『下周二下午有空吗』直接回 / 同事 Slack 问你某事 Agent 起草回复 / 客户 WhatsApp 询价提醒你

L14

W2.4 Live Demo — 邮箱+日历+消息打通后的完整工作流

30 min Live Demo
🎬 老师跑一个真实工作日

9 点整理过夜邮件按紧急排队 → 10 点会前提醒 + 昨天 follow-up 状态 → 中午 3 个会自动生成 follow-up 草稿 → 客户 WhatsApp 进来标紧急 → 6 点总结今天跟进。

📦 你看的不是截图

用老师本人真实工作日（脱敏）演示，让你看到 Agent 接管后『一天能省多少』的具体画面。

💡 行业差异

律师场：客户咨询邮件 + 案件日程 + 同事 Slack；地产场：客户跟进 + 看房日程 + WhatsApp；会计场：客户问账 + Xero 异常通知；内容场：评论提醒 + 选题群消息。

L15

W2.5 Workshop — 接你自己的 CRM / Notion / Excel

60 min Tutor 1:8 陪练
🏢 接你工作主战场

邮箱日历是基础设施，主战场是律师 Clio / 地产 Domain / 会计 Xero / 内容 Notion。这节 60 分钟接本行业核心工具。

🛠 3 种路径

① OAuth 走得通的（Notion/Xero）— 跟 Quest 同套路
② API Key（自家 CRM）— Tutor 帮联系 IT 拿 Key
③ 没 API 的老系统 — 用 OpenClaw『浏览器 Skill』模拟点击

⚠️ 公司 IT 限制

真碰到限制接不上的工具（很常见），Tutor 帮你判断能不能接、怎么跟 IT 申请、能不能用替代方案。
