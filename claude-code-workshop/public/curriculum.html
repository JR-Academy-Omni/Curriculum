💻

主受众 A · IT 在职工程师

现状：每天在用 Cursor / Copilot / Windsurf，AI 帮你补完下一行很快。但让它「写一个新模块」「迁移一个 service」「review 一个 PR」 —— 它就抓瞎，因为它只看光标附近几十行代码。

痛点：AI 帮你写下一行，但任务级的活还是你手工拆步、手工跑、手工 review。

来这里能得到：换到 Claude Code 之后 —— 它能读你整个 repo，一个指令同时开 3 个 AI 助手分工干活（一个查代码 / 一个写实现 / 一个做 review），把团队规范沉到「项目规则书」让 AI 自动遵守，让 Claude 自己开浏览器测自己写的页面。

🎓

主受众 B · 在校 / 转码学习者

现状：写过几个 demo，会基本命令行 + git，但项目永远停在「能跑」阶段，没有工程感。

痛点：作品集打开 README 就一段「This is a Next.js project」，面试官一看就是教程项目。

来这里能得到：用 Claude Code 把作品集从「demo 项目」做出「工程项目」的样子——有架构文档、有 review 流程、有自定义 agent 自动跑检查。

⚠️ 不适合：从没写过代码的纯小白。前 30min 拉齐起点段也要求会基础 git + 命令行。如果你这俩都没接触过，建议先看 1 小时入门视频再来。

主线流程：开发前准备（需求清单 + 架构设计）+ 关键命令 → 同时跑多个 AI 助手干活 → 让 AI 自动遵守你的规则 → 完整开发流程跑通 → Q&A。顺序刻意 —— 先教学员"动 Claude Code 之前怎么准备"，而不是上来教按钮。每段都有跟着敲的 demo，不是纯讲。

❓ 常见问题
完全没用过 Claude Code 能跟上吗？
可以。前 30 分钟会拉齐起点，带你从零跑通第一个任务。但要求会基础 git + 命令行（如果你这俩都没接触过，建议先看 1 小时入门视频再来）。
可以用 Cursor / Copilot 代替来跟课吗？
不可以。Claude Code 能做的事（读整个 repo / 同时开多个 AI 助手分工 / 自动跑测试 / 给 repo 写规则 AI 自动遵守）和 Cursor / Copilot 是两种工具，替换不了。

线上 A$99 早鸟和 A$199 全价的差别只是价格吗？
对。内容、时长、讲师、cheat sheet 完全一致，只是早鸟有时间窗口（截止时间报名时见），过期自动调到全价 A$199。
线下 A$299 比线上贵那么多，到底贵在哪？
三件事：① 5 小时实操（vs 线上 3 小时） ② 小班 ~15 人 + 老师 + 助教 1v1 救场 ③ 午餐茶歇 + 现场互评。线上更适合"想自己快速上手"的工程师，线下更适合"想被陪着把 Claude Code 真正落到自己 repo 里"的团队负责人。
和市面上其他 "AI 编程课" 有什么区别？
市面上 90% 的课在讲「怎么写 prompt 让 ChatGPT 给你写代码」。这场讲的是切到第三代工具（Claude Code）的工作方式——让 AI 读你整个 repo、自己改文件、跑测试、提 PR。这是工具的差异，不是技巧的差异。
