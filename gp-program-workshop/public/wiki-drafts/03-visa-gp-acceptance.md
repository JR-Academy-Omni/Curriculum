# 485 / PR / Citizen × GP 公司接受度对照

> ⚠️ **重要免责声明**：
> 1. 本 Wiki 是 GP **求职策略参考**，**不是签证 / 移民咨询**。
> 2. 签证 / 移民问题 — 请咨询**持牌 Migration Agent / MARA**。
> 3. 公司 visa 政策可能每年变 — 以**目标 JD / 公司 careers 页面 / HR 回复**为准。
> 4. 本表基于公开招聘信息整理，不构成任何 offer 保证。

---

## 一、基础认知

### 澳洲 GP 申请常见 visa status

| Status | 含义 | GP 普遍接受度 |
|--------|------|--------------|
| **Australian Citizen** | 公民 | ✅ 全部接受 |
| **Permanent Resident (PR)** | 永居 | ✅ 全部接受 |
| **485 Post-Study Work Visa** | 毕业生工签 (2-6 年) | ⚠️ 因公司 / sector 而定 |
| **Student Visa (500)** | 学生签 (在读) | ⚠️ 部分接受（毕业前 12 月可申） |
| **Bridging Visa A/B/C** | 过渡签证 | ⚠️ 多数公司不接受（视为不稳定） |

### "Sponsorship" 是什么意思

公司说 "We sponsor international students" / "We're happy to sponsor work visa"，意思是：
- 招你时接受你当下的 visa（通常 485）
- 你 485 到期前，公司会给你申请 **482 (TSS) 工作签证**赞助
- 工作 2-3 年后可走雇主担保 PR (186 nomination)

公司说 "Citizen / PR only" — 通常**不开放给国际背景**。

## 二、Big 4 (PwC, EY, KPMG, Deloitte)

### Audit / Assurance
- ✅ **485 接受** — 普遍欢迎国际学生
- ✅ Audit 是 Big 4 最国际化 service line（人手缺、客户多元）
- ⚠️ **CA / CPA 注册路径**：通常公司给 sponsor，但路径相同

### Tax
- ✅ **485 接受** — 多数城市 / 多数 service team
- ⚠️ 部分 Specialty Tax（International Tax / Transfer Pricing）偏好 PR

### Advisory / Consulting
- ✅ **485 接受**（人数最少但开放）
- ⚠️ **Government Practice** 子团队（PwC Public Sector 等）— 部分项目要 Security Clearance → 这部分 PR/Citizen Only

### Risk / Cyber / Forensic
- ⚠️ **混合** — 私营 client 项目接受 485，Gov / Defence client 项目要 Security Clearance → PR/Citizen

### 实战建议
- 投 Big 4 **优先投 Audit / Tax**（最大 visa 接受度）
- Advisory / Consulting 投 **私营行业 stream**（Banking / TMT / Retail / Health）
- **避开** Government / Defence 标签的子团队

## 三、Major Banks

### Commonwealth Bank (CBA)
- **Retail / Business Banking**：✅ 485 接受（部分 stream）
- **Banking & Markets** (Wholesale / Markets)：⚠️ 偏好 PR，但 case-by-case
- **Technology**：✅ 485 接受度高
- **Risk & Compliance**：⚠️ Compliance 子团队偏好 PR

### Westpac, NAB, ANZ
- 整体政策类似 CBA
- **NAB Tech** 特别欢迎国际背景
- **ANZ Institutional**（Markets）偏 PR

### Macquarie Group
- **Macquarie Capital** (投行)：⚠️ Competitive, 历史上偏 PR 但接受顶尖 485
- **Banking & Financial Services**：✅ 485 接受
- **Technology**：✅ 接受
- **Macquarie Asset Management**：⚠️ Mid-tier 485 ok / 高级偏 PR

### Bendigo Bank / Bank of Queensland / Suncorp
- 通常 ✅ 485 接受
- 规模小但门槛友好

### 实战建议
- **Tech / Operations / Retail stream** 是 485 友好入口
- **Banking & Markets** 准备好被多筛一轮
- 投递前看 **JD 底部 "Eligibility"** 段落 — 通常写清楚

## 四、Government Graduate Programs

### Federal (APS Graduate)
- ❌ **绝大多数要 Australian Citizen**
- 部分 specialised stream（Cyber / Data / Digital） — 偶尔接受 PR
- **不接受 485**（极少例外）

### NSW Government Graduate Program
- ❌ **要 Australian PR or Citizen**
- 不接受 485

### VIC Government Graduate Program
- ⚠️ 多数 stream **PR / Citizen only**
- 极少数 specialised role 接受 485（需 case-by-case）

### QLD / WA / SA Government Grads
- ❌ 普遍 PR / Citizen only

### Defence / Intelligence Agencies
- ❌ **Australian Citizen Only** + Security Clearance

### 实战建议
- 485 → **Gov 普遍不是 viable path** 这一年
- 路径：先走 Big 4 / Banks / Consulting → 工作 → PR → 转 Gov
- 例外：**特殊技能 stream**（Cyber / AI / Health Data）偶尔有 sponsorship

## 五、Tier-1 Consulting

### MBB (McKinsey · BCG · Bain)
- ✅ **485 接受** — MBB 全球流动性高，对国际背景友好
- ⚠️ 落地后 2-3 年内 sponsor PR 是常见路径
- 要求：英语极强 + Case 极强

### Tier-2 Strategy (Deloitte S&O · EY-Parthenon · PwC Strategy& · Accenture Strategy)
- ✅ 多数 ✅ 485 接受
- 走 Big 4 框架的 Consulting 通常和 Big 4 Advisory 同政策

### Mid-tier (Kearney · Oliver Wyman · L.E.K.)
- ⚠️ 较小 office，case-by-case

## 六、Tech Companies

### 澳洲本土 Tech (Atlassian, Canva, REA, Xero, SafetyCulture)
- ✅ **485 接受度极高** — 这是国际学生最友好的 sector
- Atlassian / Canva 历史上招了大量国际背景 grad
- 部分公司**主动赞助 PR** (186 employer nomination)

### Telco (Telstra, Optus, TPG)
- ✅ 普遍 485 接受
- **Telstra Cyber** 部分项目要 Security Clearance

### Big Tech AU (Google AU, Amazon AU, Microsoft AU, Meta AU)
- ✅ 485 接受
- 但极度竞争 · 全球招聘标准

### 实战建议
- **Tech 是 485 留学生最优路径之一**
- Atlassian / Canva 应当作核心目标
- 准备 GitHub repo / 开源贡献 / personal project 比 GPA 更有用

## 七、Engineering Consultancies

### Tier-1 (Aurecon · WSP · AECOM · Arup)
- ✅ **485 接受** — 澳洲缺 engineer
- 多数公司 sponsor 482 路径
- ⚠️ Defence / Critical Infrastructure 项目要 Citizen

### Mid-tier (GHD · Jacobs · SMEC · KBR · BG&E)
- ✅ 普遍 485 接受
- 矿业 / 油气项目（FIFO）通常给到 PR sponsorship

### 实战建议
- Engineer 背景 + 485 → **强势 sector**
- 准备 **Engineers Australia (EA) accreditation** path
- 关注 **mining / energy** 公司，普遍 PR-track 友好

## 九、🚨 关键 Tips

### 1. 投之前先看 JD 底部
99% 公司在 JD 末尾会标：
- "You must have full working rights in Australia"
- "Australian Citizen / Permanent Resident only"
- "We welcome applications from international students"
- "Eligibility: must be Australian Citizen due to security clearance"

**仔细读这一段，不符合直接 skip 不要浪费时间**。

### 2. "Full working rights" ≠ "Sponsorship"
- "Full working rights" 通常 = 至少 485 / PR / Citizen
- 不代表公司给 sponsor PR 之后

### 3. 485 时长很重要
- 2 年 485 vs 4 年 485 vs 6 年 485 公司接受度不同
- **越长的 485 越受欢迎**（Engineering 类专业有 6 年 Subsequent 485 优势）

### 4. Cover Letter 主动陈述（短句）
```
"I'm currently on a 485 Post-Study Work Visa with X years of full working
rights, and would welcome the opportunity to work towards long-term
employer-sponsored pathways."
```

### 5. AC / 面试时主动 confirm
- 不要等到最后 offer 阶段才被告知 visa 不符合
- 在 1v1 interview 末尾礼貌问："Just to confirm — does this role accept 485 visa holders?"
