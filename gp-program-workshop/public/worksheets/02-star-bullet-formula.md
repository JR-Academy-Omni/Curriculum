# STAR Bullet 改写公式

> GP Program Workshop L03 + L05 配套资源。一个公式 + 10 个澳洲岗位 before/after — 你简历每条 bullet 都按这个跑一遍。

---

## 一、STAR 4 维度展开

| 维度 | 含义 | bullet 占比 | 关键 |
|------|------|-----------|------|
| **S**ituation | 什么场景 / 什么时候 / 什么角色 | ~20% | 一句话能让 HR 入戏 |
| **T**ask | 要解决什么具体问题 / 衡量标准 | ~10% | 不要泛泛说"目标是提升业绩" |
| **A**ction | **我**做了什么（不是团队） + 我为什么这么做 + 我和别人不同的地方 | ~50% | 主语必须是『我』 |
| **R**esult | 数字 / 百分比 / 影响范围 + 学到了什么 | ~20% | 没数字 = 没结果 |

---

## 二、改写公式（直接套）

```
[Action Verb] + [What you did, specifically] + [Tool / method / scale]
   + [Result with number] + [Context if useful]
```

**3 行 bullet 公式**（多用于核心 Experience）：
```
- [Strong verb] [specific action] [for what scope/audience],
  [using what tool/method], [achieving what quantified result];
  [optional: secondary impact or reaction from stakeholder]
```

**1 行 bullet 公式**（短经历）：
```
- [Verb] [thing] for [scope], delivering [quantified result]
```

## 三、强动词清单（按场景）

### Leadership / Initiative
- Led · Spearheaded · Initiated · Founded · Established · Pioneered · Directed · Drove

### Analysis / Problem-solving
- Analysed · Investigated · Diagnosed · Evaluated · Audited · Modelled · Forecasted · Synthesised

### Build / Create
- Designed · Developed · Built · Implemented · Engineered · Architected · Launched · Deployed

### Improvement
- Optimised · Streamlined · Reduced · Eliminated · Accelerated · Refined · Restructured

### Communication
- Presented · Briefed · Negotiated · Influenced · Persuaded · Advocated · Coordinated

### Collaboration
- Partnered · Collaborated · Coordinated · Facilitated · Liaised

### ❌ 弱动词（不要用）
- Helped · Worked on · Participated · Assisted · Supported · Was responsible for · Was involved in

## 五、自查 5 问

每写完一条 bullet，问自己：

1. **主语是『我』还是『团队』？** — 必须是『我』
2. **有具体数字吗？** — 没有就硬挤 ($, %, 人数, 时长, 频率)
3. **动词够强吗？** — Replace any 弱动词
4. **场景对 HR 来说清楚吗？** — Suburb / Company / 项目背景
5. **读起来 7 秒内能 get 重点吗？** — 太长就拆 2 行

## 六、Common Pitfalls

| 坑 | 怎么改 |
|----|--------|
| "Helped team to deliver X" | "Co-owned delivery of X (3-person team) — drove [specific piece]" |
| "Was responsible for managing Y" | "Managed Y, [quantified scope]" |
| "Participated in project Z" | Drop or → "Contributed to Z by [specific action]" |
| "Strong analytical skills" (Skills section 用) | Move to bullet with evidence |
| 故事太长 (5 行 bullet) | 拆成 2-3 行，砍 Situation 和 Task |
| 没有 Result 数字 | 加 — 多少人 / 多少 % / 多少 $ / 多少时间 / 多少频率 |
