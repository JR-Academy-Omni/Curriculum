## 0. 项目背景与目标

### 0.1 业务背景

JR Academy 已上线 **OpenClaw 4 小时沉浸式工作坊**（`curriculum/openclaw-workshop/`），主打"4 小时把 OpenClaw 装到你电脑上跑通 2 次真实任务"，受众是非技术岗位职场人（销售/市场/行政/房产/律师/会计等）。前期跑下来发现一类高价值用户没被服务到：

- **私营小老板**（餐饮 + 电商 + 服务业混合）
- 4 小时跑通 2 个任务对他们来说**节奏太赶 + Skill 数量不够**
- 他们要的是"一整天给我搞定 3 件公司里最烦的重复活儿"
- 愿意为更深、更系统、能直接复用到自己生意的版本付更高单价

参考对标：[ivanlu867.github.io/openclaw-landing](https://ivanlu867.github.io/openclaw-landing/) RMIT 第四期模式（$599 AUD / 60 人 / 一整天 / 6 大 Skill / 讲师 Andy 宋 + Ivan Lu / 前 3 期 120+ 学员包括 72 岁老太太成功装机）。我们要做的是**这个模式的私营小老板专场版**。

### 0.2 学员一天结束能带走什么（4 项必交付）

1. ✅ 跑起来的 OpenClaw（自己电脑 + Key 配好 + 跑通"你好"）
2. ✅ **3 个深度跑通的 Skill**（FAQ 客服 / GST 发票-PDF 提取 / 老板日报）— 不是看演示，是自己跑通
3. ✅ 1 个能在自己微信 / WhatsApp 直接调用的 AI 数字员工（带回去周一上班就能用）
4. ✅ 1 份"下周一动手"的行动清单 + 课后 30 天社群兜底

## 1. 受众画像（Persona）

### 1.1 核心人群

**「私营小老板」** — 公司 1-30 人，年营业额 50w-1000w，老板自己每天还在干 N 件重复活儿。

| Persona | 行业 | 痛点 | 期望 |
|---------|------|------|------|
| **Lily · 38 岁，墨尔本华人川菜馆老板** | 餐饮 | WhatsApp 订位咨询从早回到晚 / 菜单问题答 100 遍 / 想招前台又怕走 | AI 把 80% 重复客服扛掉，留人际部分给自己 |
| **Tony · 42 岁，Shopify 跨境卖家** | 电商 | 每天看后台订单 + 客诉 + 物流问题 / 时差导致半夜也要回 / 想招 VA 又不放心数据 | AI 自动看后台、按急切程度分级、早上 9 点推送日报 |
| **Sarah · 35 岁，墨尔本房产中介自创公司** | 服务业 | 整理客户跟进记录 / 看同行挂牌价 / 给客户做项目材料 | AI 接管跟进记录 + 同行监控 + 标准化材料生成 |
| **Wang · 50 岁，会计师事务所老板** | 服务业 | 客户合同 / 财报 PDF 处理 / GST 发票一堆 / 合规敏感不能用云 AI | OpenClaw 本地跑 + GST 自动 + PDF 提取 |

### 1.2 共同特征（招生筛选用）

- ✅ 公司 1-30 人 / 自己是老板或核心决策人
- ✅ 已经用过 ChatGPT / Claude / 豆包，至少能聊上几句
- ✅ 愿意自己带笔记本来现场（不带公司管控电脑）
- ✅ 真心要在自己公司落地 AI，不是"听个讲座了解一下"
- ❌ 不要"系统学 AI 工程化"的人 → 推 AI Engineer Bootcamp
- ❌ 不要"概念党"完全不想动手的 → 推免费 /learn/openclaw

### 1.3 老板们能上手到什么程度

**零代码**：复制粘贴 API Key 都要手把手。

我们的应对：
- API Key 现场 JR 统一发临时 Key，老板不用自己注册卡住
- 装机走 Quest 模式（沿用 4h 版 AI 小花一对一）+ 2 名人类 Tutor 兜底
- 3 个 Skill 全部**预置模板**，老板只改自己公司的数据 / FAQ / 邮箱
- 全程投屏 + 助教散坐 1:25

## 2. 学习目标（Learning Outcomes）

课程结束后，学员**应能**：

1. 在自己 MacBook / Windows 电脑上跑起来的 OpenClaw + 配好至少一个 LLM Key
2. 把 **Skill 1（FAQ 客服）** 用到自己生意 — 微信 / WhatsApp 收到客户消息能自动按预置规则回复
3. 把 **Skill 2（GST 发票 / PDF 订单提取）** 用到自己生意 — 客户邮件附件 PDF 进来能自动落表，反向也能一句话生成澳洲合规 GST 发票
4. 把 **Skill 3（老板日报 + 邮件分级）** 用到自己生意 — 每天早晨 9 点 OpenClaw 自动看邮箱 + 群消息 + 订单推送一份"今天该关注什么"
5. 决策清楚 OpenClaw 7×24 部署方案、API Key 月度成本、哪些数据安全敏感不要喂 AI
6. 写出下周一在自己公司部署的 1 件具体事 + deadline

## 4. 课程内容大纲（Content Spec）

### 4.1 整体节奏

- **总时长**：8 小时（10:00-18:00）+ 30 分钟课前预习
- **互动比**：讲解 30% / 演示 30% / 学员动手 40%（比 4h 版更重动手）
- **节奏单元**：4 大段 × ~90 分钟，每段后 15 分钟茶歇 / 午餐
- **里程碑**：11:30 前 100% 学员装机成功（沿用 4h 版的 Quest + Tutor 双保险）

### 4.2 分镜（10:00-18:00）

| 时段 | 时长 | 模块 | 类型 | 产出 |
|------|------|------|------|------|
| **10:00-10:30** | 30min | L01 开场 + 痛点征集 | Lesson | 白板写出全班痛点清单 |
| **10:30-11:30** | 60min | L02 OpenClaw 是什么 + 3 个真实案例 | Lesson | 决策心智模型 |
| **11:30-12:30** | 60min | L03 🌸 Quest 装机里程碑（沿用 4h 版） | Quest | OpenClaw 装好 + 跑通"你好" |
| **12:30-13:30** | 60min | 🍱 午餐 + 1v1 答疑 | — | 加微信群 + 摸清自己生意能用 |
| **13:30-14:45** | 75min | **L04 Skill 1：FAQ 客服**（深跑） | Lesson + Lab | 接通微信/WhatsApp 跑通 |
| **14:45-15:00** | 15min | ☕ 茶歇 | — | — |
| **15:00-16:15** | 75min | **L05 Skill 2：GST 发票 + PDF 提取**（深跑） | Lesson + Lab | 真实合同跑一遍 + 反向开发票 |
| **16:15-16:30** | 15min | ☕ 茶歇 | — | — |
| **16:30-17:30** | 60min | **L06 Skill 3：老板日报 + 邮件分级**（深跑） | Lesson + Lab | 自动日报触发器配好 |

总: 净教学 **6.5h** + 午餐 1h + 茶歇 2×15min = **8h**

### 4.3 每节 Lesson 详细规格

#### L01 开场 + 痛点征集（30min · Lesson）
- 每人 30 秒自我介绍："我是做什么的 + 我最烦的 3 件重复事"
- 白板汇总所有人痛点 → 板书"下午 3 个 Skill 要解决的就是这些"
- 数据对照：澳洲请人 $24+/h / 招人难 / 24h 客服做不到 / 培训贵
- 课程地图：装机里程碑 → 3 Skill 深跑 → 部署收尾

#### L02 OpenClaw 是什么（60min · Lesson）
- 5 分钟讲透："你电脑里的 AI 员工"，不是聊天框，是干活的
- 三张对比图：OpenClaw vs ChatGPT vs Claude.ai
- **3 个真实案例**（每个 ~12-15min，覆盖混合行业）：
  - **餐饮**：墨尔本华人川菜馆 — Skill 自动回 WhatsApp 订位 + 菜单咨询，月省 1 个前台
  - **电商**：Shopify 跨境卖家 — Skill 看后台订单 + 客诉分级 + 9 点日报
  - **服务业**：地产中介 — Skill 监控同行挂牌 + 客户跟进记录 + 项目材料生成
- 数据：参考 4h 版的 27 资源 + 现场合影 + 续报率（待补真实数字）

#### L03 🌸 Quest 装机里程碑（60min · Quest，沿用 4h 版机制）
- 5 步装机 Quest（Mac + Windows 双演示）：
  1. 官网下载 OpenClaw 桌面端
  2. 注册账号
  3. 接 API Key（**讲师统一发临时 Key，老板不用自己注册卡住**）
  4. 跑通"你好"
  5. 接通 Claude Desktop 聊天界面
- AI 小花 1:1 引导 + 2 名技术 Tutor 现场救火
- 守住 4h 版踩过的 3 个坑：Windows 没 WSL2 / Node ≥22 / 不要用 Bun 装
- **12:30 前 100% 学员都跟 OpenClaw 说上话**

#### 🍱 12:30-13:30 午餐
- 提供餐食（含素食选项）
- 讲师 + 助教散坐答疑：摸清每个老板的"我这个生意能用吗"
- 加班级微信群（课后社群直接落地）

#### L04 Skill 1：FAQ 客服自动回复（75min · Lesson + Lab）

适用：餐饮订位 / 电商客服 / 服务业咨询通用

| 时间 | 动作 |
|------|------|
| 13:30-13:45 | 演示：讲师手机给 OpenClaw 发"今晚 6 点 4 个人有位吗" → 自动回复 |
| 13:45-14:15 | 老板填模板：自己生意的 10 个最常问问题 + 标准回复（**预置 3 套行业模板**） |
| 14:15-14:35 | 导入 OpenClaw → 接微信 / WhatsApp（讲师投屏带做） |
| 14:35-14:45 | 学员互发消息测试 + 拍照打卡发群 |

> **预置文件**：餐饮 / 电商 / 服务业各 1 份现成 FAQ Excel 模板。

#### L05 Skill 2：GST 发票 + PDF 订单提取（75min · Lesson + Lab）

适用：所有要开 GST 发票 / 处理供应商 PDF / 客户合同的老板

| 时间 | 动作 |
|------|------|
| 15:00-15:20 | 演示：客户邮件附件 PDF → 自动提金额 / 日期 / 收货地址 → 落 Excel / Xero |
| 15:20-15:30 | 反向演示：一句话"给客户 ABC 开 $1200 GST 发票，今天日期" → 生成澳洲合规 PDF |
| 15:30-16:00 | 学员上传自己最近 3 份真实合同 / 订单跑一遍 |
| 16:00-16:15 | 接通 Xero / MYOB / 微信收款（讲师演示 1 个，老板按需问） |

> **预置文件**：GST 发票模板、ABA 银行批量转账模板。
> **澳洲 GST 注意**：CLAUDE.md 明确"澳洲 GST 不要假设统一 10%"，模板要支持自定义税率。

#### L06 Skill 3：老板日报 + 邮件分级（60min · Lesson + Lab）

适用：所有老板（睁眼就知道今天该关注什么）

| 时间 | 动作 |
|------|------|
| 16:30-16:45 | 演示：每天早上 9 点 OpenClaw 自动看邮箱 + 微信群 + 订单 → 推送 5 条关键信息 |
| 16:45-17:00 | 演示邮件分级：紧急 / 待回 / 群发 / 垃圾 自动归类 |
| 17:00-17:25 | 学员配置自己的日报时间 + 接通 Gmail / Outlook |
| 17:25-17:30 | 手动触发自己的日报 → 收到 ✓ |

#### L07 部署 + 成本 + 行动清单（30min · Lesson）

- **7×24 部署 3 方案**：
  1. 家里旧电脑（免费，最常用）
  2. NUC 迷你主机一次性 AU$400（推荐 5-10 人公司）
  3. 云服务器 AU$10-15/月（AWS Lightsail / Linode，适合不想管硬件的）
- **成本控制**：Claude API Key 月用量预估（小公司 <AU$50/月够用）
- **安全 / 数据**：本地存储 / API Key 怎么保管 / 哪些数据**别**喂给 AI（客户隐私、合规敏感）
- **行动清单**：每人写下"下周一在公司部署的 1 件具体事 + deadline"
- 加群 / 颁发结业小证 / 现场合影
