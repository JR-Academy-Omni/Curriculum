电脑必须 awake
+ Cowork 必须开

Cowork Scheduled Task 不是云端 daemon — 跟 cron server / GitHub Actions 不一样。
睡眠期间触发的会跳过，电脑醒了 + Cowork 重打开会自动补跑一次。
✓ 笔记本常开（sleep + display off 算 awake） ✓ 装家里 / 公司常开台式机 ✓ 接受『偶尔跳一天』也行
