🤝 Claude Cowork 是什么

Anthropic 2026 GA 的桌面 App 产品。在 Claude Desktop 里跟 Chat / Code 并列的第三个 tab — 你给它一个目标，它在你桌面上跑：开你的 Excel / 读你本地文件 / 调你的 Connectors / 按 /schedule 每天自动跑 / 还能从手机 Dispatch 远程派活儿。

它把『Claude Code CLI 那一套能力』搬到了非技术人也能用的图形界面里 — 不用终端、不用 npm、不用写命令，全是说人话。本节课就是手把手带你跑通这个界面。

🤝 选 Cowork 如果

  • 你想 1 小时内上手，最好图形界面
  • 已经在用 / 准备订 Claude Pro
  • 工作主要在 PowerPoint / Excel / 浏览器（桌面 App 重度用户）
  • 不在意数据走 Anthropic（销售 / 市场 / 内容 / HR / 普通办公）
  • 想要手机 Dispatch 通勤路上派活儿

🦞 选 OpenClaw 如果

  • 客户数据 / 合同 / 病历不能上云端（律师 / 会计 / 医疗 / 合规岗）
  • 想用 Claude 以外的 LLM（OpenAI / Gemini / 本地 Ollama）
  • 团队已经在用 WhatsApp / 飞书 / Slack 沟通，想直接 IM 派任务
  • 需要 daemon 后台常驻不依赖电脑 awake
  • 已经会一点终端 + 想要完全开源可改
其实大多数实战场景两个搭配最爽 — Cowork 干桌面 GUI 活（出 PPT / 改 Excel），OpenClaw 干合规敏感 + IM 协作活。本节课主讲 Cowork，OpenClaw 看 4 小时进阶 Workshop（openclaw-workshop）。

Connectors

Dispatch

手机扫 QR 跟桌面 Cowork 配对（一次配对长期有效）。通勤 / 开会 / 出差路上想到的事，手机派给桌面跑。Pro / Max 含。

✓ 可同时派多任务并行跑 · 用桌面所有 App
⚠ 桌面 Cowork 必须开 + 电脑 awake

Skills
L05 / L06 · 工序卡

把『每次都这么干的事』固化成 Skill。下次 Cowork 输入框 / 一秒触发。可被 Plugin 打包、可被 /schedule 调用。

✓ 团队共享 = 团队 SOP 喂给 Cowork

📊 L05 · Case 1 — 手把手用 Cowork 出 PPT15:20 – 16:05 · 45 min · 实战

常见问题
报名前最常被问到的 8 个问题
我没用过 Cowork，只用过 ChatGPT，能跟得上吗？
能。Cowork 跟 ChatGPT 操作上 80% 像 — 都是输入框打字。差别在 Cowork 不只是回答你，它会真的开你电脑上的 App、动你的文件、按计划自动跑。本节课就是手把手带你装、配、跑，每一步老师等所有人完成才往下走。完全没碰过 AI 工具的同学，建议课前先开 ChatGPT 或 claude.ai 跟 AI 对一句话找下手感。
Cowork 必须付费吗？哪个档？
必须。Cowork 只在 Claude Pro / Max / Team / Enterprise 付费档有 — 免费档没 Cowork tab。最便宜路径：Pro A$30/月。课上没付费档的现场用 JR 共享测试账号过流程，回家继续用必须自己订 — 这工具是真的会替你干活的，A$30/月不算贵。
Mac / Win / Linux 都能上吗？
Mac + Win 都行（GA April 2026）。Linux 暂时没 Cowork 桌面 App。Win 用户务必下 MSIX 安装包，老的 .exe 版本没 Cowork tab — 这是踩坑率最高的点。本节课 L02 会强调。
课前要准备什么？
5 件事：① 自带笔记本（Mac 11+ 或 Win 10+）② 下 Claude Desktop（claude.com/download）③ 订 Claude Pro（A$30/月，没订发共享号过流程）④ 桌面建空文件夹 ~/Desktop/claude-workshop ⑤ 准备个人 Gmail + 想清楚 Case 1 PPT 主题。详见上方课前清单。
Cowork vs Chat vs Code 怎么选？
三个 tab 在同一个 Claude Desktop 里。Chat 适合『跟 AI 聊天 / 问问题 / 写文字』。Code 适合工程师改代码 / 跑命令行项目。Cowork 适合『让 AI 替你干电脑上的事』— 开 Excel / 改 PPT / 抓网页 / 定时跑 / 手机派活儿。本节课 2 个 case 必须 Cowork（Chat 没 Computer Use，Code 对非技术不友好）。回家三个一起用最爽。
/schedule 配的 daily-jobs 任务，电脑关机会怎样？
🚨 跑不动。Cowork Scheduled Task 不是云端 daemon — 必须电脑 awake + Cowork app 开着才跑。睡眠期间触发的会跳过，电脑醒了 + Cowork 重打开会自动补跑一次（你会收通知）。3 个解决：① 笔记本就让它常开（睡眠不影响 — Mac sleep + display off 算 awake，看 Energy 设置）② 装家里 / 公司常开台式机 ③ 不需要每天必跑就接受『偶尔跳一天』。本节课 daily-jobs 预设场景是『笔记本常开 + 8 点跑』，最简单可行。
Cowork 会不会偷偷动我电脑？数据安全吗？
Cowork 干每件事前都会让你 Approve（除非你在 Settings 关了确认 — 不建议关）。文件读写默认只在 working folder 内 — 跨文件夹要单独授权。Connector 读 Gmail / Drive 走 OAuth，权限细到具体作用域。Anthropic 默认对 API 用户 zero data retention（消息发完即删）。但敏感数据（客户身份证 / 银行账户 / 未公开合同金额）上传前还是用 [客户A] [金额X] 打码 — 这是好习惯不是 Cowork 的锅。
课后有支持吗？录播？答疑群？
3 件兜底：① 课程录播开课后发学员 ② 老师 L05 / L06 用的所有 Skill 模板 + Schedule 配置打包发学员（拿回家直接套自己场景） ③ JR Academy 平台 27+ 个 Prompt Lab 互动练习配套巩固。不开长期答疑群（避免变无效信息池）。想要长期社群和系统学习，看 JR Academy 的 OpenClaw Workshop（4 小时进阶）或 AI Adoption Bootcamp（8 周）。
