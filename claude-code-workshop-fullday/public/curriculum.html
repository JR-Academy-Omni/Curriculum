🎯 为什么要做「非技术友好版」

Claude Code 是 Anthropic（Claude 那家公司）出的命令行 AI 编程工具。听起来「编程」「命令行」吓人 —— 但你完全不用懂代码就能让它替你干活：批量改 100 个文件名、PDF 提数据塞 Excel、给每个客户写个性化邮件、把会议纪要变成 PPT 大纲。

问题是：非技术学员自己装 Claude Code 90% 会卡 —— Node 装不上、终端不肯打开、API Key 复制错位…… 任何一步卡住就放弃了。这就是为什么我们专门做一个 1 整天的非技术友好版：上午 2h 装机大救援 + 终端入门，下午 3h 用你自己的真实工作素材跑 3 个办公自动化，1:5 助教坐你旁边，卡住直接喊人。

❌

3h Zoom 工程师版（A$99）不适合你

那个班假设你会基础 git + 命令行，节奏快，3h 讲完 sub-agents 并行 / 完整 PR 流程。如果你连终端都没打开过，前 30min 就会掉队，剩下 2h 跟不上。

✅

1day 非技术版（A$299）才是给你的

节奏慢 4 倍，不假设你懂任何技术，把每一步拆给你看。上午全程 1:5 助教带装机，下午用你的真实素材跑实战，走时带走 3 个能直接复用的办公自动化模板。

👥 谁来上 · 5 类非技术学员

不需要会任何技术。只需要带笔记本 + 一份你日常处理的真实工作素材（邮件 / 合同 / Excel / 客户名单 / 公众号草稿都行）。

📊 销售 / 行政 / HR

每周大量重复处理客户名单 / 邮件 / 报表，想把这些用 AI 自动化。下午 L04 直接套你的真实客户表跑。

⚖️ 律师 / 会计 / 顾问

处理合同 / 报税资料 / 客户文档不想数据上 ChatGPT 云端。Claude Code 跑本地，文件不离开你电脑。

🏪 小老板 / 自由职业者

没钱雇助理，想让 Claude Code 当兼职运营。3 个自定义命令 = 3 件每周必做的杂活一键化。

📝 内容创作者 / 老师

写公众号 / 备课 / 改作业重复劳动太多，想让 AI 帮跑流水。L05 学会让 AI 自动爬资料 + 写初稿。

🔄 转行学习者

❓ FAQ

Q: 我连 Mac 都没怎么用过，能跟得上吗？
能。1:5 助教比例就是为这种情况准备的。L01 装机环节我们不允许任何人独自卡 10 分钟以上，宁可全班停下来等你装好。

Q: 我电脑实在装不上 Claude Code 怎么办？
JR 备 5 台已装好的备机，万一你电脑实在装不了直接借机用一天。课后我们远程帮你装回家。

Q: 跟 ChatGPT 上手课有什么区别？
ChatGPT 在它的网页里跑，看不到你本地文件、不能改你 Excel、不能批量重命名。Claude Code 在你电脑上跑，能直接读你文件 / 改文件 / 跑命令。所以 ChatGPT 是「对话工具」，Claude Code 是「替你干活的助理」。

Q: 下午带什么工作素材合适？
选一份你本周或下周本来就要处理的：客户名单 Excel / 一封要写的邮件 / 一份会议纪要 / 一段公众号草稿 / 一份合同片段都行。带真实的，跑完直接能用，比带模板有价值 10 倍。

Q: 课后能问问题吗？
能。3 个月持续答疑社群，进群随时问。还有全程录播 + 资料包 PDF。

Q: 学完能干嘛？下一步去哪？
L06 最后 5 分钟讲「下一步去哪里继续学」。如果你今天玩爽了想继续深入：JR 的 3h Zoom 工程师版（A$99）讲 sub-agents 并行 / 完整 PR 流程 / 跟你的 GitHub repo 结合，是 1day 班的进阶路径。
